## 一、马来西亚大促快速起内容场（★最相关，专属打法）

### 1.1 大促重点直播间的"报名换扶持"门槛（培训口径，以官方为准）

报名马来斋月大促重点直播间后，满足以下条件**有机会获得高销量 + 平台补贴**：

- 发布 **2 个预热短视频**；
- **每日开播 ≥ 2 小时**；
- 结合**合资券**（平台+商家共担的优惠券）；
- 平台补贴**最高 10%**（⚠️ 比例为培训口径，且为示例大促，**以当期官方活动页为准**）。

### 1.2 三步起内容场

1. **Step1 找当地合作机构（MCN/服务商）**：资料点名马来本地机构示例 —— **ULTRA MOON**（美妆首选，100+ 全约达人、5000+ 创作者）、**SEA IDEAL**（短视频+直播打法）、**The Alps**（签约达人 3000+、合作达人 30000+）。➡️ 美妆类优先 ULTRA MOON。
2. **Step2 按"斋月合作包"启动**：平台与本地机构合作提供专属档位资源包（预算分档，**具体以达人实际情况和机构条款为准**）。
3. **Step3 对接本地机构**：加官方群直接沟通。

## 二、短视频基础玩法

### 2.1 账号包装（初级 + 进阶装修）

| 模块 | 要点 |
|---|---|
| 名字 | 与产品相关、容易记住、与店铺名一致 |
| 头像 | 产品 / 真人 / Logo。真人头像增信任（例：白大褂牙医）；想做品牌放 Logo |
| 简介 | 一句话说清账号做什么；分行 2–4 行，**字数 ≤ 80 字符** |
| 店铺/直播活动 | 例：`At 8:00 p.m. on Jan 1, new products on sale in the live room, 20% discount!` |
| 直播时间/价值 | 例：`Our live streaming hours are 16:00-20:00 every night!` |
| 进阶 | 挂链接、优化视频封面 |

> 用当地语言或习惯用语，图仅参考。

### 2.2 新号冷启养号（避免 0 播的关键）

- 账号注册好后**先不要急着绑店运营**，先刷对标同类目视频 **3–5 天**（每天 ≥ 半小时），提高活跃度、给账号缓冲期。
- 之后**仿拍并发布**创作视频。
- **发布时间测试**：按店铺所在国家时间，分早/中/晚批量测：早 **7–9 点**、午 **11–13 点**、下午 **16–18 点** 各发一条；测出优质曝光时段后第二天集中在该时段发。
- **基础播放量** 起始 **200–500** 属正常，继续优化内容。
- **音乐**：必须用商业音乐（CML，商业音乐库），否则后期视频会被静音。
- **标签**：`#` 关键词，**3–5 个**够用，垂类词 + 泛词搭配（例宠物：`#pet #pets #cute #dog` + `#ForYou #fyp`）。
- 建议**运营 1 周~半个月**后再绑定店铺。

### 2.3 推流逻辑（货找人 / 流量池赛马）

- 视频发布后进入**第一级流量池（200–500 曝光）**，按内容匹配；看核心指标（播放量、点赞量、评论数、**完播率**）决定是否升级到**第二级（2000–5000）→ 第三级（5000–上万）**。
- **第一级流量优先推给当地国家的人**（IP/数据户口/SIM/机型/语言时区定位决定"推给哪国人"）。
- 内容会先过 **①内容检测**（封面/画面/音频是否违规）+ **②内容查重**：若判"搬运"→ 低流量推荐 / 仅推粉丝 / 零播不推。
- 自然流量推荐算法窗口约 **72h**；自然流数据越好，后续叠加付费广告效果越好（自然流点赞/播放与广告完播率、互动率正相关）。➡️ 先用优质内容跑自然流，再用广告做放大器（详见 `付费广告GMVMax.md`）。

### 2.4 带货短视频 5 种"种草题材"（用主推品引导进店成交）

| 题材 | 表现形式 | 推荐类目 |
|---|---|---|
| 工艺教程类 | 记录亲手操作、强调品质/效果 | 服饰配件、平价饰品、发饰 |
| 测评推荐类 | 真人使用 + 数据 + 前后对比 | 美妆个护、手机数码（横向测评 2–3 款热销品） |
| 口播讲解类 | 真人出镜深度讲解（难短时间展示效果的品） | 美容护肤、服装、食品、保健品 |
| 情景软植类 | 情景故事中自然软植商品 | 服装、全类目 |
| —— | （以上为种草，目的：测品 + 激发需求） | —— |

### 2.5 短视频制作"5 步走"

1. **前期准备**：选拍摄品（普适性强、有特点有卖点、便于镜头放大）；设备 = 手机（iPhone 11+/华为 P40+）+ 三脚架 + 背景灯/补光灯。
2. **撰写脚本**：多看当地视频找"网感"；用产品关键词找对标账号 → 拆解高播放视频（带问题拆：整体感受/选题台词/场景/分镜/音效/剪辑）→ 按自身长处二次创新。
3. **作品拍摄**：按脚本分镜拍（中心构图法）。
4. **后期制作**：剪辑/字幕/音乐/调色滤镜/特效音效/封面 → 过审。
5. **作品发布**：加话题文案 + 发布。

**爆款短视频通性**：产品为主、突出核心、创意为辅；**10–20 秒内高效种草+割草**；强种草属性（"居然还能这样？"，可视性强的新品或效果明显的功效品）+ 强收割属性（"这么划算不买就亏"）+ 极其精练（一句多余话/一秒多余画面都不要）。

### 2.6 优质短视频六大基础要素（美区资料，通用）

1. 画质清晰、曝光正常（别过度美颜磨皮）；2. 不遮挡关键信息（人脸/品牌/产品细节）；3. 音质良好、人声稳定；4. 背景干净整洁；5. **真人出镜、外语流利**（不建议全程 AI 配音）；6. （配合内容质量）。

### 2.7 短视频选品特点

- 最大用户群体 **18–24 岁女性**；优先**普适性强**的品（过于小众/风格化时装受众小）。
- 有特点、有卖点、便于镜头放大。
- **备货充足**：起量后备足支撑 **1–2 周**库存，避免供应链跟不上断货。

## 三、直播基础玩法（★本簇最完整，2025/05/30）

### 3.1 影响直播 GMV 的 5 个环节（拆解流量漏斗）

用户路径：看到直播间 → 点击进入 → 驻足停留 → 交流互动 → 点击商品 → 购买商品 → 服务体验。拆为 5 个可优化因素：

1. **直播时长** = 直播次数 × 每场时长 → 直接影响曝光量。
2. **进房率**（曝光-点击转化率）= 进房人数 / 曝光量。
3. **人均停留时长** → 内容优质度信号，越长越利推流与转化。
4. **商品点击率 CTR** = 商品点击数 / 商品曝光数。
5. **商品转化率 C→O** = 成交订单数 / 商品点击数。

> **GPM = CTR × C→O × 客单价(AOV) × 1000**，代表每 1000 次观看带来的成交金额，高 GPM 是吃自然流的关键。

### 3.2 内容基础运营 3 步

1. **账号运营**：先定位（决定运营方向，定位偏差后续做不下去）→ 内容策划 → 形式测试。
2. **发布视频**（4 种）：① 日常更新（吸精准粉、提权重）② 预告视频（开播前导流）③ 直播切片（直播中发，抢免费精准流量）④ 播后视频（答谢粉丝、强化人设 IP）。
3. **积累粉丝**：开播时有垂直粉先进场、长停留，喂好开场数据。

### 3.3 直播 4 步流程

- **Step1 前期准备**：账号/网络/权限（电商权限自动获取、直播权限绑店后获取、OBS/LiveStudio 权限）；主播（形象气质贴合、表现力/互动力/销售力、专业讲解、**当地语言流畅**）；产品（边播边测找爆品）。
- **Step2 搭建直播间**：氛围感、明亮有层次。三种类型：品牌专场 / 工厂 / 实体店。
  - **站播**（~20㎡，站立）：服饰/家纺/家具/箱包；**坐播**（~10㎡，坐着）：美妆/珠宝/玩具/饰品。
  - 设备：手机/摄像头（11 以上）；运营手机（7 即可）+ 拍摄灯光。
- **Step3 撰写脚本**：以 **10–15 个品、约 2 小时** 规划每品销售时间；**黄金三分钟**——买家停留 6 分钟已算久，前 3 分钟最关键，单品介绍节奏别太慢。
- **Step4 开播**：封面设计 + 开播前 3 步准备（排品）+ 开播中技巧 + 直播后 4 步复盘。

### 3.4 直播排品逻辑（★核心数字，培训口径）

**四类款型与占比**：

| 款型 | 作用 | 商家占比 |
|---|---|---|
| 引流款 | 折扣价、中低客单，保流量留存 | **≤ 10%** |
| 动销款 | 低毛利、高热度、普适性强，拉转化 | **≤ 30%** |
| 利润款 | 毛利适中、新品/特色战术品，放大 GPM | **40%–50%** |
| 特色款 | 探索性类目，长期粉丝留存 | **≤ 10%** |

- **上架顺序**：引流款 → 动销款 → 利润款 → 动销款 → 特色款 → 引流款，依次循环。
- **价格顺序**：低 → 中 → 高 → 中 → 低 → 中 → 高，依次循环。
- **排品三原则**：品类一致（爆款后排衍生品）/ 价格衔接（逐渐递进，便于提客单）/ 避免同质（同款同功效别批量铺、一个链接别塞 2–3 品）。
- **组货**：① 单 SKU 多件（品少商家，爆款多件、量增价增加赠品）② 多 SKU 组合（品多商家，选 2–5 个高知名度爆款，组合要连贯如"外套/内搭/裤子"，客单价做到单爆款的 2–3 倍）。

### 3.5 进阶：提升进房率的 4 种方式

1. **活动引导**：平台活动（斋月大促）/节日活动/店铺活动（周年庆、清仓）；优化标题=活动主题+利益点；场景布置贴合主题；主播服化道贴合（圣诞老人/医生讲护肤）；卖场类直播主播语速更快制造抢购紧迫感。
2. **商品吸引力**：颜值高（服饰上身）/稀缺性（珠宝、控库存）/时效性（夏天降温品）/功能性（防脱发洗发水）；可视性差的品（化妆品）靠主播演示+成分口播；难展示的品（灭蚊药）用玩偶服帮理解。
3. **品牌效应**：品牌 logo/色/slogan + 专柜/门店/工厂销售场景增信任（适合旗舰店、代理商）。
4. **兴趣与人群契合**：用脱口秀/演唱会等娱乐玩法；通过成交人群偏好 + 数据罗盘市场搜索词找兴趣点，再定场景（原产地采摘、流水线、舞蹈室）。

### 3.6 进阶：提升互动率 3 法（重点关注"评论+关注"）

- **话术利他法**：产品福利/卖点 + 引导关注/评论（"想要红色评论红色，想要绿色评论绿色"）。
- **指定人群法**：特定人群痛点 + 引导（"对身材不自信的姐妹评论有，我上波福利"）。
- **情感共鸣法**：共鸣场景/观点 + 引导（"感谢粉丝陪伴，想做更好产品分享给大家"）。

### 3.7 进阶：提升转化率 3 法

- **主播话术 ASBC 单品框架**：A=Appeal 吸引（提问/自问自答引入）→ S=Sales 销售（销量背书/先抑后扬/对比产品）→ B=Benefit 利益点（真实福利 + 限时限量）→ C=Customer 用户（圈定人群、对比产品）。配"动作引导下单"（另一台手机演示如何点小黄车）。
- **商品属性结构**：标题加行业热词+功效词（来源：数据罗盘-市场分析-关键词分析，找热搜+飙升词）；主图=品牌名+卖点+价格利益点（拍一发三/到手价/买一送一）；详情页打消疑虑。
- **营销活动场景**：跟行业大促（66/77/99）；造自有节日（宠粉日/会员日/周年庆，3–7 天一节奏循环）；营造稀缺性（限时秒杀/限量库存/有限赠品 + 播报下单数）。

### 3.8 服务体验指标（影响信任与流量，以后台为准）

- **店铺差评率**：1–2 星评价占总评论比。
- **商责订单取消率**：商家责任取消的 SKU 订单占已支付 SKU 订单比。
- **延迟履约率**：已发货但未满足发货 SLA 的 SKU 订单占比（仅卖家自发，不含 TTS 仓发出）。
- ➡️ 履约口径与 SLA 以 `04-履约发货/` 及官方后台为准。

### 6.3 美区矩阵打法（《美区爆品玩法》，美区，谨慎迁移）

- **货找人 4 要素**：①符合推荐算法 ②内容质量 ③内容广度 ④付费流量做放大器。
- **四类内容号**：流量号（街拍/影视剪辑，流量大但标签不准）/ 人设 IP 号（粉丝粘性强，KOL/KOC 合作）/ 好物种草号（开箱测评/OOTD）/ 品牌号（体现调性）。**品牌号 + 矩阵号 = 全域品宣+品销**。

## 七、避坑与故障自查清单

### 7.1 短视频"0 播放 / 低播放"自查

- **环境问题**：
  - 硬件：手机 / 手机卡 / 路由器（影响小）。注意：**不能插国内手机卡**，需插目标地区卡或不插卡（手机卡非必要）；昨天能刷今天黑屏，插目标地区卡可解。
  - 软件：节点 / 伪装度（地区/广告追踪/语言）/ 邮箱（Gm/Outlook，**域名邮箱有被关联风险**）。
  - **关联风险**：同时注册 10 个号，1 个重大违规可能连带其余号被封。
- **内容问题**：静态/静音/模糊视频；搬运重复度高。
- **低播放归因（新号）**：质量原因约 **80%**（优化内容）/ 账号原因约 **20%**（前期上千上万、2–5 天后断崖到 0 → 查节点，必要时弃号重起）。
- **有粉账号低播**：断更（预估播放降低）/ 严重违规。

### 7.2 账号绑定/关注问题

- **TikTok 绑定失败**：多因注册时环境/国家没选对，**店铺 IP、账号 IP、手机 IP 必须一致**。
- **新号无法关注他人**：等 **1–3 天**给缓冲期再试。

### 7.3 直播"不进人 / 流量跑偏 / 被判录播"

- **流量跑偏（非精准国家/用户）**：查 ①网络环境 ②内容受众（看自己内容是否更受其他地区喜欢，针对性优化目标国内容）③极速推流把外地流量带进来后转化更好 → 平台推更多外地流量。
- **不进人/没推荐（6+1 自查）**：禁忌词违规 / 侵权（主播穿 LV·NIKE、放未授权音乐或吉祥物如 Mickey → 穿普通衣物、别放未授权音乐与物品）/ 水军账号涌入（不活跃号搜索进场长停留不互动会压流量）/ 留人能力差（看大屏紫色实时在线人数线是否陡降）/ 网络不稳 / 直播内容（衣衫不整、出现未成年人）/ 大促期流量抢占加剧。
- **被判录播降误判**：主播勤换衣服；主播账号绑定店铺（同主播同时出现在两间且无团队绑定会被判录播）；直播间放一个显示当地开播时间的大时钟。仍被误判 → 带真人直播证据（场景视频+内容+roomID 对应）填"录播加白表格"提报。
