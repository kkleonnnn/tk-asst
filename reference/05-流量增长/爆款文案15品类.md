## 一、核心方法论（最该记住的部分）

### 1.1 文案黄金结构（每条带货脚本的主干）

```
痛点切入 → 产品植入 → 卖点展示 → 效果证明 → 行动号召(CTA)
```

### 1.2 文案六大模块（按出现频率排序，可拼装）

| # | 模块 | 作用 | 话术举例 |
|---|------|------|----------|
| ① | 用户痛点 / 需求产生（**最高频**） | 3秒内抓注意力 | "你在吃塑料？！" "受够了为一次性用品买单？" "背痛或姿势问题？" |
| ② | 产品卖点 / 产品信息 | 用具体卖点建立信任 | "天然抗菌" "可重复使用" "自磨刀系统" |
| ③ | 使用说明 / 功能演示 | 现场演示、边操作边讲，增强真实感 | — |
| ④ | 优势讲解 / 产品功效 | 量化对比 | "比传统清洁器吸力更强" "三年保修" "预缩水不变形" |
| ⑤ | 价格促销 / 优惠活动 | 制造"赚到了" | "立减100美元" "37% off" "限时折扣" "优惠券" |
| ⑥ | 行动号召 CTA | **每条文案的最后一步都是 CTA** | "链接在下方" "点击橙色购物车" "趁库存还在赶紧抢" |

**3 个高转化加分项**：
- 真实体验/使用感受："闻起来像五星级酒店" "这感觉太棒了"
- 竞品对比/痛点对比："不要买自动猫砂盆，是个巨大骗局"（反向建立信任）
- 情绪营销/场景代入："拖延症终于有了回报" "我简直要陷进去了"

> **开头铁律**：前 3 秒必须抛出痛点或悬念，否则用户划走。

### 1.3 六大万能公式（可直接套用的模板框架）

| 公式 | 结构 | 适用品类 | 案例 |
|------|------|----------|------|
| **一、痛点炸场型** | 震撼提问/反常识陈述 → 痛点放大 → 你的解决方案 → 效果展示 | 健康/安全/环保类 | "你在吃塑料？！"→微塑料危害→竹砧板替代→37% off |
| **二、竞品打击型** | 劝退热门竞品 → 揭露竞品缺陷 → 自家产品优势 → 性价比碾压 | 有明显竞争者的功能性产品 | "不要买 Litter Robot！是骗局！"→乱抹/发臭→1/5价格+更好功能 |
| **三、效果对比型** | 使用前状态 → 产品介入 → 使用后结果 → 数据/视觉对比 | 美妆/个护/健身/清洁等效果可视化产品 | 上牙贴美白贴 vs 下牙不贴→30分钟后→"白了好多！" |
| **四、专业背书型** | 专业身份亮牌 → 科普行业知识 → 对比传统方案成本 → 产品解决方案 | 需要建立信任的高价/复杂产品 | "我是注册护士"→微针原理→省$600医美费→专利精华液 |
| **五、生活 Vlog 型** | 日常场景切入 → 自然过渡到产品 → 真实使用感受 → 轻松收尾 | 服装/穿戴/日常用品，需亲和力 | 晨间散步→束腰带使用→三次剖腹产经历分享→推荐 |
| **六、短剧情/场景型** | 设定有趣场景/人物关系 → 对话中植入产品 → 冲突/反转 → CTA | 数码/家居/工具等偏理性决策的产品 | 二手机交易场景→"跟官方一样但半价"→TikTok Shop 更便宜 |

**公式选择指南**：
- 低单价快消品 → 公式一（痛点）或 公式六（剧情）
- 中单价功能品 → 公式二（竞品）或 公式三（效果）
- 高单价任务品 → 公式四（专业）或 公式五（Vlog）

> 所有公式的最后一步都是 **CTA**："链接在下方 / 点击橙色购物车 / 趁库存还在抢"。

## 四、实操避坑指南（落地最关键）

### 爆款文案的共同特征（照做）
- **前3秒定生死**：必须抛痛点/悬念/冲突，绝不能慢热开场
- **语言口语化**：像跟朋友聊天，不用书面语，适当加入语气词
- **具体数字胜过形容词**："立减100美元""37% off""5磅装""30分钟""比'很便宜'更有效100倍"
- **价格锚定**：先说原价/竞品价再说优惠，让用户感到"赚到了"
- **稀缺感**："卖断货""补货了又卖光""限时折扣""库存有限"
- **多角度展示**：尤其美妆/服装，正面侧面都要给到

### 常见错误清单（别犯）
| ❌ 错误做法 | 原因 |
|------------|------|
| 开头讲品牌故事/公司介绍 | 用户没耐心听 |
| 只罗列产品参数不讲好处 | 用户只关心"对我有什么用" |
| CTA 不够明确 | 要说"感兴趣可以看看""不如链接在橙色购物车" |
| 视频太长（>30–60秒） | 爆款文案多 30–60 秒，信息密度要高 |
| 忽略目标市场语言偏好 | 拉美市场用英语会大幅降低转化 |
| 没有效果证明 | 光说不练，用户不会信 |

### 拍摄制作建议
- **设备**：手机即可，重点在于光线充足
- **收音**：清晰的人声比画质更重要
- **出镜**：真人出镜 > 纯产品展示，有亲和力 > 完美的模特
- **剪辑**：节奏要快，**空镜头不超过 3 秒**

> **终极建议**：先模仿再创新。从这套 45 条文案里选跟你品类最接近的，套用其结构和节奏。
