## 0. 一句话理解联盟带货

你把商品设好佣金放进联盟，达人帮你拍视频/直播带货并挂车，**卖出去才给达人付佣金**（CPS，按成交付费）。核心是「**对的货 → 找对的人 → 寄样 → 推动他发内容 → 跑出爆品 → 投流放大 → 维护关系**」的飞轮。

---

## 1. 两种合作方式：公开合作 vs 定向合作

| | 公开合作（Open Plan） | 定向合作（Targeted Plan） |
|---|---|---|
| 玩法 | 把商品设好佣金放进联盟「货架」，**所有达人**都能看到、主动申请挂车/申样 | 你**主动挑选**达人、设好佣金和合作期，逐个或批量发邀请 |
| 建联范围 | 最广 | 最精准 |
| 适合 | 新手起步、测达人偏好与数据 | 已有合作关系、想定向放量 |
| 是否要审核 | 可开「需要批准」开关，逐个审核达人申请 | 达人在收件箱收到邀请，接受后即可挂车 |

- 两种可**同时使用**；官方建议新手**先用公开合作**熟悉机制。〔可信度：中，官方手册口径〕
- 入口：商家中心 → **联盟带货** → 左侧菜单「公开合作 / 定向合作 / 寻找达人」。〔以实际后台为准〕

## 2. 后台操作 SOP（东南亚各站通用，含马来西亚）

### 2.1 定向合作三步（创建邀请）
进入「联盟中心 → 定向合作 → 邀请达人合作」，右上角点「邀请达人合作」：
1. **选择达人**：按用户名/用户ID搜索，或从「保存的达人」里添加。
2. **选择产品**：选好产品后**逐个设佣金比例**，系统自动估算达人预估佣金。⚠️ 添加前先确认库存，别因售罄错失订单。
3. **创建邀请**：填邀请名称、**有效期至**（达人可接受并推广的截止时间）、联系邮箱、可选一段说服文案。
- 开启后可看每个定向合作的表现：多少达人接受、多少已发内容、协作/达人/产品详情。

### 2.2 公开合作设置
- 入口「公开合作」页有两个标签：**已添加的商品** / **未添加**。
- 加品方式：逐个「添加到联盟」，或顶部「批量添加到联盟」一次全加。
- 「设置公开合作详情」弹窗里设：
  - **佣金比例**（系统给建议值，最终卖家定；掘金宝典建议**10% 起**）
  - **需要批准**开关（开 = 达人挂车前你要逐个审）
  - **设置免费样品**开关（开 = 达人可申请免费样品拍内容）
- **自动添加新产品**：右上角「自动添加商品」→ 开开关 + 设统一佣金 → 以后新上架商品自动按该佣金进联盟（适合 SKU 多的店）。
- 删品：勾选商品 → 「批量删除/停用」。

### 2.3 「寻找达人」页面（找人主战场）
入口：联盟中心 → 寻找达人。分**筛选区**+**达人列表**两部分。

**可用筛选维度**（均基于达人近 30 天数据为主）：
- 达人筛选：**产品类目**、**粉丝数量**、**平均佣金比例**、**内容类型**（短视频/直播）、**达人机构**（机构管理 or 独立）、**快速增长**（绿色徽章=近30天交易/收入等显著增长）。
- 粉丝筛选：**粉丝年龄段**、**粉丝性别比例**。
- 数据表现筛选：**GMV**、**销量**、**视频平均观看量**、**直播平均观看人数**、**互动率**（点赞+分享+评论/观看）。多项支持「**仅看可购物内容**」开关。
- **品牌合作**筛选：可查达人是否带过竞品或同类目品牌。

**达人详情页能看到**：联系方式（电话/邮箱）、销售表现与趋势、内容表现、畅销品类及合作品牌、粉丝画像、可购物视频示例。
- ⚠️ 若达人未授权，部分数据不可见。
- ⚠️ 视频预览：**PC 端暂不支持看视频**，需用 TikTok App 扫达人详情页二维码跳转主页看内容。〔可能已优化，以实际为准〕

**保存与批量邀请**：点达人旁**心形**保存 → 右上角「已保存达人」标签 → 单个或「批量邀请」发定向合作 / 发私信。

**达人排行榜**：发现达人 → 查看排行榜，按主类目筛你店铺类目里表现最好的达人，可按日/周/月切换。

### 2.4 取消合作
- **公开合作**：找到商品 → 点「达人商品橱窗」数字进详情 → 找到达人 → 右侧垃圾桶删除。
- **定向合作**：进定向合作 → 找计划 → 右侧 edit → 达人列表里删达人 → 弹窗提示该达人正在推广 → Got it 确认 → 保存。
- ⚠️ 取消后**达人橱窗和历史发文里挂的商品会立即消失**，建议先和达人沟通原因、维护关系。〔可信度：高，官方 FAQ〕

## 3. 全链路实操 SOP（掘金宝典——最实用，重点）

> 主线：**对的货找对的人**。这是「招商部」培训材料，方法论扎实但带营销口吻，数字多为举例。

### 第一步 选品：不是所有货都适合达人带
适合达人带货的商品特征：
- **毛利高**：能留出 **8–15% 佣金空间**，佣金越高达人越优先选。〔参考值〕
- **普适性强**：受众别太窄（过于风格化的时装单坑产出低、易被换品）。
- **有卖点、便于镜头展示放大**：才有内容可做。
- **货值不高**（走「蚂蚁雄兵」海量达人路线时，要压低样品成本；靠头部达人爆发则不强求）。
- **转化好但曝光低**：CTR、转化率高但 PV 少的品，灌入达人流量能快速转单。
- **库存深**：起量后能撑 **1～2 周**库存，供应链快速补货防断货（联盟是「爆品+短时间蜂拥下单」逻辑，断货很可惜）。
- **链接历史销量高、评论多、店铺体验分高**：信任度高更易转化。

**测品 → 集中**：测品期可多品；测出爆品后把达人资源集中到几个头部爆品。规律——一个店 **90% 销售额常来自前 5 个爆品，甚至 1 个爆品贡献 70% GMV**。
**货盘要垂直**（同类目/价格带/风格，面向同一批达人和消费者）：① 只需建一个达人库；② 对一个达人可批量寄全店样品；③ 建立信任后达人能反向从店里选品。

### 第二步 选达人：不是所有达人都适合带你的货
- **人货匹配**最重要：达人粉丝画像要跟商品消费者画像匹配（品牌型商家找错调性会伤品牌）。
- **内容形式适配**：客单价高的商品更适合**直播型**达人；非标品（服装配饰）也适合直播型（可批量挂车过款）。
- **内容频率**：发文太低的（短视频 < 1条/周、直播 < 1场/周）履约率差。
- **内容质量**：质量不稳定的达人，拿样后产出常不理想。
- **有电商历史**：带过同类货且出过单的，带货能力和意愿更强、更易转化。
- **达人库要多大**：走蚂蚁雄兵模式，**每个产品对应 3,000～4,000 个潜在达人入库**（按当前回复率/申样率/履约率估算，仅供参考）。〔数字会变〕
- **粉丝数 ≠ 出单**：目前未发现粉丝数与订单量有关联，几万粉的达人也可能很能带货。

### 第三步 找到达人：5 种触达方式（建议多管齐下）
1. **定向合作**——建联效率最优：对库内精筛达人发定向计划，接受邀请的达人合作意图已明确。要设**专员每日查看谁接受了**，主动对话、引导申样。
2. **公开合作**——建联范围最广：全店设好佣金（≥10%）、样品门槛和数量，达人自主申请。专员每日查看谁挂车 → 按第二步标准二次筛选入库 → 寄样跟进。
3. **陌生拜访**——站内+站外全渠道：
   - 站内：达人广场私信，⚠️ **达人不回复时触及次数上限只有 5 次**。
   - 站外：商用账号常主动展示联系方式；也可从短视频/直播间留言、或达人同名社媒账号建联。
4. **机构合作**——找 MCN / TAP 团长：MCN 签约管理达人矩阵（赚达人服务费）；TAP 团长批量撮合中腰尾达人（赚分销佣金）。合作一家 MCN = 接触其旗下网红矩阵。
5. **UGC 自来水**——把买家变达人：
   - 下单 30 分钟内客服触达，问对方是否在 TikTok 发内容，是则抛利益点引导发推广视频；
   - 包裹附小卡片引导拆箱视频；
   - 利益点设计：发一条视频可 X 折买第二件/领券；变带货达人卖出给佣金；要求挂店铺专属 #hashtag、@ 官方账号，专人监控做投流。

### 各国达人偏好联系渠道（重要，按国家区分）
| 国家 | 偏好渠道（优先级） |
|---|---|
| **马来西亚** | **WhatsApp** |
| 泰国 | LINE > 电话 > TikTok Messages > Email |
| 菲律宾 | Viber |
| 越南 | Zalo > Facebook Messenger |

### 第四步 寄样品：尽量走 TikTok 系统（便于追踪）
- 在**公开合作**里设免费样品的申请标准和库存；样品成本由商家承担。
- 达人申请 → 商家在「样品申请」界面**审核+监控履约**；通过审核的样品**自动并入后台订单发货**。
- **省运费小妙招**：若有多个适配该达人的品，引导达人**一次申请多件、一个包裹寄出**，减少反复寄样的时间和运费成本。后台站内通信可一键发商品卡。

### 第五步 推进履约：高频沟通 + 给爆文脚本
关键沟通节点（话术见第 4 节）：样品寄出后 → 到货后（激励发文、发同款爆文链接）→ 约定发布时间 → 到货多日未发（催 + 抛阶段性福利）。
- **给达人爆文脚本**：爆文是该货品最佳内容实践，让达人批量学习能最大化每条内容动销率。脚本来源：① 借鉴中国已爆的同款视频/直播脚本并本地化；② 卖家自己组织；③ 归纳本地达人已跑出的爆款。
- 设少量投流预算牵引达人多发、维护重点达人关系。

### 第六步 产出数据后复盘
- **方法论沉淀**：拆解爆单内容——呈现了哪几个卖点、用什么话术/镜头/脚本结构；拆解差内容找避雷点；看评论区和商品评价归纳用户在意的卖点，迭代给达人的指引。
- **更新达人库标签**：记录履约情况和动销，便于针对性运营（建立 S/A/B 分级，见第 5 节）。
- **投广加热爆单视频**：自然流量已验证的爆视频可小额投广放量、巩固达人关系。
  - ⚠️ **成本警告**：已给达人分佣，投广是额外叠加成本，要控成本。可在视频爆单后与达人重谈分润（如**买断素材**后换无佣金链接投广）。

### 第七步 维护达人关系（话术见第 4 节）
- 发了但内容差/没出单的：鼓励提质、再发，给爆文脚本参考。
- 内容好+出单但只发一条的：鼓励多发，用「最好的达人一周发 X 条、播放 XX、赚 XX 佣金」做激励。
- 多次发、质量好、出单的：**重点维护**，预告上新、保持关注。
- **特殊节点**（上新品/库存紧缺/佣金价格机制调整）对重点达人**尽量人工告知**。
- 日常：和达人交朋友，倾听诉求、尊重意愿、及时回复。

## 4. 话术模板库

> 通用原则：**简短、表情符号点缀 😊、展示商品卖点和公司背书、避免硬推销、尊重达人意愿、做多个话术 A/B 测试**。涉非中英文达人需翻译成本地语言；**群发邮件务必用密送 BCC**，勿泄露达人隐私。

### 4.1 马来西亚邀约文案（马来语，WhatsApp 优先）
> 出自《达人合作邀请文案参考》——马来西亚部分。用 `YOUR NAME`/`SHOPNAME` 占位替换。

**马来语版：**
```
Hai, saya YOUR NAME dari kedai SHOPNAME
Kami sangat menyukai video anda dan ingin menjemput anda untuk mempromosikan produk kami.
Jika anda berminat, sila beritahu saya.
Kami ingin menawarkan anda komisen + bahan kandungan video produk, berharap untuk mendengar daripada anda!
```
**英文对照：**
```
Hi, I'm XX from the shop of XXX
We love your videos and would like to invite you to promote our products.
If you are interested, please let me know.
We would like to offer you commission + video content materials of products, look forward to hearing from you!
```

### 4.2 通用结构化模板（掘金宝典，需本地化翻译）
- **定向邀请**：
  > 你好 XXX 🥰，我们是【XX品牌/店铺】，邀请您推广我们的【XX产品】。主打【XX卖点】，跟您的粉丝画像很匹配！我们提供【免费样品🎁、XX%佣金、爆款视频投流】，感兴趣请联系我们，非常期待🤩
- **初次陌生触达**（三段式）：① 简短自我介绍+公司背书（如「XX国 TikTok Shop 类目第一/合作过 XXX 位达人」🔥）；② 说明来意，体现对达人账号的熟悉（「你的【XX视频】获得 XX点赞」）；③ 介绍货盘优势+利益点（免费样品/佣金/内容指导剪辑/投流/长期好货供给）。
- **引导多品申样**：
  > 除了你申请的【XX商品】，还有【XX】【XX】也很适合你，分别有【XX卖点】【XX历史销量/好评/头部达人带过】🤩。告诉我你还对哪几个感兴趣，可以一起申样一个包裹寄过去。
- **样品寄出后**：
  > Hi【达人名】样品已发出，物流单号 XXX，请查收哦~😄 拍摄前记得检查样品是否完好，有问题随时讲，好期待和你合作🥰
- **样品到货后激励**：发同款本地播放量最高的 1-2 条视频链接 +「他们一条就赚了 XXX 佣金，你也可以的👍」+ 发爆文脚本 + 约定发布时间。
- **催未发文**：
  > Hi【达人名】样品到了有些日子了视频还没发，有什么顾虑呀？预期什么时候发布？我们近期有【促销/投流/库存紧张】专属福利，要抓住机会🙌
- **维护话术**（三档）：差内容→鼓励提质+给脚本；好内容只发一条→鼓励多发+用 Top 达人数据激励；多次好内容→重点维护+预告上新+@官方账号投广。

### 4.3 60 条现成短话术（海外小店达人邀约话术）
> 一份「大卖孵化中心」整理的 60 条即用话术，中英/马来/泰/越多语种混排。**风格直白、营销味重、偏「工厂老板/高佣金」叙事**——适合海量陌拜起量，但**不适合塑造品牌调性**，谨慎用于品牌店。可挑用、改写。常见模板要素与参考佣金区间：

- 「专业手机壳工厂，你只出链接，发货我全包，还能提供短视频」——**低价高佣**叙事
- 「商人联盟，提供免费样品，**10–30% 佣金**计划」
- 「中国工厂老板/厂长，一起致富，给你一部分利润 + 产品视频」
- 「先发带图视频，粉丝喜欢就寄样，销量好给**底薪 + 月度提成**」（穆斯林女装/女包/箱包等多用）
- 美妆/美甲类：「20% 佣金起，效果好提到 **30%-40%**」「提供免费样品或产品视频」
- 手链/手镯：「**40% 佣金，RM17.9-19.9 价格带**，免费样品请提供收货地址」（马来西亚 RM 报价示例）
- 阶梯激励：「5 单送免费样品 → 30 单进定向计划拿更高佣金 → 150 单成合作伙伴 → 499 单 99 奖金 + 特别佣金」
- 马来语整段示例（女包/箱包，含底薪+佣金）：

- 空气炸锅（马来西亚场景）：「我们的空气炸锅产品在马来西亚很有名，能否帮我们带货？发 TikTok 视频和每日直播，感兴趣 WhatsApp 回复我」

## 5. 达人分级与考核（精细化 SOP 补充）

> 出自《精细化 SOP》——这份更偏**品牌内容营销/投流**，部分内容（坑位费、信息流广告、跨平台二剪）超出纯 Shop 联盟范畴，**作为进阶参考**，新手跨境店不必全做。

- **达人分级体系**（复盘后入库打标）：
  - **S 级**：ROI > 1:3 → 年度战略合作、给流量扶持。
  - **A 级**：数据达标但无亮点 → 下次试新内容形式。
  - **B 级**：未达标 → 分析原因再定是否二次合作。
- **筛选/参考指标**：曝光（粉丝量×历史均播）、转化率（如美妆类历史带货转化 2%-5%）、粉丝重合度（>60% 需换不同领域达人做矩阵）。〔均为举例数字〕
- **内容/发布技巧**（通用且实用）：竖屏 9:16、人物占比 60%+、产品特写清晰；发布后 0-2 小时盯 **5 秒完播率**（<30% 优化开头）、互动率（点赞+评论 ≥5% 为健康）；24 小时看流量来源（推荐流量 <40% 需加投流）。
- **东南亚发布黄金时段**：**12:00–14:00、20:00–23:00**（当地时间）。〔参考〕
- ⚠️ 文中「坑位费 2000-1万美元 + 佣金 10%-30%」「腰部达人 500-2000 美元/条」等为**头部品牌投放报价举例**，与跨境店纯 CPS 联盟不是一回事，别误当行情。

## 6. 避坑要点速查

- 添加产品到联盟前**先确认库存**，别因售罄断了正在跑量的爆品。
- 站内私信对不回复的达人**最多触及 5 次**，珍惜次数；想多渠道就走站外社媒/邮箱。
- 群发邮件用 **BCC 密送**，不泄露达人隐私。
- 取消合作会**立刻清空达人橱窗里你的商品**，先沟通。
- 投广爆视频时记得：**佣金成本 + 广告成本会叠加**，先算账，或买断素材后换无佣链接投广。
- 60 条话术那份**营销味很重**，品牌店慎用；优先用 4.1/4.2 的有礼版本并本地化。
- 《爆品表》是**某时点的图片清单、已过时**，只可作选品「品类灵感」，不可照抄链接（见附录）。
