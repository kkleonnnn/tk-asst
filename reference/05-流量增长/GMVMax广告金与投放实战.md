## 0. 一句话抓重点

GMV Max **是放大器，不是发动机**——能不能跑出 ROAS，**先看品、再看素材**，后台参数只是「别把水龙头拧坏」。投放最大的纪律是：**不要频繁调整**（每天盯、但不是每天动）；最大的安全网是：**有自动赔付机制**，ROAS 没达标系统会赔，所以更要耐心让算法学完。

## 1. 广告为什么「跑不动 / 没消耗」——先懂竞价 5 步

广告从入库到投放经历 **5 个环节**，卡在哪一环就表现为「没消耗/投不出去」：

1. **查账户余额**（没钱不投）
2. **查广告合规**（违规不投）
3. **分发到人群流量池**
4. **出牌**（粗排/初筛，第一轮公式）
5. **金牌**（精排/竞价胜出，第二轮公式）——过了金牌才上线、才开始消耗。

> 排查口诀：**没消耗 = 一定卡在「不可投放 / 没过出牌 / 没过金牌」三者之一**，逐环节自查。

**广告跑不动的三大直接原因：**

| 原因 | 机制 | 对策 |
|---|---|---|
| **ECPM 过低** | 进不了竞拍期，竞争失败 | 提素材质量、提出价合理性 |
| **CTR / CVR 低 + 素材差** | 系统判定低质，不推荐 | 换/优化素材（见 §6） |
| **出价策略不当（RY 设太高）** | 触发赔付保护→系统主动压低出价→更难跑量 | **降 RY**（见 §3、§4） |

## 2. 投放前置 3 步（搭建 SOP）

> 入口：**TTAM → GMV Max**（创建计划时**仅需设 Campaign 层**，比手动 VSA 简单）。详细创编路径见 `付费广告GMVMax.md` §2.4。

1. **准备动作**：账户**绑定 + 充值**。
   - ⚠️ **账户余额建议 = 7× 日预算**，最少 **2–3× 日预算**。讲师说法：「系统倾向给有钱的账户分配流量」。
2. **搭建广告**：进 GMV Max 入口 → 点「创建 GMV Max 广告」。
3. **选品 + 设参数**（见 §3）。

**选品方式（二选一）：**
- **推广全店**：全面铺，吸潜在客。
- **推广特定商品**：精准推重点品。
- ✅ **关键经验**：**一个计划最好放 ≥2 个商品同时跑**（「像多投几份简历」），增加覆盖和竞争力，跑量更顺。

## 3. 关键参数：RY（ROAS 目标）与预算

### RY（投产比目标）设置

- **新手/无销量积累**：用**系统推荐值**，或直接投 **RY=1** 起步。
- **店铺已有自然流销量积累**：可按商品实际 RY 值出价。
- ⚠️ **核心避坑（与赔付机制联动）**：**RY 设太高 → 系统为避免赔付会主动压低出价 → 跑量极难**。这是新手最常见的「投不出去」根因之一。
- 数据稳定后再逐步上调（调整幅度见 §4 纪律）。

### 预算计算公式（经验值）

```
日预算 ≈ (20~30 个订单的 GMV) ÷ RY目标 ÷ 0.6
```

- **举例**：目标成交额 600、RY 目标=2 → 合理日预算 ≈ 500（量级，非精确）。
- **触发赔付/放量信号**：若日消耗到 100 美元、单日出单 ≥20 单且触发赔付条件 → **预算上浮 40%** 保计划充分跑量。
- ⚠️ **创意预算平均分配的坑**：系统会把预算平摊给创意，**低消耗素材可能占用 >60% 预算**，导致**头部计划组被流控下线**、影响跑量 → 要主动清理低效素材（见 §6）。

### 不同时期的 RY/预算策略

- **平销期**：正常 RY。
- **大促期 + 大促结束后一周**：流量竞争激烈、出价水涨船高 → **必须下调 RY（投产比目标）**，否则「RY 维持平销值 → 广告极可能跑不出去」；同步按参考值调预算。

## 4. ⭐ 全周期优化 SOP（一条计划：冷启动→扩量→衰退）

> 这是本篇最核心、最「可照做」的部分。总纪律写在最前面。

### 4.0 最高纪律：不要频繁调整（重要的事说三遍）

- 开关计划、调 RY、调预算 → **一天最多 1 次，顶多 2 次；不是每天都要动**。
- **每天盯，但不是每天操作**。GMV Max 学习期长（**表面 1 条计划，背后有 AI 优化师在搭多组广告**），**至少 3 天才学得完**，比手动广告周期更长。
- ❌ 反面教材：按手动广告习惯「今天不出单就关、明天新建、后天又关」——**完全错误**，会让模型永远学不完。

### 4.1 冷启动期（约 7 天，非固定值）

**前 3 天 —— 只看「消耗 + 转化量」，不要看 RY：**
- 前 3 天成本波动**完全正常**（系统还没找到人群，在学习）。
- **保消耗是第一要务**：先有消耗、先积累数据，才谈优化。过早看 RY 会让计划没法消耗。
- **常见问题：消耗 < 50%。** 三种应对（按温和度排序）：
  1. **增加优质素材**；
  2. **手动把素材投放出去**（强制起量，见 §5 手动抓取/加热）；
  3. **下调 RY**：**单次 ≤20%，一天 ≤1 次**。
- 3 天后大概率跑出好品（如已有 ~20 个转化），看成本是否逐步下行、RY 是否回升。

**后 4 天 —— 看「消耗 + RY」，目标累计 50 个转化：**
- **转化达 50 个 ≈ 算法学好了**，RY 开始稳定。
- 看优化后的 RY 能否到**目标的 80%**：
  - **达到 80% → 可以考虑放量**（样本越多学得越好）。
  - **不达标的诊断树**：
    - **流速过快导致 RY 低** → **先调预算别动 RY**：把预算「水龙头拧紧一点」，看流速放慢后 RY 是否回来。
    - **流速正常但 RY 偏低** → 正常情况，再观察。
    - **RY < 目标的 50%（异常）** → **小幅上调 RY，单次 ≤5%**，看能否拉回。
    - 调完仍不行 → **回到素材/品**：核心 RY 由素材和品决定。查素材（前端 6 秒完播率、CTR 好但转化差 → 查后端 listing/落地页）；素材没问题 → **就是品的问题**。
- **品 vs 素材的根因判断**（讲师反复强调「先品再素材」）：
  - 冷启动内转化够 50 但 RY 始终不达标 → **可能品的利润不够**。
  - 整个冷启动转化都突破不了 50 → **品本身竞争力小、感兴趣的人少**。
  - 「品好的话，素材随手一拍都能出单」。
- 跑满 7 天、累计 50 转化后，消耗和 RY 趋稳 → **冷启动通过**。

### 4.2 稳定放量/扩量期（约 2 周）

- **爆品单独建计划**扩量；腰部品按自己规划设 RY 和预算目标。
- **加预算：单次 +20%~50%，不超过 50%**（>50% 会让人群模型跑偏）。
- 除加预算外，每天还要做**创意素材迭代**、保证每周素材上新量（见 §6）。

**加预算后按「消耗 + RY」走向分情况处理：**

| 现象 | 判断 | 动作 |
|---|---|---|
| 消耗↑ 且 RY↑ | 最乐观 | 进计划层找好计划→好单品→好素材，**稳住高消耗素材**；某素材消耗下滑可用 **$10 小预算 boost 救 1–2 天**，按其脚本方向迭代裂变；**消耗能稳定跑到日预算 70% → 再按 ~30% 加预算放量** |
| 消耗↓（先别动出价）| 多为「空号素材」拖累 | 进计划找**曝光高但不出单**的空号素材→**先移除**→看 RY 是否回升；没回升再小幅提 RY 找高 RY 人群 |
| 移除空号后消耗↓且 RY 没回 | 可能误删了「**跑单担当**」素材 | 把它**加回来**——这类素材曝光大、用户隔 1–2 天再回来买（贡献归它但未被广告系统计入），一关就断流 |
| 消耗↓ 且 RY↓（联动下滑）| 曝光不够→出单也少 | 当天先**下调 RY 放量保销量**；**连续 2 天都降 → 模型跑偏 → 重新搭计划**重学人群 |
| 消耗↓ 但 RY↑ | 头部素材在衰退 | 找衰退头部素材（看消耗/曝光/CTR/播放率是否下滑）→**开启加热**观察 2 天能否回升→不行再下调 RY 看消耗→仍不行 → **重新搭计划**（模型有问题） |

### 4.3 衰退期判断 + 检查清单

一条计划一般跑 2 周稳定、**可能撑到 1 个月**后进入衰退。

**判定衰退**：近 3 天或近 7 天的 **RY 和消耗下浮 >30%**；或近 7 天持续波动但大趋势向下。

**衰退检查清单（排除外因再下结论）：**
1. 最近是否**频繁互相操作**影响了学习？
2. 是否有**素材停投了**？
3. 看**大盘**：是否有对手出高价导致竞争变激烈？
4. 以上都没有 → 计划**真衰退** → 尝试重新搭建激活、重跑模型；拉不起来 → **品的生命周期到了**。

> 免责：广告无绝对标准答案，**每个品/每个店跑法不同**，以上是不熟 GMV Max 卖家的初始参考，需结合自己实操优化。

## 5. ⭐ 自动赔付机制（本篇独有，入门篇未覆盖）

- **GMV Max 有自动赔付机制**：**ROAS 下滑但满足赔付条件时，系统会自动赔付**（按时段/计划，具体规则以后台为准）。
- **这正是「不要频繁调整」的底层原因**：
  - 一方面可**耐心等系统赔付**（不用每天慌着调）；
  - 一方面等待期就是**让算法自动学习**的时间。
- ⚠️ **与 RY 的因果链（最重要）**：**RY 设太高 → 系统为规避赔付主动压低你的出价 → 你反而跑不动**。所以高 RY 不是「更安全」，而是「更跑不出去」。
- ⚠️ 具体赔付条件/比例/结算方式**讲稿未给数字**，**务必以商家中心当期赔付规则页为准**。

## 6. ⭐ 素材抓取规则 + 素材保鲜（实操重点）

### 6.1 自动抓取 vs 手动抓取

| 维度 | 自动抓取（推荐默认） | 手动抓取 |
|---|---|---|
| 优势 | 省时省力、**源源不断新素材自动进入** | **可控性强**，按策略/产品/人群精准筛 |
| 劣势 | **学习期约 5 天**（系统收集→分析→优化才达最佳） | 需人工挑 |
| 数量限制 | — | **最多选 50 条视频** |
| 适用 | 追求高效便利 | 时间紧、要快速调素材方向 |

- ✅ 建议：**默认用自动抓取**（省工时）；同时**可对没挂车的链接手动「强挂车」**。

### 6.2 自动抓取的来源范围（哪些视频会被自动拿来当素材）

可自动获取**带对应商品锚点（描点/挂车）链接、且已授权**的视频，来源包括：
- 店铺产品的 **TikTok 账号视频**
- **TikTok 企业号**视频
- **Seller Center 中可用的 TikTok 账号视频**
- 投放广告账户 **TTAM 素材库**的视频
- **Spark Ads（达人授权帖）**
- **已开白（开放授权）商家可用的达人/UGC 视频**
- ❌ **限制**：**来自不同店铺商品锚点的多链接视频无法使用**。

### 6.3 素材结构规则（避坑）

- **一条链接只能用于一条计划**（保独立性 + 数据准确）。
- ✅ **达人素材 与 自营素材拆成 A/B 两条链接、分别建计划**——两类成本不同，分开才能各设 RY、精准投放；也**避免自营素材被达人素材挤压**。

### 6.4 素材保鲜与储备（来自课程14，提效策略）

- 平台「**喜新厌旧**」周期约 **7 天** → **每周新增 ≥10 条素材**。
- 重点做**开箱 / 对比 / 痛点-解决方案**类视频。
- **及时清理表现差的素材**，让预算集中到高效素材（呼应 §3 的「创意预算平摊」坑）。
- 单个广告理想储备 **50–200 条混合素材**，形式多样：产品展示、口播讲解、用户实拍、场景植入；**重点拍真实使用场景**（上班族穿搭、宝妈试穿童装等）。
- ⚠️ **素材上新不要每天零散上**——模型需要时间，**最好一周集中上新 2 次**。

## 7. ⭐ 素材状态流转 + 创意加热（creative boost）

### 7.1 素材状态流转（看懂后台标签）

```
授权 → 排队（学习中）→ 学习 → 投放
                              ├─ 学不出来 → 手动移除 / 已删除
                              └─ 异常 → 不可用（被删/不合规/账户封禁）
```

**6 种状态速记**：排队学习中、投放中、建议授权、加热中、已移除/空号、不可用。

- GMV Max **优先保 RY 目标**，给新创意探索的预算很少 → 多数素材会**排队**。
- 系统判断素材能否进入投放：**该素材在你 RY 下花费 > $1** → 值得投放；**< $1 → 投不出去**。
- **空号素材**（已移除/无效）不要再进推广；**未授权**的需自己补授权。

### 7.2 创意加热 creative boost（让潜力素材快速起量）

- **潜力素材定义**：**ROAS（均微）降序 Top 且上传日期较新**。
- 对排队/学习中/未投放但质量不错的素材，用**小预算加热**，**保送至「金牌」**（跳过出牌金牌排队）。
- **建议完整加热 2 天**；**3 天后**：
  - 变「投放中」→ 加热成功，且说明素材还有优化空间；
  - 仍没投放 → 说明素材类型判断没错，但**素材本身要优化**。

## 8. 大促要不要开 GMV Max？（开启逻辑）

- **大促日历**：Double Day（2.2/3.3/4.4…双11/双12）+ **Payday 发薪日大促**（东南亚人「没存钱习惯、发薪即消费」，平台专门配对的活动）。**活动开启后广告自动参与配对。**
- **开启后系统默认**：**预算 +50%、RY 目标 −10%**，并在大促日**优先最大化 GMV、捕捉增量**（甚至进一步降条件拿更多 GMV）。
- ✅ **建议开**：想**突破量级**的成熟计划/客户。
- ❌ **不建议开**：**新店、或 RY 跑得低的商家**——「预算+50%、RY-10%」会让经营不佳的店**雪上加霜**。

## 9. 好素材标准（脚本框架，落地可抄）

- **形式优先级**：**达人视频 > 自拍视频**（达人自带流量+说服力）。
- **脚本结构 = Hook–痛点–CTA**：
  - **开头 3 秒**抓注意力 + 明确受众（让人想继续看）；
  - 中段：**场景化应用 + 产品亮点 + 用户证言**；
  - **CTA**：给优惠信息、引导点击、制造紧迫感。
- **原生感**：看起来**不像广告**更易被接受。
- **时长 20–45 秒**（符合快节奏观看习惯）。
- **素材创作 4 板块**：①展示场景（真实使用/前后对比）②促进行动（限时优惠）③强化信任（达人推荐）④锁定人群（点明目标受众痛点）。
- **AB 测试方向**：同商品不同脚本结构 / 同脚本不同出镜人物 → 对比找最佳组合。
- **素材库管理**：达人/自营/混剪分类存，标注 **CTR / CVR / GMV 贡献** 性能数据，供后续筛选与策略决策。
- **找优质素材的工具**：讲师提到「**出海象**」「**Atria（actio?，转写不清）**」等平台作为素材灵感库（工具名以实际为准）。

## 10. GMV Max 提效的三个抓手（课程14 汇总，与上文呼应）

> 课程14 把「效果不佳」归为三因、「提效」给三招，本质都被 §3–§7 覆盖，这里只做索引：

| 效果不佳的原因 | 对应提效招 | 详见 |
|---|---|---|
| 素材供给不足/质量差（少量混剪→判低质限流）| 定期更新优化素材、保鲜 | §6.4、§9 |
| 选品错误致冷启动失败（全新上市品/衰退品无转化路径）| 科学选品：优选**腰部品**（日均自然单稳）、避限制类目、避衰退品 | §4.1 品根因 |
| 参数设置不当（RY/预算偏离合理区间）| 合理 RY + 预算 + **正向调参、一次只动一个、一天≤2 次、调后观察 48h** | §3、§4.0 |

## 11. 已知短期挑战（设预期）

- **互斥规则**：同一商品 GMV Max 与 VSA/PSA 二选一；**LSA 不能同时开单品推广和整店推广**（详见 `付费广告GMVMax.md` §4）。
- **手动调控空间被压缩**：全自动投放下，商家**无法手动调受众定位、竞价策略**等细节。
- **品/素材不行→成本飙升**：GMV Max 是放大器，放大好的也放大坏的。
