# 商城自然流量（Shopping Center / 商品卡 · 官方自然流量）

> 所属板块：05-流量增长
> 主题：什么是 TikTok Shop「商城」、商品卡如何获得**免费的官方推荐/搜索自然流量**，以及用「数据罗盘（Data Compass）」诊断哪些商品没被推荐、怎么优化标题/详情/服务分把流量做起来。

## 0. 一句话理解

「商城」是 TikTok Shop 在内容流（兴趣电商）之外开辟的**货架式电商入口**——用户**带着明确购物意图**进来搜索/逛商城。商品卡只要**满足平台标准就会被自动免费推荐**，不用投流、不用绑直播。核心动作是：**用数据罗盘的「推荐渠道诊断」逐周扫描 → 找出没被推荐的商品 → 按三大维度（服务分、商品信息质量+库存、活跃经营）补齐 → 优化标题/详情吸引精准搜索流量 → 报名商城活动放大曝光。**

---

## 1. 商城是什么 / 有哪些流量入口

商城为店铺带来**区别于兴趣电商的「垂直电商意图流量」**（用户主动找货，转化意图强）。入口集合包含：

| 入口 | 含义 | 商家可做的优化 |
|---|---|---|
| **搜索 Search** | 抖音综合搜索 + 商城搜索 | 用热搜词优化标题/商品信息/内容标题，针对性补内容和货品供给 |
| **商城推荐 Recommend** | 商详页「猜你喜欢」、购后页「猜你喜欢」、商城推荐页、任务页、榜单页 | 按「曝光→成交」转化效率优化商品信息、保上架状态、稳库存与价格 |
| **活动频道 Promotion** | 秒杀等活动频道 | 报名活动拿活动页流量 |
| **橱窗** | 主要指**达人橱窗** | 引导达人在橱窗展示商品（→ 见 `达人联盟.md`） |
| **店铺 Store** | 商家店铺、企业号 | 按用户画像/动线/商品偏好做店铺装修，沉淀私域 |
| **其他** | 消息、分享、购物车等 | — |

> 本笔记重点是**搜索 + 商城推荐**这两条**免费自然流量**渠道（橱窗/达人见 `达人联盟.md`，活动见 `06-活动大促`）。

## 2. 商品卡「被推荐」会出现在哪些位置

商品卡（非短视频/直播形态）的推荐，本质是类似 TikTok「为你推荐」的**个性化货架推荐**。用户购物链路中会出现在：

- **Feed**：进入商城后向下滚动 → 商城推荐页。
- **Order Center**：进入「订单」页面向下滚动 → 推荐区域。
- **Product Detail Page → You may also like**：商详页滚到「Description」下方。
- **Product Detail Page → More from this shop**：商详页滚到「评论」下方。

> 含义：**只要一个商品具备「推荐资格」，它就会自动在这些位置免费曝光**。所以提升商城自然流量 ≈ **让尽可能多的 SKU 进入「可推荐」状态**。

## 3. 核心工具：数据罗盘（Data Compass / 商城分析）

> 建议**每周固定扫一次**「推荐渠道诊断」。〔可信度：高，两份材料都强调常规巡检〕

### 3.1 看整体商城生意盘 —— 商城核心数据
- 路径：`Seller Center → Data Compass（数据罗盘）→ Shopping Center Analytics（商城分析）`
- 指标：商城成交金额/订单数/件数、商品点击人数、成交人数、**点击-成交转化率**、成交客单价、**曝光-点击转化率**、曝光人数、退款金额/订单数。
- 用法：看周期内波动趋势，对应自己近期动作判断有效性；点【对比 / Compare / Trend Comparison】选 2 个指标对照（例：商品曝光 vs 曝光-点击转化率），定位是转化掉了还是曝光掉了。

### 3.2 看分渠道表现 —— 渠道成交转化
- 看搜索/商城推荐/店铺/活动频道/橱窗各渠道的**流量转化漏斗**（曝光→点击→加购→成交→退款），逐层看留存/流失。
  - **曝光→点击**转化率低 → 优化主图、标题、描述。
  - **点击→成交**转化率低 → 优化价格、卖点、优惠券等营销工具。
- 看秒杀单独表现：点【Promotion】→ 渠道类型选【秒杀】。

### 3.3 看渠道商品排行
- 看各渠道 TOP 热卖商品（可按类目，可按直播/视频/橱窗内容形式）。
- 用法：① 活动频道里的 TOP 商品 → 增加它参加商城活动的频率；② 搜索渠道里表现差的商品 → 对标 TOP 商品的图/标题去改。

### 3.4 看「Recommend 推荐」渠道专项（推荐流量诊断）
- 路径：`Seller Center → Data Compass → Shopping Center → Channel Diagnosis（渠道诊断）` →「推荐 / Recommend」。
- 步骤：① 进入推荐渠道诊断 → ② 切换时间对比 → ③ 看推荐渠道产生了多少 GMV/曝光（可切 Compare 选 UV 看曝光趋势）。
- **价值**：它会**自动汇总相关指标、识别哪些商品因为什么原因没被推荐**，并给出对应优化点；做完优化后回到同页点「查看详情」可验证**新增多少商品开始被推荐、各拿到多少推荐曝光**。

## 4. 让商品「入池被推荐」的三大维度（核心 SOP）

> 来自《推荐入池诊断》——这是本主题**最可执行的清单**。目标：让大部分 SKU 满足平台标准、自动获得推荐自然流量。逐项在数据罗盘「推荐渠道诊断」里对照修。

### 维度 ①：满足基本服务体验指标（店铺服务分达标）
- **SFCR**（商家取消/履约相关分）：目标降到 **< 5%**。优化点：检查**商品价格是否设置正确**（价错→取消）。
- **LDR（延迟履约率）**：把 35 天 LDR 降到 **< 5%**。优化点：**给爆品提前筹备库存**；备货充足后 SFCR/LDR 会逐渐改善。
- **NRR（差评率）**：≤2 分（1-2 星）的评论算差评；**带文字或图片的差评才算「有效差评」**。数据罗盘会自动高亮 NRR 过高的单品 → 按评价内容优化商品质量/迭代。

### 维度 ②：提高单商品的信息质量 + 库存保障
- **描述与实物一致**：避免商品与描述不符、避免发错货 → 更新详情页信息。
- **商品描述 ≥ 200 字符或含图片**（含空格和符号都算字符，例：「Keset」算 5 个字）→ 描述太短就扩写到 200+ 或加图。
- **主图清晰完整展示商品主体**：主图放完整主体，细节放子图。
- **库存保障**：售罄尽快补货；缺货可暂时下架；完善供应链/销售预测/与供应商关系/缩短采购周期；**确保关闭「假期模式 / Holiday Mode」**。

### 维度 ③：保持活跃经营（让新品尽快「破 0」）
- **逻辑**：单商品**在任意渠道产生过销量（破 0、动销）后，在商城推荐页会更容易被推荐**。所以上新后要尽快开单。
- 操作：① 新品用**至少 5 次直播 + 3 条短视频**推广（直播中至少挂该新品 5 次、短视频推 3 次）；② 做 SEO 扩搜索渠道；③ 积极找达人合作开单（→ `达人联盟.md`）。

> **入池成功的标志**：推荐渠道诊断页显示「你的商品都已具备获取自然流量推荐的资格」之类的全绿状态。

## 5. ⚠️ 服务指标阈值的口径冲突（重要，需以官方为准）

两份文件给的服务指标**口径不一致**，整理消化时必须留意，签约/做 KPI 前以**马来西亚站后台「推荐渠道诊断」实际显示的达标线为准**：

| 指标 | 《入门手册》 | 《推荐入池诊断》 |
|---|---|---|
| 延迟履约率（LDR） | ≤ **4%**（下单 5 天后才更新为发货状态的占比） | 35 天 LDR 降到 **< 5%** |
| 商家取消率 / SFCR | 商家原因取消率 ≤ **2.5%**（按过去 15-45 天订单） | SFCR 30 天 **< 5%** |
| 差评率 | 过去 60 天 >40 条评价中 >10% 差评、且商品分 <4.0 会被管理 | NRR（≤2 分）有效差评（含文字/图片）需降低 |

> 〔可信度：低-中〕这些数字是教学材料口径，**LDR 4% vs 5%、取消率 2.5% vs 5% 直接矛盾**，且 LDR 的定义（「下单 5 天后才发货」vs「35 天 LDR」）也不同。**不要把任何一个数字当硬规则**。与 `07-客服与店铺健康` 中的店铺健康规则若冲突，以那边/官方后台为准。

## 6. 提升业绩的其他动作（来自《入门手册》）

### 6.1 商品诊断（先定位问题商品）
- 路径：`商家后台 → 数据罗盘 → 商品分析 → 商品诊断`。自动诊断四类问题商品：成交下降、曝光/转化有问题、商品信息质量差、服务体验差。
- **成交下降商品**：鼠标悬停【直播成交下降】看跌了多少、悬停【下降原因】看主因；点【Details】进商品详情定位是哪个达人/直播/视频拖累。先判断是季节性/主推策略调整，还是异常下跌。
- **高流量低转化**：主推品可能是直播讲解过长导致曝光虚高；非主推品则查价格/运费/折扣/详情是否清晰，并对标同行。
- **低流量高转化**：本身转化好 → 后续**加大直播讲解时长 + 视频带货曝光**。
- **信息质量差**：点【优化】跳商品信息页，按诊断标签改标题/图片/详情/类目。
- **高投诉/差评/品退率**：按评价投诉内容优化质量、迭代商品。

### 6.2 标题优化（拿搜索流量的关键）
- 好标题 = **品牌名 + 品类 + 功效 + 特征**，让有需求的用户搜得到。
- 工具路径：`数据罗盘 → 市场分析 → 搜索词分析`。
  - 选热门搜索词 + 目标类目 → 看该类目近期**搜索量最高的关键词** → 把相关词用进自己标题。
  - 在搜索框搜自己标题里的关键词 → 看其搜索量级 → 决定要不要换成「相近但搜索量更高」的词。

### 6.3 商品详情页优化
- 优质标题/图片/描述能同时提升曝光和转化（对照第 4 节维度 ② 的清单逐项改：描述 200+ 字符/含图、主图完整、描述与实物一致）。

### 6.4 价格有竞争力
- **有竞争力 = 不高于你给其他平台的价格**。好价格同时带来更高转化（推动下单）和更高曝光（平台更愿推荐）。〔可信度：中，方向正确；具体比价规则以官方为准〕

### 6.5 报名商城活动拿额外曝光
- 活动天然吸量、平台给活动页更多流量。活动类型：**秒杀 Flash Sale、品牌折扣 Brand、新客推荐 First Order Offer**。
- 报名路径：`Seller Center → Promotions → Campaigns →` 找到商城 Flash Sale / First order offer 报名。
- （活动玩法细节 → `06-活动大促`。）

### 6.6 新品选品方向
1. **销量选品**：把在其他平台已卖得好的品，用有竞争力价格上传到 TTS。
2. **市场趋势选品**：看 TTS 各行业热销品 + 上升趋势的潜力品。
3. **蓝海词选品**：`数据罗盘 → 市场分析 → 搜索词分析 → 蓝海词（Potential Terms）`——找**搜索热度高但供给少**的词，提前布相关商品抢市场。
- （系统化选品 → `02-选品`。）

## 7. 落地动作清单（给自己的 checklist）

- [ ] 每周扫一次 `数据罗盘 → 商城 → 渠道诊断（推荐 Recommend）`，把「未被推荐」的商品按三大维度补齐。
- [ ] 服务分：盯 LDR / 取消率 / 差评率（阈值以马来西亚站后台为准，别照搬本文数字）；爆品提前备库存，确认**没开假期模式**。
- [ ] 每个 SKU：主图完整、描述 ≥200 字符或带图、描述与实物一致、价格不高于其他平台。

- [ ] 标题：用「搜索词分析」的高搜索量词 + 蓝海词重写。
- [ ] 持续报名 Flash Sale / Brand / First Order Offer 拿活动曝光。
- [ ] 优化后回「推荐渠道诊断」点「查看详情」验证新增入池商品数 & 曝光增量。
