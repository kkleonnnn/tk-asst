## 1. 后台联盟设置·逐字段实操（核心增量，来自《达人邀约教程》）

> 入口：商家中心左侧 **联盟** → 点下方「**前往 TK 联盟首页**」→ 跳到**联盟中心**页 → 顶部「**合作**」下有【公开合作】【定向合作】。

### 1.1 公开合作设置·每个字段怎么填

进「公开合作」→ 标签分【已添加的商品】/【未添加】；未加的点「**添加合作**」即可补进。逐字段：

| 字段 | 怎么设 | 说明 / 避坑 |
|---|---|---|
| **自动新增商品**（右上角开关） | 建议**开** | 开后店铺**新发布的商品自动进公开合作**，省去逐个手动添加（SKU 多的店尤其值） |
| **标准佣金率** | 两种选法：①按**商品类目**分别优化；②全店**统一一个费率**。讲师建议 **10%–15%**，按定价定 | 这是达人带货成交后必拿的基础佣金 |
| **店铺广告佣金率** | **建议关掉**（直接关） | ⚠️ **双佣金陷阱**：标准佣金率 + 店铺广告佣金率**两个都开 → 达人会拿两笔佣金**（成本翻倍）。只开标准佣金率即可。且该项需先**设置过店铺广告**才能操作 |
| **买返样品**（样品设置之一） | 可开。设**退款门槛**（讲师举例「**出10单**」「客单价40以上」之类） | 机制：达人**先自费购买样品**；当达人帮你**出够 N 单**（你设的门槛）后，**系统自动把样品费退还给达人**。对商家几乎零样品成本，适合筛真心带货的达人 |
| **提供免费样品** | 可开。设**样品数量**（建议**1~3件，别设多**） | ⚠️ **免费样品由商家自费**。数量是"名额"——设1件，被1个达人申走后其他达人就申不了；设多了成本高 |
| **申请期限** | 选「**永久生效**」或「**已终止**」 | 按活动节奏定 |
| **达人内容类目（内幕）** | **选与自己店铺一致的垂直类目** | ⚠️ 做女装就只找女装类目达人；让其他类目达人推，成功率很低。**垂直最重要** |
| **粉丝数** 门槛 | 越高越好；**别设太低** | — |
| **30天内销量** 门槛 | **别设太低** | 过滤无带货数据的达人 |
| **每个视频播放量** 门槛 | 设一个值 | 满足全部门槛的达人才能申你的样品 |
| **发布率** | **直接选「高」** | ⚠️ **防白嫖关键开关**——只放发布率高的达人申样，避免领样不发 |
| **内容语言** | 选与店铺市场一致（泰国店选**泰语**；马来西亚店选对应语言） | — |

设完点「**提交**」→ 显示「自动添加到联盟设置成功」。

### 1.2 定向合作设置·创建邀请逐字段

> 比 `达人联盟.md` 第2.1节更细的字段级填法。入口：公开/定向页**右上角「邀请合作 / 邀请达人合作」**。

| 字段 | 怎么填 | 说明 |
|---|---|---|
| **邀请名称** | 自定义（可按活动/批次命名） | 便于后台区分多个定向计划 |
| **有效期** | 设达人可接受并推广的截止时间 | — |
| **账号**（栏位） | 没有就**随便输几个数字**即可 | 讲师明示：**带星号 \* 的才是必填项**，没星号的可不管 |
| **邀请文本** | 输入**邀约话术**（用店铺市场对应语言，泰国店用泰语） | 话术模板见 `达人联盟.md` 第4节 |
| **内容形式** | 三选一：**带货 / 带货直播 / 暂无偏好** | 按实际需求选 |
| **添加商品** | 搜商品名 → 点「添加」 | — |
| **佣金费率** | 逐个设 | — |
| **免费样品·审核方式** | ⚠️ **务必选「手动审核」，不要选「自动批准」** | 自动批准 = 达人一申请系统就自动通过；免费样品涉及商家成本，**必须人工把关** |
| **选达人** | ①点「**添加推荐的达人**」勾选；或②搜索框直接输**达人用户名 / 用户ID** 发邀请 | — |

点「**发送**」→「合作邀请发送成功」→ 可点「**查看邀请**」；达人接受后该处显示「**已接受**」，已推广也会有显示。

> **建联方式效果排序（讲师口径）**：**定向合作 > 公开合作**。定向是精准筛出、且达人主动接受邀约 → 合作意图最强、出单可能性更高、效率最高。公开合作范围最广但需在「样品申请处」定期捞符合条件的达人。〔可信度：中〕

## 2. 建联前的"三件套"准备清单（来自《达人建联实操》）

> 与达人沟通前要先备好 3 份资料，建议用表格管理、**及时更新**。

1. **建联表格（达人鉴定表）**——尽量详细，字段建议含：
   - 基础：昵称、联系方式、账号类型
   - **粉丝画像**：年龄分布、性别比例、地域分布（用来判断和你产品目标人群是否匹配）
   - **带货数据**：过去一段时间的带货转化率、平均客单价（评估带货能力强弱）
   - 维护：定期跟进信息变化，保证沟通时数据准确
2. **样品档案**——样品信息要全：批次、颜色、规格；**服装类**还要记面料成分、尺码范围、颜色选择。配套做：
   - **物流跟踪**：实时掌握样品到哪了，及时告知达人发货/预计到达时间；延误第一时间沟通，别耽误达人创作计划
   - **产出记录**：达人收样后的发布时间、形式（视频/图文）、内容质量反馈 → 评估样品利用率
3. **合作脚本 / 手卡**——提前给达人备稿：
   - **脚本结构**：开头（创意引入/抛问题吸睛）→ 中间（详述卖点，可对比竞品突出性价比/性能/续航）→ 结尾（引导购买或关注后续）
   - **手卡**：1~2句话列**核心卖点 + 创作方向**（例：卖点=高效清洁/静音设计，方向=家庭清洁场景展示）。简洁明了，让达人快速抓重点

## 3. 寄样判断·实操铁律（核心增量）

> 寄样前判断「**该不该给这个达人寄样**」。后台路径：店铺后台 → **联盟带货** → 查看「**样品申请**」情况。

### 3.1 可以寄样的三条件（讲师口径）
1. **履约率 = 100%**（硬指标）
2. **销售额**：无硬性标准，**越高越好**
3. 点进**达人主页看视频数据**，数据 OK 即可寄

配合复盘三看：①**粉丝画像**对不对应你的客群；②**沟通中带货意愿**强度（积极 vs 敷衍）；③**履约率100% + 有带货数据**最佳。

### 3.2 ⚠️ 拒绝寄样的 6 类达人（防白嫖/防翻车清单）
1. 视频**平均播放量不高**的
2. **不挂车**的纯视频播放量也不高的
3. **近期挂车视频只有一两个**的
4. 账号内容与你推广产品**差距太大**的
5. **发布率低、但历史申样次数很多**的 → **很可能白嫖样品**
6. 视频**过于低俗/质量差/擦边/有风险提示**的

### 3.3 寄样操作要点
- **尽量走 TikTok（剔透）系统寄样**，便于追踪物流。〔注：讲稿多处把样品系统说成"剔透/CIV"，疑为语音转写误差，应指 TikTok 联盟内置的样品/达人系统；以后台实际为准〕
- **免费样品成本由商家自担**；公开合作里设好申请标准和库存数量。
- ⚠️ **手动审核有时限：7 天**。免费样品设手动审核后，**超过7天就无法操作**该申请 → 要**勤点"样品申请"**核查筛选，别错过优质达人。
- ⚠️ **寄样前先谈好价格+佣金**：避免样品到了，达人嫌价格/佣金不合适不愿播，白寄。
- **省运费**：一个达人有多个适配品时，引导其**一次申请多件、一个包裹寄**（详见 `达人联盟.md` 第4节话术）。

### 3.4 达人主页能看什么（辅助判断）
点达人头像进主页可看：**主营类目、近30天成交件数、主成交类目、各指标流量趋势、带货前3名视频**。例：某达人30天成交300多件、主类目时尚配饰 → 若你做女装则高度匹配，可考虑寄样。

## 4. 样品申请·状态机与跟进节奏（增量）

### 4.1 样品申请跟踪状态机（5态）
```
待审核 → 待发货 → 已发货 → 内容输出中 → 已完成
```
- 看到「**已发货 / 达人已收到**」时 → 主动问达人「是否收到样品」。
- 确认收到 → **发产品橱窗链接 + 约定发视频时间**。
- 过 1~4 天还没发 → 跟达人确认"视频素材什么时候能给"。**这步需反复催促**。

### 4.2 跟进的"频率铁律"（细化 `达人联盟.md` 的5次上限）
- **达人消息**：发第1条不回 → **还可再发4条（合计最多5条）** → 之后**必须隔1个月**才能再给该达人发消息。把握频率与时机，别在上限内浪费、也别超限。
- **中期**：建立稳定沟通节奏，避免无意义问号/反复啰嗦询问，给达人留创作时间；及时寄样、跟物流、处理异常、提醒完成发布。
- **尊重达人作息**：了解其工作/生活规律，尽量在工作时间沟通，别打扰。

### 4.3 多颜色/多样品达人的"含糊"话术（增量）
对一次想要多个/多色样品的达人，原则**不得罪、不引起不适**：
> 先给你寄一个样品，等后续"领样限制"解除后，我再给你寄其他的。

（先含糊过去，控制单达人样品成本。）

### 4.4 通过样品申请后的同步动作
达人通过样品申请后，**及时把样品的订单号同步给达人**；对带货能力强的达人，话术里可点明**商品佣金 + 最终成交价**——价格有竞争力达人更愿带。

## 5. 达人建联全流程（速览·已在批次1详述，此处只列骨架）

> 完整方法论见 `达人联盟.md` 第3节。本课流程一致，骨架如下，便于对照：

**选品 → 筛选达人 → 联系 → 建达人库 → 寄样品 → 推动履约 → 数据复盘 → 维护**

- 复盘分叉：数据 OK → 继续深化合作；数据不 OK → **委婉提醒达人调整内容** → 调整后仍不佳 → 本次合作后**不再续约**。
- 筛选达人 4 维度（与批次1一致）：①**达人广场 + CRM工具**使用；②**粉丝量**（非越多越好，按品类定）；③**互动率**（点赞/评论/转发率）；④**内容质量**（专业性、创意性）。
- 前期注意：**人货匹配度**（别只看热度/粉丝量）；**数据隐私合规**——⚠️ 严禁用未授权第三方工具非法抓取达人粉丝/带货数据，否则陷品牌于法律纠纷。
- 后期维护：发布表现跟进（曝光/赞/评/转发/转化率 + 粉丝反馈）；定期感谢+节日问候；挖掘新合作（联名产品、定制礼盒、按粉丝需求开新品线）。

## 6. 本文增量要点速查（给赶时间的人）

1. **双佣金陷阱**：公开合作里**关掉"店铺广告佣金率"**，只留标准佣金率，否则达人拿两笔佣金。
2. **买返样品**：达人先自费买样，**出够N单后系统自动退样品费** → 近零成本筛真心达人。
3. **免费样品数量是名额**，设1~3件即可；**审核务必选"手动审核"**，免费样品**手动审核时限7天**，要勤捞。
4. **发布率选"高"** = 公开合作防白嫖最有效的门槛开关。
5. **拒绝寄样6类**：低播放/不挂车低播放/近期挂车仅1~2条/内容跨类目/发布率低但申样多/低俗擦边。
6. **跟进频率铁律**：达人消息最多发5条（1+4），不回隔1个月再发。
7. **样品状态机**：待审核→待发货→已发货→内容输出中→已完成；"已发货"后主动问收货+发橱窗链接+约发布时间。
8. **建联三件套**：建联表格（含粉丝画像+带货数据）/ 样品档案（含物流跟踪+产出记录）/ 脚本手卡。
9. **建联效果排序**：定向合作 > 公开合作。
10. **寄样前先谈好价格+佣金**，免得白寄。
