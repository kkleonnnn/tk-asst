## 0. 三条核心成交公式（贯穿全册）

理解流量从哪来、如何拆解，是后面所有动作的纲。

### 0.1 商城/全域多元成交公式（直播篇 P4）

```
成交(GMV) = 内容供给(PV) × 流量效率 × 转化效率(GPM)
```

| 拆解维度 | 含义 | 三类/三因子 |
|---|---|---|
| **内容供给（PV，曝光）** | 不同成交体裁分层 | 直播 GMV / 短视频 GMV / 货架(橱窗) GMV |
| **流量效率** | 三类流量来源 | 推荐流量(自然) / 付费流量(广告投流) / 私域流量(粉丝访问) |
| **转化效率（GPM，每千次曝光产生的销售额）** | 3 个驱动因子 | CTR(点击率) / C_O(转化率) / AOV(客单价) |

**经营根本（公式背后的隐性关注点）**：优质商品 · 经营合规 · 按时履约 · 服务体验。
→ 这四点是"看不见但同样重要"的底层，不达标会被平台限流/治理。

### 0.2 直播成交公式（直播篇 P44）

```
直播GMV = 直播时长 × 直播流量 × 直播转化
```

| 驱动 | 关键指标（监控这些） |
|---|---|
| **直播时长** | 日均开播时长、开播场次、**开播时间段（选目标市场黄金时段）** |
| **直播流量** | 内容价值：看播时长、关注率、评论率、直播间点击率；转化效率：GPM、看播下单率、点击-支付转化率、购物车点击率；服务体验：好评率、差评率、延迟履约率；投放：广告出价；私域：粉丝量、复购率、复访率 |
| **直播转化** | 商品点击率、点击购买率、客单价（多指标需协同调整，保证 GPM 全局最优） |

### 0.3 短视频成交驱动（短视频篇 P36）

- **成交驱动**：挂车短视频数量 + 稳定高频更新（频次/发布时间）是提升短视频 GMV 的关键。
- **关键指标链**：短视频流量(播放/观看) → 用户价值(时长、留存、粉丝积累、二次传播、互动点赞留言) → 短视频转化(视频点击率、购物车点击率、支付率、客单价)。

## 1. 商城篇（货架电商）

> ⚠️ **对马来西亚跨境店的关键提示**：本册 2023 年 8 月发布时，明确写"**马来西亚市场即将开放商城入驻，敬请期待**"。也就是说当时马来站商城尚未对卖家开放。**今天（2026）马来站商城是否已开放、入口形态、频道规则务必以卖家中心实时为准。** 本册的频道治理门槛、秒杀场次等具体数字主要以菲律宾/泰国为样本。

### 1.1 商城是什么 / 入口

- 商城 = 站内**货架电商**解决方案：以商品分类、中心化呈现为核心，承接用户主动购物需求。
- 用户购物链路（需求由弱到强）：发现(刷 TikTok) → 逛街(逛商城) → 搜索 → 复购。
- 触点：Tab 入口 / 达人橱窗 / 活动频道区 / 商城首页推荐 / 商品详情页推荐 / 首页搜索 / 商城搜索 / 店铺入口。
- 入口位置：**马来西亚商城入口位于导航栏顶部**；其他东南亚国家位于"首页"右侧。
- 商城同时最多展示 **1000 件**在线产品（⚠️ 旧数据，以后台为准）。

### 1.2 商城频道（流量来源）

- 商品搜索、购物车、**First Order Deals 新人专区**（仅商城新客可见，促首单）、Flash Sale 秒杀、商品分类推荐、商品卡 / 直播卡 / 短视频卡。

### 1.3 选品 SOP

1. **销量选品**：把在其他平台已有较好销量的商品，以**有竞争力价格**搬到 TTS。
2. **"机会中心"选品**：卖家中心 > 商品 > 机会中心 → 查看商品推荐，选适合的上传。
3. 参考官方文档：《TikTok Shop 商家 SEO 优化指导手册》。

### 1.4 竞争力价格

- 定义：**不高于**卖家给其他平台的价格。
- 收益：更好价格 → 更高转化率 + 更多平台流量推荐（双向正循环）。

### 1.5 商品信息质量优化（核心 SOP）

**入口**：商家后台 → 数据罗盘 → 商品分析 → **商品诊断模块**（自动诊断成交下降 / 曝光转化异常 / 信息质量问题 / 服务体验问题的商品）。

四类问题商品的处理：
- **a 成交下降商品**：定期看诊断，区分是季节/策略变化还是异常下跌；鼠标悬停【成交下降】看下降金额、悬停【下降原因】看主因；点【Details】定位是哪个达人/直播/视频渠道导致。
- **b 高流量低转化**：主推品可能是讲解时长过长导致；非主推品要查价格/运费/折扣力度/详情是否清晰，并与同行对比。
- **b 低流量高转化**：本身转化效率高，应**增加**直播讲解时长 + 视频带货曝光。
- **c 信息质量问题**：点【优化】跳商品信息页，按问题标签改标题/图片/详情/类目。
- **d 高投诉/差评/品退率**：按评价投诉内容做商品迭代。

**标题优化**：
- 参考**热搜词、飙升词**榜单做标题覆盖。
- 参考**蓝海词**（平台搜索量大但供给相对少）补充商品 = 缺货词选品。

### 1.6 商品详情页优化清单（高价值，可直接照做）

| 项 | 合格 / 优质标准 |
|---|---|
| 标题 | 字符数（含空格）**>10 个**，无重复堆砌 |
| 首图 | 商品主体明确，建议像素 **1280×1280px**，**白底图**，非拼接图 |
| 主图清晰度 | 比例 **1:1**，像素 **>600×600px**，**不含中文**（品牌 Logo 除外） |
| 主图数量 | **≥3 张**（优质 >3 张）且不重复 |
| 描述 | **分段 + 图文混排**，**无中文信息** |
| 视频 | 上传视频（有视频优于无） |
| 销售属性 | 配图，且多 SKU 属性值的配图不重复 |
| 商品属性 | 填写数量 **≥5 个** |
| 尺码（服饰鞋靴） | 含尺码说明，主图 / 描述 / Size Chart 入口均可 |

### 1.7 商品质量 / 服务（治理红线）⚠️ 以官方最新规则为准

- **差评率**：用户物流妥投后 **60 天内**可评价订单，1 星或 2 星算差评。详见官方《跨境电商规则手册》Vol.1 差评率篇。
- **延迟履约率**：商家需 **48H 内**与物流服务商完成交接（平台揽收）/ **120H 内**成功入库（商家送入仓）。详见规则手册 Vol.2 延迟履约率篇。

### 1.8 参与商城频道活动（秒杀 / 新人频道）

**为什么提报**：入选商品不仅出现在活动页，搜索、推荐也获更多流量；且享专属补贴：
- **秒杀频道补贴最高 20%**，**新人频道补贴最高 40%**（硬通货除外）。⚠️ 旧数据。
- 补贴同样作用于搜索页、推荐页；新人模块与秒杀模块可同时参加。
- 活动提报价 + 平台补贴**仅在商城内生效**，不影响商城外短视频/直播间/达人佣金的售卖价格。

**频道治理准入门槛**（⚠️ 2022-2023 东南亚通用口径，必须以后台实时门槛为准）：

| 指标 | 阈值（统计窗口） |
|---|---|
| 累积销量 | >20（lifetime，无具体周期） |
| 商责取消率 Seller Fault Cancellation | <2.5%（T45–T15） |
| 延迟履约率 Late Dispatch | <4%（T45–T15） |
| 商责退货率 Seller Fault Return | <2.5%（T45–T15） |
| 瑕疵退货率 Defective Return | <2.7%（T60–T30） |
| 差评率 Negative Review | <6%（T60–T1） |
| 投诉率 Complaint | <0.3%（T60–T30） |

**秒杀频道节奏**：菲律宾每周二/周五更新商品池（提报截止在周二/周五节点前），每天 1 场（0:00–23:59）；泰国每天 2 场（11:00/19:00），相邻两场产品不可重复。**新人频道**每周更新，符合标准则长期上线。
→ 参考官方《平台营销活动商品报名商家端操作手册》。

## 2. 直播篇

> 适用性：方法论东南亚通用。**强调"海外直播属于探索，保持单一主线，不要把逻辑复杂化"**——这是和国内直播的核心差异。

### 2.1 直播前须知

**权限**：
- 自动获取：电商权限（店铺开通后）、直播权限（店铺开通后绑定账号）。
- 需手动申请：**OBS（电脑开播）权限**。OBS = Open Broadcaster Software，免费推流软件，可推高清画面、叠加文字/图片素材。使用步骤：到 obsproject.com 下载 → 设置里把"Cast on PC/Mac"（Stream Key）→ 输出选自定义 Service Choose "Custom"，填推流地址 + 串流密钥。

**设备选择（手机直播 vs 电脑直播）**⚠️ 价格为旧参考（人民币）：

| 方式 | 适用 | 关键设备与参考价 |
|---|---|---|
| **手机直播**（更灵活） | 户外/移动场景、应用手机直播玩法 | 桌椅 ¥100–1000；美颜灯 ¥100–3000；手机支架 ¥20–100 |
| **电脑直播**（更高清，需 OBS） | 高画质宽屏、200m+ 距离、屏幕共享叠加图文 | 桌椅 ¥100–1000；美颜灯 ¥100–3000；高配电脑 ¥5000–10000；摄像头 ¥500–2000；球形灯/直播灯 ¥1500–5000；摄像头支架 ¥20–100 |

**直播间产品功能**（须知熟练度）：
- **添加商品**：APP 可加 50 件、电脑端无明显上限；直播商品可组合活动券；管理可置顶 PIN 商品讲解。
- **直播管理员/场控（moderators）**：可设最多 **20 个** moderators（数值可配置），需为互粉好友或直播间用户，可禁言/拉黑、管理屏蔽闭麦列表。
- **评论与分享**：维护评论区氛围，可设关键词过滤（添加最多 **200 个**字词过滤）；可鼓励一键转发到其他社交平台。
- **直播互动 Q&A**：选题问答互动，可关闭停止；不恰当问题可举报/删除或闭麦屏蔽用户。
- **暂停和重播**：主播直播中可主动暂停 **3 次，每次仅 5 分钟**。
- **特殊效果**（主要针对手机直播）：镜头翻转/镜像、美颜滤镜、特效（绿幕等）、声音特效。
- **主播多麦 & 多人聊天**：Co-Host 连麦邀其他主播一起卖货；Multi-LIVE 多人聊天最多邀 **3 位**嘉宾。
- **宝盒（打赏/送礼物）**：通过"更多工具 → 宝盒"设置金币数量，可参与领取用户数、倒计时期限。
- **预告推广**：LIVE Event（直播活动，需用户关注/管理权限推分享）+ LIVE Countdown Sticker（直播倒计时贴纸，订阅后倒计时提醒）。

### 2.2 直播前准备

**团队配置**：
| 级别 | 人数 | 角色 |
|---|---|---|
| 基础配置 | 2 人 | 主播 + 运营/主播 + 场控 |
| 进阶配置 | 4 人 | 主播 + 助理 + 场控 + 运营 |
| 高阶配置 | 6 人以上 | 主播 + 副播 + 助理 + 中控运营 + 数据运营 |

高阶角色职责：副播（协助讲解、报价格、配合主播切品/拍摄切片）、助理（配合直播间内容讲解、传递商品、直播工具）、中控运营（品库后台、改价、优惠券发放）、数据运营（短视频策划/脚本/拍剪/引流 + 直播数据复盘、商业投放、分析数据优化）。

**主播挑选标准**：形象气质契合定位、有较强表现力/互动/销售力、对产品专业讲解、有较强学习能力/学习意愿/了解目标市场用户习惯文化、语速适中有亲和力有节奏感。
**找主播渠道**：服务商引荐 → 自行寻找（海外学生/英语外教/外模等）→ 客户经理咨询匹配。

**直播脚本/流程六要素**：
- 直播主题：「做有灵魂的直播」——仓库清仓、工厂直供、老板娘档口、达人推荐等。
- 直播目标：每场目标明确（人气/销售/单品目标）。
- 开播时间：选时间事半功倍，固定开播养成心智，**选目标市场黄金时段**让粉丝掌握规律。
- 开播时长：**建议每场 3–4 小时**（保证时长 + 勤奋度是最简单的运营技巧）。
- 活动策略：巧用营销工具（满减、开场满送、整点抽奖、限量秒杀、入团福利、神秘福袋）。

**货品组货（核心，可直接套用）**：
- 海外直播保持单一主线，沉淀精准用户/粉丝，形成"流量→成交"正循环。
- 商品数量：**非标类建议 >30 款**，标品建议 **≥10 款**；单品讲解时长控制在 **5–10 分钟**。
- **货品结构比例**（⚠️ 经验值，非硬规则）：
  | 类型 | 作用 | 占比 |
  |---|---|---|
  | 引流款 | 折扣价、中低客单保障流量留存 | **≤10%** |
  | 动销款 | 低毛利、高热度、普适性强，确保流量有效引入+拉升转化 | **≤30%** |
  | 利润款 | 毛利适中、新品/特色款，放大 GPM | **40%–50%** |
  | 特色款 | 探索性类目/商品，为长期粉丝留存提供可能性 | **≤10%** |
- **排品类型顺序**：引流款 → 动销款 → 利润款 → 特色款，依次循环。
- **排品价格顺序**：低 → 中 → 高 → 中 → 低，依次循环。
- 不断测试找爆品并快速放大转化。

**产品话术结构**（话术种类：留人/互动/产品/成交/催单/结束话术）：
放大痛点 → 挖掘需求 → 产品特质 → 突出卖点 → 打造案例 → 展示效果 → 价格锚点 → 优势价格 → 营造稀缺 → 限量抢购。

**账号定位与设置**：头像/账号名能体现主营类目+心智；个人介绍可传递品牌价值、产品特点、售卖类别、店铺优惠、直播预告。

**直播形式**：达播 / 店播 / 半屏 / 「121」混播等（注意点：真人出镜、人货搭配、画面整洁、活动节点）。

**场景布置**：
- **灯光（暖白光照人，正白光照物）**：暖白光打造氛围/场景柔和、适用家居美妆；正白光接近商品本色少色差、适用服饰彩妆。三灯结构：主灯（在主播正前方上方）+ 补光灯 + 辅助背景灯。
- **色彩**：实景直播间避免大块红/黑/纯白色（易过曝/吃光/反光），建议中性背景突出商品；虚拟直播间用纯色绿幕便于扣像。
- **开播前 Before/After 自检**：灯光太暗→打够光；背景不搭→体现行业卖什么/用户能得到什么；声音太小→站播/坐播注意收音；画质模糊→真人出镜、清晰、产品细节。

**开播前预热（引流）SOP**：学习同区域爆款做模仿创作 → 制作短视频脚本（开头直接突出直播间/产品/亮点）→ 用小工具引导用户关注直播间 → 视频内容不出现引导下单（避免误判）→ 发布短视频后用 Promote 预热 → 直播开始后用直播切片二次引流。

### 2.3 直播中实用技巧

**销售玩法列举**：
- 开播即用、迅速拉升成交：**多级门槛优惠券、秒杀**。
- 提升单品转化：**单品折扣、包邮、直播间一口价**。
- **直播间定向券**（提升流量与转化的重要工具）：商品满减提转化；限时限库存特价刺激低迷流量；看播满一定时长可领券拉长观看时间。

**直播节奏要素**：产品展示、带动氛围、控制讲解频率、关注在线人气、带动互动率、促进商品转化、调动粉丝情绪、主播话术与销售能力、实时数据诊断。

**目标设定玩法**：提升人气 / 提升互动 / 提升停留 / 提升转化（分目标驱动直播动作）。

### 2.4 直播后复盘

- **复盘工具**：手机端可立即看当场数据；商家后台用【实时直播】+【直播分析】对照成交公式/直播数据做复盘（看实时表现、历史回顾）。
- **持续传播效应**：把精彩瞬间剪成短视频，降低自制成本，形成多次宣传。
- **复盘对象**：对照 0.2 的直播 GMV 三驱动（时长/流量/转化）逐项归因。

## 3. 短视频篇

> 适用性：方法论东南亚通用。**直接点名马来场景**：自产短视频卖货适合"东南亚发饰/墨镜、马来穆斯林服饰"等高消费品+价格优势品。

### 3.1 短视频基础认知

**内容分类（按目标）**：
- 娱乐营销类（曝光）：好玩有趣内容获推荐流量，符合品牌调性，目标=增粉/互动/粘性/传播价值观。
- 内容种草型（种草/涨粉/销售）：融入产品，从场景/功能/特色出发，可看性 + 产品表达。
- 直播预热与引流：预热（爆款/价格/时间/玩法/权益）+ 引流（用直播切片在开播后投放）。

**账号运营**：定位 → 内容策划 → 展现形式测试；想清优势/卖什么/能提供什么/消费者为什么从你买。
**发布作品四类**：日常更新（吸引精准粉、提账号权重）/ 预告视频（通知开播）/ 直播切片（直播中发布、获免费精准流量）/ 播后视频（答谢粉丝、增强人设）。
**积累粉丝**：启动期可用付费广告快速建标签、获精准流量与粉丝。

**建联达人合作（重要 SOP）**：
- 设置店铺计划（所有商品一个基础佣金）；主推商品设公开计划，**建议佣金 10% 起**，并设免费样品数量。
- 针对不同达人设**定向计划**，给不同佣金比例。
- **计划生效优先级：定向计划 > 公开计划 > 店铺计划**。
- 官方找达人渠道：`creatormarketplace.tiktok.com/home`；`affiliate.tiktokglobalshop.com/seller/dashboard/tcm/creator-marketplace`。
- 建联 tips：**建议建联≥100 个**，按粉丝数/类目匹配设不同定向佣金；观察粉丝数/播放量/历史推广数据/互动紧密度/是否电商挂车视频；不硬推销、当朋友；尽量设公开计划+免费样品（联盟选片区有专门资源位）；首次建联缺信任可引导达人申请公开计划免费样品（可追踪记录产出）；用官方账号建联介绍品牌故事更易回复。

**商家低成本自产短视频**：
- 适合自产卖货的三类商品：① 新奇特/强功能/特别设计、目标市场缺供给；② 目标市场高消费品+明显价格优势（**如东南亚发饰/墨镜、马来穆斯林服饰**）；③（类）标品且最好刚需品，可日更打造垂类账号。
- tips：保证量（可每天更新）；打造垂类账号增信任、捕获精准用户；多展示细节（材质/成本/功能），露脸非必选；反复利用素材组合剪辑 + 英文卖点；有模特展示佩戴/上身/前后对比/多搭配；有工厂可拍生产打包发货过程（连续剧式讲创业故事建人设）；积极回复留言，甚至针对某条评论单独拍视频回应。

**团队配置建议**：达人建联对接 + 选品开发（可含投流、剪辑、售后）。
**拍摄辅助工具**：拍摄=手机/相机/云台/无人机/运动相机；稳定=自拍杆/三脚架/手持云台/滑轨；收声=指向型/领夹型/USB 麦克风；打光=补光灯/环形灯。
**剪辑工具**：手机=剪映/字说；电脑=Photoshop/Premiere/After Effect/Audition。

### 3.2 打造优质电商短视频

**文案发布**：
- 视频内容文案结合当地特色，用疑问型/教导型/总结型/真相型。
- 描述文案模板：加合适标签 + emoji + 末尾留**开放式问题**。
- 用 Stickers 强调小黄车：编辑页 → Stickers → 搜 "Arrow/Point" → 提前算好小黄车位置加箭头。
- 发布封面：风格统一、有重点、明确传产品卖点。
- 热点话题植入：结合当地热门选题和音乐。

**账号主页设计**：背景图（话术引导+IP图）/ 头像 / 名称（体现账号类型）/ 账号简介（突出 2-3 个特点）/ 置顶视频（高赞高互动、体现人设）。

**短视频选品排款**：按电商玩法选（跟平台政策激励选潜力品类）/ 按账号定位选（风格调性匹配产品属性）/ 按粉丝需求选（有粉丝量后按画像选）。

**拍摄六要素**：画质清晰曝光正常（勿过度磨皮）；背景干净整洁；不遮挡关键信息（品牌+产品细节）；画面稳定不晃；音质良好人声稳定（BGM 不嘈杂）；真人出镜外语流利（**不建议全程 AI 配音**，缺真实感）。

**脚本六步（核心方法）**：脚本含场景/任务/道具/台词/动作；让用户第一眼接收有效信息 + 抛痛点/痒点 → 解读分析体现专业 → 收尾鼓励互动。
| 步骤 | 用户核心指令 | 关键数据 | 切入点参考 |
|---|---|---|---|
| 1 | 对视频关注和兴趣 | 播放量 | 第一印象、条件反射 |
| 2 | 有耐心看完视频 | 完播率 | 引发思考、增加悬念 |
| 3 | 喜欢该视频 | 点赞率 | 产生情绪共鸣 |
| 4 | 留言互动 | 评论率 | 埋下留言梗、提出问题 |
| 5 | 转发分享（二次传播） | 分享率 | 强需求、强认同感 |
| 6 | 关注你 / 进直播间 | 直播间/头像点击率 | 认同感、安全感 |

**四大黄金运营法则**：
1. **开场 5 秒是关键**（吸引用户关键窗口）。
2. **挂车商品详情页优化**（作短视频内容补充，承接下一层漏斗）。
3. **引导购买做到位**：口播导购 + 锚点箭头引导点小黄车 + 评论区置顶购物锚点/引导语/预埋互动。
4. **商品卖点要说清**：主动戳利益点建信任，介绍材质/功能/新奇特/价格优势/物流服务。

**优秀视频类型**（可复制题材库）：横向测评（选 3-5 款比卖点功能材质价格，尤适 3C；客观分享结果；重视主题信息量）/ 制作过程展示（适手工品，还原产地工厂车间、讲工艺、强调真实不可虚假溯源）/ 商品深度讲解（可邀有天然链接、讲目标国语言的 KOL）/ 使用教程攻略（购买技巧、真人演示、实用知识）/ 多元场景展示（vlog 日常、主题小剧场、高质感稀缺视频如 INS/动漫/电影/舞台风）。

**不同阶段营销玩法**：初期=引流积累粉丝、提人气；中期=薄利多销迅速起量、不断提权重；后期=产品搭配提客单价和用户价值。

**评论区运营**：评论区是素材来源（整理同地区高赞回复库）；结合账号 IP 维护；制造神回复提互动；置顶优质互动；优惠券/引导购买/明确卖点；积极答问；设关键词菜单。

**流量获取与提升**（核心策略：用付费撬动免费、用长效代替短效、用数据优化数据）：
- 免费流量（粉丝粘性高、有长尾、需长期运营）。
- 付费撬动免费：启动期付费快速建标签获精准推荐；增长期付费获精准用户、提直播间质量继续获推荐。
- **放大转化方式**：播放量高或有成交的短视频，先做 **$50–100 投流测试**，找到高 ROI 爆品后加大投放；直播前发高性价比爆品短视频引流直播间。

## 4. 跨册速查与避坑

- **价格一致性原则**：商城活动提报价/补贴**仅在商城内生效**，不污染短视频/直播间/达人售卖价（商城篇 P25）。
- **达人计划优先级**：定向 > 公开 > 店铺（短视频篇）。
- **中文出现处罚点**：主图、商品描述均要求**无中文**（品牌 Logo 除外）——上架前务必检查（商城篇 P21）。
- **AI 配音**：短视频不建议全程 AI 配音，缺真实感影响推荐（短视频篇）。
- **直播暂停限制**：仅 3 次、每次 5 分钟，别当休息工具（直播篇 P18）。
- **海外直播原则**：保持单一主线、不复杂化（直播篇 P28）——与国内复杂玩法的核心区别。
