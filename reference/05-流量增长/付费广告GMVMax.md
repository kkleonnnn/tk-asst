## 0. 一句话理解

付费广告的目的：**自然流量触顶后，用付费流量带来更多 GMV，并做到可计算、可归因、可放大**。
TikTok 现有广告产品分两个入口：
- **小店后台（Seller Center）→ 店铺广告 → GMV Max（最大成交额）**：全域自动化，**新手首选**。
- **广告后台（TTAM, TikTok Ads Manager）→ VSA + PSA + LSA**：手动精细化投放，控制更细。

> ⚠️ **核心互斥规则**：GMV Max 广告 和 广告后台的（VSA/PSA）针对**同一商品**只能二选一投放，不能同时跑。详见 [4. 互斥规则](#4-避坑互斥规则同一商品不能两边一起投)。

## 2. GMV Max（最大成交额）— 新手首选 ⭐

### 2.1 是什么 / 优势

- **一句话**：一种**自动化推广系列**，系统自动选商品、选创意素材、选广告位，目标是把你 TikTok Shop **渠道整体 ROI** 拉到最优。
- **全域流量**：一站打通所有可购物广告位——**TikTok 信息流 + 搜索结果 + 「商城」Tab**，可用视频和商品卡两种样式。
- **自动化**：自动创建/暂停广告、挑选效果最好的创意、优化受众；针对**整体商品 ROI** 优化，同时放大**非直播**总成交额。
- **新手友好**：操作比 TTAM 简单，开户—绑定小店—绑定 TikTok account 即可上手。

### 2.2 归因规则（重要，影响数据解读）

- 你推广商品产生的**所有订单都归因给 GMV Max 系列**——**包括自然流量内容促成的订单、联盟订单**，都会体现在「最大成交额数据面板」里。
- **归因窗口：1 天的同天订单**才计入该系列成交额。
- **只为你选择推广的商品**计算成交额；**未放进 GMV Max 的商品**，其订单成交额直接归属于你（不算进广告系列）。
  > 含义：GMV Max 面板里的 GMV 会「吃掉」自然流和联盟的同款订单，看 ROI 时要明白它是**全域口径**，不是纯付费增量。

### 2.3 可用地区（含马来西亚 ✅）

> 来源：GMV Max 官方文档原文。可信度较高，但仍以后台实际可选为准。

> 美国、英国、印度尼西亚、**马来西亚**、菲律宾、泰国、越南、新加坡。

### 2.4 创建 SOP（小店后台）

1. 进入**商家中心** → 左侧菜单「**店铺广告**」（Shop ads）。
2. 点「**创建 GMV Max 广告**」（Create GMV Max ads）。系列会建在你的 GMV Max 账号下。
   - 切换广告账号：点「广告数据面板」顶部账号名旁的**齿轮图标**。
   - ⚠️ **更换 GMV Max 账号会自动关闭所有现存 GMV Max 系列**，需在新账号重新创建。
3. 点「**推广商品**」。
4. 选要推广的商品：
   - 「**推广 TikTok Shop 中的所有商品**」= 推全店；或
   - 「**+ 添加商品**」= 选特定商品。
5. 「**优化目标**」板块：选 **ROI 目标**（可用推荐目标，或自定）。
6. 「**预算**」板块：输入**日预算**（可用推荐预算，或自定）。
7. 「**排期**」板块：选开始日期/时间；到点后**持续投放**。
8. 点「**发布**」。
9. 在商家中心「**概览**（Overview）」Tab 看 GMV Max 报告（Cost / Orders SKU / Cost per order / Gross revenue / ROI）。

### 2.5 启动 GMV Max 前必须先「清场」（避坑）

启动 GMV Max 会与既有手动广告冲突，**必须先处理**：

- **推全店时**：先**关闭所有广告账号里广告组层级所有在投的 VSA**，以及针对特定商品的 GMV Max 系列。开始投放 GMV Max 后，**所有 PSA（商品购物广告）会自动暂停**。
- **推特定商品时**：先从所有其他广告账号的 **VSA 中移除这些商品**。开始投后，同账号的 VSA 系列会停止展示这些商品，在投的 PSA 系列也会自动停止展示所选商品。

## 3. 广告后台三类购物广告（VSA / PSA / LSA）

> 入口：**TTAM（TikTok Ads Manager）广告后台**，手动精细化投放。

### 3.1 PSA — 商品购物广告（没有 TikTok 账号也能投）

**适合谁：**
1. **新开 Shop、还没有 TikTok 账号**的商家——支持快速创编，用图文素材。
2. 已在投 LSA/VSA、想**开拓新流量位**的商家。

**最大特点：无需额外准备视频素材。** PSA 自动抓取商家后台的**商品详情页图片 + 产品文案**生成广告（目前不支持投放时定制化修改）。
> ✅ **避坑/前置动作**：因为 PSA 直接吃后台素材，**上架商城商品时就要做好产品视觉主图和关键文案词布局**——头图要「背景无干扰、突出核心卖点」。这等于把广告素材质量前置到了上架环节。

**创编步骤（TTAM）：**
1. 登录 TTAM → 创建新广告系列 → 目标选 **Product Sales**。
2. 产品来源选 **TikTok 商店** → 完成系列设置。
3. 投放设置：**年龄确保选 18 岁以上**。
4. 出价/定向：
   - 优化目标：**GMV（总收入）/ Purchase / Click**。
   - 出价方式：
     - **Highest GMV**（≈lowest cost）：用给定预算最大化 GMV。
     - **Set minimum ROAS**（≈cost cap）：设最低 ROAS 阈值，在此之上最大化 GMV。
   - ⚠️ **不选定向 = 自动定向**；**建议全天投放**。
5. 选商品：
   - **All products**：选全店，**商品更新后广告内商品自动更新**。
   - **Specific Products**：选特定商品。
6. 提交，右侧可看广告预览。

### 3.2 PSA 投放打法 / 优化方向（经验值，非官方）

| 项目 | 建议 |
|---|---|
| 投放类型 | PSA |
| **预算** | 单广告组 **$30–50** |
| **优化目标** | 主：**GMV Value × MiniROAS**（占 **70–90%** 预算）保消耗量级和 ROAS；辅：**Click/Convert × Lowest Cost**（占 **10–30%** 预算）提过程指标 |
| 定向 | 平销期**通投**；大促期**通投 + 180 天复购人群包** |
| 基建 | **10–15 条计划** |

**按阶段的产品选择策略：**

- **0→1 阶段（全新商家入局测试）**：首选 **All products + minimum ROAS 出价**，多计划同时跑，每个广告组日预算 **$50–100**（或至少 5× 订单 CPA）。**观察 3 天 ROAS**，给模型学习时间；达标及时加量，不达标可关停新建。
- **1→2 阶段（有主推/爆品）**：
  - All products 实际 ROAS 好且想放量/提曝光 → 用 **Specific Products** 选店铺 top listing / VSA 爆品投放；**始终至少保留一个 campaign 在跑所有产品**，逐步提预算。
  - 出现爆品想继续推 → 加 **Specific Products campaign + Highest GMV 出价**，逐步提预算，**保证预算消耗率 ≥80%**。
  - 可按**同功能/类型/场景**打包测试（如同元素：快充 / Type-C 口 / 办公数码等）。

### 3.3 VSA — 视频购物广告（手动，需视频素材）

**创编四层级（TTAM）：**

1. **推广系列 Campaign**：广告管理平台 → 推广 → 新建 → 目标选「店铺推广/商品销量」→ 自定义系列名 → 预算设置。
   - 💡 **预算放哪一层**：推荐「设置无限」即**在广告层级设预算**；广告组多时可在 campaign 层设/调预算，但 campaign 层预算可能**限制广告组无法按广告层级预算充分消耗**，影响跑量。
2. **广告组**：命名 → 选要投放的店铺和**版位**。
   - 广告类型：Video Shopping Ads / Live Shopping Ads。
   - ⚠️ **VSA 目前仅支持 TikTok 版位**（默认设置）；可开/关用户评论与视频下载。
3. **目标受众**：相似/自定义人群包（定向或排除）；人群属性可筛地域、性别、年龄、语言、行为&兴趣、设备。
4. **预算/排期/优化目标**：
   - 预算：日预算 / 总预算；排期：设起止日期或持续投放；**分时段**：全天或指定小时。
   - 优化目标：**Click / Purchase（开始下单）/ GMV（支付完成）**。
   - 出价方式：**Maximum Delivery（最大投放量）/ Cost Cap（出价上限）/ Set minimum ROAS（目标 ROAS）**。
5. **选商品 + 选创意**：单件/多件商品（选完自动生成落地页）→ 点「+」上传 TikTok 视频做创意 → 文案 + 行动引导文案（去逛逛/查看详情）→ 设广告名。

### 3.4 用达人授权帖子发广告（Spark Ads）

把达人已发的帖子直接拿来投流，更真实、转化更好：

| 角色 | 平台 | 操作 |
|---|---|---|
| **达人** | 自己的账户后台 | 操作获取**视频授权 code**，发送给广告投放者 |
| **广告投放者** | TTAM | 工具 → **创意素材库 Spark Ads** → 申请授权码 → 输入授权码；在**广告组层级「广告发布身份」下拉选择达人 TikTok 账号** |

## 4. 避坑：互斥规则（同一商品不能两边一起投）

- **GMV Max 与广告后台的产品互斥，同一商品只能二选一投放。**
- 实操顺序：要切到 GMV Max 全店推广，先关掉广告组层级在投的 VSA 和针对特定商品的旧 GMV Max；启动后 PSA 会自动暂停（详见 2.5）。
- 反过来想用 VSA/PSA 精细化投某商品，就别把它放进 GMV Max。

## 6. 速查：怎么选 / 怎么起步

- **完全新手、没视频素材、想最省事** → 直接用 **GMV Max**（小店后台）或 **PSA**（吃商品卡）。
- **没有 TikTok 账号** → **PSA**。
- **有好视频/达人素材、想精细化控人群版位** → **VSA**（必要时配 Spark Ads 用达人授权帖）。
- **新店冷启** → All products + minimum ROAS，多计划，单组 $50–100，跑满 3 天看 ROAS。
- ⚠️ 切换任何打法前，先想清楚**互斥规则**，别让同一商品在 GMV Max 和 VSA/PSA 里打架。
