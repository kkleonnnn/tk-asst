## 0. 一句话主线

无货源/铺货型小店加快出单，靠**三招**：① **上品上新**（最简单、吃新品流量扶持）→ ② **达人放量**（最暴力、当天就能出单）→ ③ **大促**（借平台节点放大）。流量要持续，靠**三类资源**撬动（达人 / 供应链货源 / 海外仓）。爆款要稳，靠**全链路漏斗逐层诊断+对症优化**（曝光→点击→加购→成交→复购）。

## 1. 三招出单（来自 `8.TK小店出单逻辑`）

### 招式一：上品上新——最简单的出单方式 ★本文重点增量

**为什么有效**：平台对新品给**流量扶持**（搜索页更多展示、新品推荐频道曝光机会）；讲稿举例「新品曝光量一周内提升 30%」〔培训口径，举例〕。新品还能给老客户「店铺有活力」的信号、拉回头客。

**上新节奏 SOP（培训口径的具体阈值，落地以实际数据为准）**：

| 店铺阶段 | 上新动作 | 备注 |
|---|---|---|
| **新店下店后（起店期）** | 选好风格批量上新 **50–100 个**（新店单次上限 100） | 抓平台起步流量，及时做曝光提转化 |
| **出单后（稳定期）** | **低频**上新：一个月 1–2 次、每次约 **20 个**；店铺总品**维持 300 个左右** | ⚠️ 上太多反而稀释流量曝光与转化 |
| **数据下滑时（救场）** | 一次**上新 20 个以上**拉曝光，且上「**当下热卖 / 达人在带的同款品**」拉动整店曝光 | 用热品给店铺引流回血 |

> 💡 **核心心法（讲稿原话提炼）**：不敢上新 → 卖不动 → 更不敢上新 = **死循环**，最后只能关店。坚持上新会被买家贴上「**这家更新快 / 货好卖**」的标签，养成「想要什么先来你家看看」的复访习惯。**把"持续上新"当成一种引流纪律，而不是有需要才上。**
> 〔可信度：中。方向合理，但「300 个」「20 个」「一周 30%」均为培训举例，不同类目/店铺差异大，以自己数据罗盘表现为准。〕

**上新前的准备**：精准选品（市场调研 + 选有潜力/有竞争力的爆款）→ 优化关键词（提搜索排名）→ 卡上新时间（**大促/当地法定节假日的消费高峰前 15 天**是上活动品的最佳窗口）。

### 招式二：达人建联放量——最暴力出单（增量见下，主体见 `达人联盟.md`）

本课对达人部分讲得较浅，**系统内容以 [`达人联盟.md`](./达人联盟.md) 为准**（那边有完整的公开/定向合作后台 SOP、寄样、各国联系渠道、话术库、S/A/B 分级）。本课**值得补记的几条筛选硬指标**：

- **近 30 天成交件数 < 300 件的达人不建议合作**（成交件数越高=带货能力越强）。〔培训阈值，举例〕
- **看销售渠道有没有直播转化**：只有视频数据、没有直播的达人，难做直播转化 → 优先选「直播+视频都好」的达人。
- **发货率 ≥ 80%** 的达人才合作（更守信用）。
- **平均佣金率要高于达人均值**——你给的佣金高于该达人的历史平均，更容易获得关注/接单。
- **粉丝画像匹配**：做女装就选女性粉丝占比高的（讲稿举例「女性粉丝占比 89% 的达人很合适」）。

> 适合给达人带的货的特征（毛利高留 8–15% 佣金、普适性强、货值不高、库存深、CTR 高曝光低、历史销量+评论多）与 `达人联盟.md` 第 3 节**完全一致，不重复**。

### 招式三：大促——借平台节点放大（主体见 `06-活动大促`）

本课大促部分与 `06-活动大促/大促作战.md` 重叠，**仅补记两条可直接照做的后台动作**：

- **领大促激励**：商家中心 → **成长任务** → 领取大促活动（拿店铺激励/扶持促转化）。
- **大促 5 件套准备**：① 商品价格检查（有问题报备平台经理）+ 库存备足；② 商品合规自查（标题/描述/图片合法、上大促氛围物料）；③ 商品描述真实不夸大（赠品要在标题和详情都标清，避免「货不对版」）；④ 品牌资质上传（**只有拿到品牌方授权才能在详情/标题写品牌名，否则易被告侵权**）；⑤ 禁售/不支持销售品不要上。
- **大促营销配置**：合理设置大促期营销工具引流 + 实时监控广告关键词做付费引流 + 站外引流预热爆款。
- 大促常见类型：年度大节（6.6 / 7.7 / 双11 / 黑五）、季节大促（春夏秋冬）、新品上新促 / 清滞销库存。**全年节奏看官方 TikTok 营销活动日历**（讲稿提到但未给具体日期 → 以官方后台当期为准）。

## 2. 三类资源撬动流量池（来自 `19.如何利用资源撬动流量池`）★框架性增量

这节课最有价值的是把「撬动流量」拆成**三类资源杠杆**，可作选品/备货/履约决策的检查清单。**数字均为讲师举例（多为印尼/越南/泰国案例），勿照搬马来。**

### 2.1 达人资源——「用来放量」的杠杆

> 一句话定位：**达人是用来放量的**，快速把店铺销量做起来；纯自然流出单要 **3–7 天**，靠优质达人**当天上架当天就能拉满 ~100 单**（短平快，商品卡/投流都难做到这种瞬时爆发）。

**三段式放量打法（冷启动）**：

```
微腰尾部达人（突破/铺量） → 中部达人（冲榜） → 头部达人（引爆）
```

达人资源的五个作用（可作"为什么要砸达人"的论据）：
1. **极速冷启动**：三段式快速累积销量+评价。
2. **精准文化翻译**：达人比品牌方更懂本土，能重新包装产品提点击/转化（举例：越南达人把防脱发洗发水包装成「熬夜党救星」，单场 GMV 10 万美元）。
3. **动态选品+库存杠杆**：拿达人直播间互动率当实时"问卷"，反推备货深度，降滞销 SKU。
4. **信任转移 + 溢价空间**：达人背书可让同品质产品溢价 20%–50%（举例：国产美容仪经印尼达人包装，客单价 USD79→119，仍保 30% 转化）；**政策突变期本土达人是合规缓冲垫**（举例：印尼站受影响时，用更高佣金吸纳本土达人，跨境卖家约两周恢复销量）。
5. **反向指导供应链**：达人视频评论/弹幕高频词反推产品开发（举例：东莞宠物玩具厂据达人视频高频词开发「防咬级」玩具，上架 3 天冲热销榜）。

> 案例锚点〔举例〕：某店 28 天卖 100 多万，其中**达人直播带货 92 万 + 短视频 23 万**——印证「快速出单几乎全靠达人」。

### 2.2 供应链 / 源头货源——降本 + 提速 + 控库存

**核心备货纪律（可直接照做）**：
- **测品/放量前先备一定数量库存**，但**不要一次备太多**——**少量多批次**：日销 100–200 件就备 200–300 件，卖得差不多再补，保住安全库存即可。（举例：有学员当天订单七八百件，才一次性备 1000 件。）
- **东南亚爆款生命周期仅 7–14 天**，要求供应链能快速生产交付、避免断货。

源头货源的好处：
- **降本提利润**（讲师举例对比：拼多多拿货 1000 件成本约 ¥22.45–22.9/件，源头供货约 ¥16 起、正常 ¥18.5–19/件——差价即纯利）。⚠️ 这是讲师在**引导找他们供货/扩盘**的话术，价格仅参考、自行比价。
- 快速响应+准确发货 → 提满意度/店铺评分/复购。
- 精细化库存管理 → 降缺货与滞销风险、提资金利用率。
- 反向定制做**差异化**（举例：开发环保类产品提议价能力）。

### 2.3 海外仓——时效/体验/成本/风险四重杠杆

> ⚠️ 本店为**马来跨境从中国发货**起步，海外仓属**进阶选项**；认证海外仓走官方认证仓（向导记录 Allsome 为核心、经客户经理申请，**非 FBT**）——细节见 `01-开店入驻` 与 `04-履约发货`。本节仅记讲稿给的"海外仓为什么值得做"的论据。

海外仓作用（讲师举例数字，参考）：
- **时效**：交付周期压缩 **70%**，提转化、降退单率。
- **体验**：支持**本地退换货**，解决跨境售后、提复购；本地退货还能**避免「无处可退=整件货白白浪费」**（讲师强调：浪费 1 件货可能要 4–5 单利润才能弥补）。
- **成本**：头程为空运的 1/2、尾程比国际快递低 20%–50%、旺季不涨价；共享海外仓降淡季仓储成本。
- **竞争/风险**：平台给**本地发货商品额外曝光**，避开低价竞争；作贸易壁垒缓冲区，降政策/物流风险。
- **资金杠杆**：海外仓「**先销后付**」模式省现金流、提旺季销售额。

> 关税提示〔讲稿口径〕：**东南亚关税问题相对较小**（讲师称「美区才头痛、关税战麻烦；东南亚总体尚好、略有波折」）。⚠️ 仍以官方当期政策为准；马来 LVG（低价货物税）由平台代收代缴的结论见 `01-开店入驻`。

### 2.4 东南亚市场认知（方向性，勿引用具体数）

- 东南亚是**高速增长大市场**（疫情后线上化加速，中产增长，越来越重**性价比与质量**而非单纯低价）。
- **「东南亚人穷只买低价」是误区**：讲师强调消费者很看重质量，**纯靠低价做核心竞争点不利于长期发展**。
- 内容电商/社交电商势头猛：**~70% 东南亚消费者通过短视频获取产品信息** → 未来「内容（短视频+直播）带货」是主方向，**找达人务必看其短视频质量**。
- 平台格局：当前 **2+N**（Shopee、Lazada 主导），TikTok Shop 为新兴力量快速上升。〔讲师预判 TK 将超越，属招商口吻，方向参考〕

> 💡 **课程给的两个落地大方向**：① **内容很重要**——多建联、沉淀长期合作的优质达人；② **产品质量很重要**——有意识地和潜力工厂建立长期合作，手握多个工厂以便快速调动产能、保质保时效降本。

## 3. 持续优化打造爆款（来自 `17.小店持续优化实现爆款打造`）

### 3.1 赚钱店铺的「前中后期」运营链路

| 阶段 | 关键动作 |
|---|---|
| **前期** | ① 设计好**主推商品**（定价/受众/类型）；② **铺货模式**多上市面热销品；③ 与达人建联快速转化拉曝光 |
| **中期** | ① 看数据：拆**商城流量 / 搜索流量 / 推荐流量**——搜索流量过高就优化商品+价格，或直接达人推广快速转化；② 挖爆品、囤优质品、聚人群，做到**日出 5–10 单以上**后再继续上品，直到**持续稳定出单**；③ 与达人聊好价格/材质/发货时效/库存，确保出单后及时发货（提时效+质量→达人口碑涨→互利共赢） |
| **后期** | 稳定后**囤货降本**；发货后**向买家要好评**、**24 小时内回复聊天消息**、多观察达人后台找更多带货达人 |

### 3.2 全链路漏斗诊断与对症优化 ★本文最实用的可照做清单

> **核心逻辑**：把成交拆成「曝光→点击(CTR)→加购→成交→复购」，每一层转化率低就有对应的固定修法。后台路径：`商家后台 → 数据分析 → 商品卡 → 详情信息`（看不到指标就点**配置指标 → 筛选**勾选对应转化率）。
> ⚠️ 注：讲稿口语里把"加购"念成"刹车"（加车→刹车，疑似语音转写错别字），下表已统一为**加购**。

| 漏斗层 / 症状 | 诊断位置 | 对症优化动作 |
|---|---|---|
| **曝光→点击 CTR 低** | 商品卡详情信息看点击/曝光 | **优化详情图/主图**：丰富详情图、加**痛点+产品搭配**吸引点击 |
| **点击→加购率低** | 配置指标→筛选→勾「加购转化率」 | **更换主图+文案**、多加**营销标签**，让主图更有吸引力勾人加购 |
| **加购→成交率低** | 同上 | 加购多但不买＝**价格问题或产品力问题** → 优化详情页介绍 + 设**优惠/秒杀/折扣/发券**降价促单 |
| **单品退货率高** | `商品 → 订单 → 商品管理`看退货 | 检查**详情页描述是否与实物相符**、**完善尺码表/材质说明**、对高退货品加**试穿视频**降退货 |
| **优惠券领取率低** | `数据分析 → 营销 → 推广工具`看券 | 在**详情页突出/重点展示**优惠券 |
| **优惠券使用率低** | 同上 | **缩短优惠券有效期**制造紧迫感（例：2 天→1 天、5 天→3 天、1 月→1–2 周） |

> 〔可信度：中-高。漏斗思路与修法通用、长期有效；后台具体菜单名以实际界面为准。本表与 `商城自然流量.md` 的「曝光→点击优化图、点击→成交优化价格/卖点」一致，此处**更细到加购层和退货/优惠券层**，是增量。〕

### 3.3 商品卡（商城自然流量）核心玩法——增量补记

> 商品卡系统内容见 [`商城自然流量.md`](./商城自然流量.md)。本课补记两点：
- **猜你喜欢占商品卡流量 80%+**（搜索来源 + 主页商城来源占比较小）。
- **两种核心玩法**：① **低价跟卖大爆品**（选市场已有爆款低价跟卖，**适合新店起店**）；② **抓运营核心指标**（高曝光点击率 + 高转化率 + 良好店铺评分，**长久有效**）。对新店最直接的就是**极致性价比**——价格是提 CTR 和转化最直接有效的杠杆。
- 提升商品卡曝光的三个关键指标：**封面点击率**（优化缩略图）、**点击转化率**（优化详情页）、**订单量**（极致性价比+佣金激励：一低价、二提佣金）。

### 3.4 达人矩阵高阶打法——增量补记

- **达人矩阵短视频出单是「最暴力最有效」的方法**，但**只适合超级大卖家**（要求产品质量高、库存充足、资金预算充裕）；可邀约达人或与机构（MCN）合作。
- 优势：高效传播 / 精准营销 / 内容多样性。
- **营销组合一句话**：**「选好品 + 做好优化 + 找达人」**，即「商城 + 达人」一起做高订单。
- 学员店铺案例〔举例，泰铢〕：自营 vs 联盟带货占比随月份演变——第1月自营 96% / 联盟 4%（总额 1.5 万泰铢）；第2月 88% / 12%；第3月 44% / 56%（总销量近 60 万），**3 个月总销量翻近 40 倍**。结论：**商品卡（自营）负责前期快速启动，达人建联负责后期迅速爆单**，达人带货拉店铺权重→拿更多平台流量→销量持续走高。

### 3.5 达人建联操作要点（增量补记，主体见 `达人联盟.md`）

- **新店吸引达人最有效的两个抓手**：**佣金率尽量高** + **库存设高一点**（增加达人对店铺的信任度）。
- **定向计划邀约**：商家后台找意向达人（发现达人 / 筛选 / 同行合作过的达人 / 带货榜·热度榜·热销榜）→ 点**邀请合作** → 填 **LINE 账号**（无 LINE 可填「12345」占位）→ 填好**本地语言邀约话术**（不同站点用当地语言、专门写）→ 添加带货商品+佣金 → **开启手动审核样品** → 输入精筛达人签约 → 发送。
  > ⚠️ 此处填 LINE 是泰国站口径；**马来西亚优先 WhatsApp**（见 `达人联盟.md` 各国渠道表）。

## 4. 跨课避坑 & 心法速查

- **别陷入"不敢上新"死循环**：把持续上新当引流纪律；新店批量上、稳定期低频上、下滑期上热品救场。
- **达人是放量主力，不是自然流**：想"上架当天就出单"几乎只能靠达人或选品极准，别指望纯自然流/商品卡/投流做到瞬时爆发。
- **备货少量多批次**：东南亚爆款只活 7–14 天，但也别一次性囤大几千件压资金，按实际销售滚动补。
- **数据看漏斗、对症下药**：CTR 低改图、加购低改主图文案+标签、成交低改价格/详情、退货高改描述/尺码/试穿视频、券领低改展示、券用低缩短有效期。
- **培训话术保持警惕**：三份讲稿都在卖课/卖供货服务（"交给专业团队""找我们扩盘""我们价格更低"），**方法可取，商业引导自行判断**；所有费率/政策/后台路径以**官方后台当期**为准。
- **市场规模数字不可引用**（同篇 2300 亿 vs 1800 亿自相矛盾），只取"高速增长、重质量非纯低价、内容化"的方向性认知。
