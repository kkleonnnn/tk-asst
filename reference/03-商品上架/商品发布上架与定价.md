## 0. 核心概念：全球商品 vs 店铺商品

跨境店发品是"两段式"，必须先理解这个模型，否则后面所有操作会乱：

- **全球商品（Global Product）**＝一个**商品库**。用户**看不到、买不到**。它存价格、库存、信息。
- **店铺商品（Shop Product）**＝某个站点（如马来 MY）**架上、用户能看能买的商品**。
- 全球商品**发布到某国站点**后，系统自动：翻译成当地语言、按货币换算+税率+物流藏价算出店铺价格、按发布国家数**均分库存**（可手动改）。

发品主路径：`卖家中心 > 商品 > 添加全球商品` → 填信息创建 → 选择要发布的国家站点（只显示已开通的国家）→ 发布。
编辑已上架：`商品 > 管理店铺商品 > 编辑`。

**店铺商品状态含义**（看商品列表时对照）：
| 状态 | 含义 |
|---|---|
| 生效中 | 审核通过、在售 |
| 审核中 | 平台审核中 |
| 审核失败 | 未通过审核 |
| 已下架 | 卖家自主下架 |
| 平台下架 | 违规被平台下架 |
| 冻结 | 疑似违法/有害/假冒 |
| 已删除 | 已删除 |

## 1. 单个发布商品 SOP

路径：`首页 > 商品 > 添加全球商品`。各字段要点：

- **关键词找类目**：只输关键词，系统自动匹配类目；再输商品名称确认类目。系统会按标题推荐类目，**自行核对是否属实**，选正确类目。
- **商品名称（标题）**：不含违禁词与营销词；**25 ≤ 字符数 ≤ 255**（这是"如何发布商品"文档口径；标题优化文档另给"40–150 字符""建议 80 以内"——见 §3，以系统校验为准）。
- **商品属性**：除"品牌"外其余多为选填，如实填。**店名或详情页出现品牌名/品牌 Logo 必须有品牌授权**，否则按山寨处罚；无品牌就填"无品牌（No Brand）"。
- **商品详情**：图片 + **白底图** + 描述（图/文/图文）+ 尺码表（**服装等特定类目必填**）+ 视频（选填）。
- **物流与保修**：⚠️ **商品实际重量决定运费计重，谨慎填写**（重量直接影响藏价与利润，见 §6）。
- **销售信息**：销售属性（color / size 等，按商品填）。

## 2. 批量上架（货叮咚 ERP 一条龙）— 实操重点

> 来源：`批量上架工具指导教程`（53 页 PPT）。这是用**第三方 ERP「货叮咚」**采集源头货（PPT 示例用拼多多 PDD）→ 编辑 → 套定价模板 → 批量发布到 TikTok Shop 的完整链路。
> ⚠️ 货叮咚是**第三方服务商工具**，非 TikTok 官方；插件/链接/收费项随服务商变化，仅作流程参考。

### 2.1 前期环境准备
1. 安装 **Chrome 浏览器**（官网 `google.cn/chrome`）。
2. 登录货叮咚（`huodingdong.com`，可用谷歌账号登录）。
3. 安装**货叮咚 ERP 插件**：打开 `huodingdong.com/member/product/general/collect`，进入扩展程序页，把下载的插件**拖进去**安装。

### 2.2 采集 → 编辑 → 发布 一条龙
1. **采集**：从源头平台（PPT 示例 `mobile.yangkeduo.com` 拼多多）采集产品；产品自带视频会自动采集，没有可手动上传。
2. **编辑图片/内容**：可快速筛选要的图片和内容；**带中文的图片可做内容翻译**转英语/泰语等 —— ⚠️**图片翻译是收费功能，需另付费**。
3. **SKU 与价格编辑**：可批量编辑 SKU 主图与产品图片。
4. **包裹信息**：
   - 包裹重量＝商品重量（**预估填写**，较重商品可问 PDD 卖家拿准确重量）—— 影响定价，要尽量准。
   - 包裹尺寸：填**小于平台限制尺寸**即可（不影响商品定价）。
5. **翻译保存**：编辑好的信息需**一键翻译为英文**才能保存。
6. **认领发布**：选择需发布的商品 → 认领 → 选平台和站点、确认认领店铺、选发布方式（也可用模板）。
   - ⚠️**发布时选择"单个站点发布"**。
7. **套用定价模板**：选提前设置好的定价模板（见 §5）。

### 2.3 批量上架避坑清单（PPT 原文）
- 选品注意商品**是否是仿品牌**的。
- 上新注意**标题有没有出现品牌词 / 年份 / 禁卖品名称**等敏感字眼。
- **商品标签不可被模糊、遮挡或空白**。
- 带 **星号(\*) 的都是必填**。
- **商品编辑好之后务必先套定价模板再发布**（强调两次）。
- 必须对上新商品**重量有较准确预估**。
- **服装类目必须上传尺码图**。
- 发布前确保定价模板正确，且发布时套用。

## 3. 商品标题优化（SEO，多份文件交叉，核心）

> 多份文件（打造商品的建议 / Fancy Listing / 商品优化指南 / 如何优化商品标题）重复讲标题，这里合并。

### 3.1 标题公式
**完整标题 ＝ 「核心关键词 + 属性关键词」+ 促销关键词 + 功效关键词 + 人群关键词 + 品牌关键词**

| 类型 | 数量 | 说明 / 注意 |
|---|---|---|
| **核心关键词** | 1–5 | 商品名称/俗称。分**精准词**（如"辣条"）和**模糊词**（如"零食"）。可选词少→两类都用；可选词多→多用精准词。可参考各国称呼习惯放多个。 |
| **属性关键词** | 0–3 | 材质/颜色/风格。**标品**用材质+颜色；**非标品**（服装）用风格。 |
| **促销关键词** | 0–1 | 包邮/折扣/赠送等，须合规（参考官方赌博/赠品/盲盒规则）。⚠️ 标题里**不要写 "X% off"** 这类营销信息（标题优化文档明确禁止），促销词慎用。 |
| **功效关键词** | 0–2 | 美白/保湿/显瘦/保鲜等。**不能夸大、不能用绝对词**。塑身衣用 shaping/塑形，**别用 slim/slimming**（单独语境=瘦身，触发违禁）。 |
| **人群关键词** | 0–1 | 老人/小孩/青年/孕妇等。 |
| **品牌关键词** | — | 无授权**不要乱用**，易侵权。仿牌兼容写法须用 **"for"**（如 `Compatible for iPhone…`），**不可直接加品牌名，图片不可露 Logo**。 |

### 3.2 标题格式规则（合并各文件，数字以系统校验为准）
- 字符数口径**不一致**（需以官方为准）：
  - "如何发布商品"：25–255 字符；
  - "如何优化商品标题"：**40–150 字符**；
  - "商品优化指南"：含空格 **255 字符上限，建议 80 以内**。
- **每个单词首字母大写**（连词、冠词、介词除外）。
- **减少重复单词/短语**；不要罗列同一属性的不同变体（如 "red yellow blue green"）；不要写库存信息（如 "Only 8 pieces in stock"）。
- **禁绝对词**："最好/最高/100%/最佳/物超所值/畅销"等。
- 标题里**空格 2–3 个为宜**，让标题显得紧凑。
- 内容结构建议：[品牌] + [产品内容] + [适用范围] + [产品类型] + [主要功能/特点/优势]。
- **3C 文案建议**：【大促/达人效应/引导词】+【品牌名】+【商品词】+【营销信息/定向人群卖点】+【规格参数/货号】；白牌或低知名度品牌→营销/商品词前置、品牌名后置；厨电/充电宝等规格决定决策→规格参数前置。

### 3.3 找关键词的 3 条路径
1. 参考三方平台对应市场的**爆品/增长新品命名方式**。
2. **Google 指数**：检索对应国家对应商品，找飙升与热门词。
3. **卖家中心 - 数据罗盘 - 搜索 - 关键词榜单**（热门搜索 / 上升 / 高潜力）。后台没榜单＝系统抓的数据不够，**多上品就会出现**。

### 3.4 改标题的纪律（避坑）
- **不宜频繁改标题**。最好每周对局部词做替换，**每次 3–5 个字符**。
- **属性词尽量不改**；热卖品改标题更慎重，能不改就不改。
- **避免关键词内耗**：同品类、上架时间接近、关键词相同的多个商品会互相抢量，应适当替换部分关键词。

### 3.5 官方"商品优化工具 / Product Optimizer"（推荐用）
- 入口 1（管理商品）：`商家中心 > 商品 > 商品优化工具`。系统给**推荐词**（代表消费者购买诉求），列表直接展示"推荐加入标题的词"和"优化后 28 天曝光提升预期"；hover 看该词搜索指数/销售指数（指数越大热度越高）；抽屉页给搜索词榜单（热搜词/热趋词，每商品最多 500 个，支持日/周/月）。
- **批量优化**：列表右上角"批量修改标题"，批量采纳推荐词后统一提交。
- 入口 2（经营分析）：`数据罗盘 - 商品卡片 - 搜索 - 商品标题优化`，可看搜索渠道增量曝光、PV、GMV 趋势。
- 案例口径：商家采纳推荐词后 **7 天内搜索曝光显著提升**；采纳后看 **28 天**曝光收益。⚠️ 招商宣传口径，数字仅参考。
- 注：工具内含 **SEO 功能**，平台对符合条件的商家有**激励政策**（具体政策以官方为准）。

## 4. 主图 / 素材 / 详情页

### 4.1 主图规则
- **1:1 正方形**，尽量凑满 **9 张**。
- **首图**：主体突出、背景整洁干净、**不能有文字/拼接**，建议场景图或白底图。
- **白底图标准**：纯白/纯色背景；主体占比约 **2/3**、必须展示整体不裁剪；商品色与背景色要有差异；**至少 600×600 像素**（"增强竞争力"文档另建议 >1200×1200，至少 3 张图——更高更好）；不能有阴影、毛糙抠图痕迹、过暗、模糊、过曝、变形。
- **场景图禁区**：禁名人/版权元素、禁杂志或电商 App 标志；禁暴力色情/药品/枪械等；禁用同类商品做道具。
- **3C 行业两种主图打法**：①简约纯底图（突质感/科技感/高端，适合 3C 或高客单，卖点≤1–2 个并放大最核心卖点）；②场景卖点图（重代入感、共鸣，从使用方法/效果/体验入手）。副图风格要与主图统一。

### 4.2 商品素材库（官方，省事工具）
- 入口：`商品 - 素材中心`。云端存图片/视频，发品/维护时直接导入，支持搜索/预览/下载/移动/删除；**目前免费**（左下角看已用/总容量）。
- **图片规格**：JPG/PNG/JPEG，**最大 5MB**，尺寸需 >300×300。
- **视频规格**：MP4，**最大 20MB**，比例 **9:16 ~ 16:9**，时长 **<60s**，单次传 1 个。
- 官方账号若有直播切片/讲解短视频，系统自动导入 "Live Clippings" / "Short Videos" 文件夹；没有则隐藏。
- **批量上传商品时用法**：选图后点 "Export URLs" 拿图片 URL，填进头图/尺码表/资质/SKU 图片等列。白底图可在线点 "Select main image" 创建头图白底图。

### 4.3 详情页结构（"Fancy Listing"模板，记得翻成目标国语言）
1. **入店欢迎语**（详情页第一部分）——见 §7 话术。
2. **Product information**：Brand（无则不写）、Weight、Color、Size、Launching time、Texture、Style、Using scenario 等。
3. **Shop service**（按店实际改）：About Logistics（中国仓发货，一般付款后 **5–7 自然日**发出，海运可能 **15–20 自然日**到 —— ⚠️ 时效是模板话术，**以马来实际物流/官方时效为准**）、About Products、About Service。
4. **产品详情描述 + 图片**：多 SKU 按颜色/尺寸做场景展示；单 SKU 多场景/多角度；服装配尺码图。
5. **细节展示**（区别于竞品的特色）。
6. **结尾语**（翻成对应国语言）。

### 4.4 详情完整性硬性要求（避免冻结/罚单）
- **含电池商品**：上架必须写明电池信息（容量、电压、电流等），否则按禁运品处理、链接被冻结。
- **需充电线商品**：必须配适配机型的充电设备，平台会重点抽检，缺充电线会拦截包裹甚至罚单。

### 4.5 增强竞争力三维度（"如何增强商品竞争力"）
- **显示（被看到）**：主图优化（白底/完整/不重叠/无汉字）、标题 >25 字符不重复词、描述含图、SKU 配不同图、关联品牌信息、适当加 SKU 库存。
- **优惠活动（促转化）**：店内活动（折扣/代金券/闪购/买更多存更多）、价格竞争力、货到付款（COD）。
- **信任感**：品牌授权认证、其他用户评价；控制**负面评价**和**高退货率**。

## 5. 定价模板（5 国）与折扣设置 — 利润核心

### 5.1 定价模板逻辑（`5国定价模板`）
- 这是**新手前期通用模板**，不是一成不变；熟悉后按实际调整，不是所有店做一样折扣。
- 模板内置 **5% 达人佣金**（因现在多为视频带货，佣金低，控制在 3–5 个点）。若后续与达人深度合作做直播带货，佣金可能谈到 **10%–15%**，届时单品价格要相应上调，不要硬亏。
- 模板是**前期定价**，后期要按实际减免扣除（免佣、货源涨价、汇率波动、达人高佣金等）酌情调价：**成本升→调高价，成本降→可酌情调低**。
- ⚠️ **越南特例：折扣不能低于 5 折**，所以越南要把折扣做高一点、留后续调整空间。
- 5 国均有对应模板（泰国/马来/菲律宾/越南/新加坡），按国家分别创建。

### 5.2 定价模板五项成本（藏价构成）
模板要把以下成本"藏"进定价（数字**不定期更新，以官方邮件/卖家后台/学习中心为准**）：

| 项 | 说明 / 数字 | 可信度 |
|---|---|---|
| **运费** | 只藏**跨境物流成本**；买家支付的运费系统自动叠加，**不做藏价**。费率见学习中心，不定期更新。 | ⚠️ 数字以官方为准 |
| **佣金/手续费** | 各站点佣金费率见学习中心，不定期更新。 | ⚠️ 以官方为准 |
| **电商服务费** | 见学习中心，不定期更新。 | ⚠️ 以官方为准 |
| **增值税/VAT** | 见下表 | ⚠️ 以官方为准 |
| **达人佣金** | 模板默认 5%（视频带货） | 见 §5.1 |

**各国 VAT 参考（务必以官方为准，可能已变）：**
| 国家 | 税率 | 规则 |
|---|---|---|
| 泰国 | 7% | 对所有商品征收 |
| 新加坡 | 9% | 售价 **<400 新币** 征 SST，超过则不收 |
| **马来西亚** | **10%** | **适用售价 ≤500 马币的商品，超过则不收税** ✅ 与本店相关，重点 |
| 越南 | 10% | — |
| 菲律宾 | 无税费计算 | — |

**官方增值税计算公式**：`增值税费 = 买家实付价 ÷ (1 + 增值税率) × 增值税率%`

> 平台政策调整都会发到**绑定店铺的邮箱**，要多留意官方邮件。

### 5.3 折扣设置实操（营销中心-促销工具）
- 入口：`营销中心 - 促销工具`（商品折扣、秒杀、多买优惠、运费折扣、优惠券等组合促销）。
- 路径：`营销中心 - 促销工具 - 商品折扣`，按定价模板里的折扣打折。
- ⚠️ **让利折扣填法（容易填错）**：如果定价模板里做的活动预设是 **6 折**，这里要填的"折扣"是**让利幅度 = 40%**（即 6 折＝让利 40%）。
- **新店建议折扣最多设到 5 折**，给后续新品折扣留空间。

## 6. 重量 / 物流对定价的影响（贯穿性提醒）
