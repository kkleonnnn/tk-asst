## 0. 装修的前置条件（不满足装不了）

> 来源：`店铺装修教程`。装修前先确认下面三条，否则进不去或没意义：

1. **不能是"练手店铺"**——练手店铺**无法装修**。
2. **必须已缴纳保证金**。
3. **必须已上架至少 1 个商品**——"店铺装修需要先上架产品，再去装修"。后台若提示"添加商品说明"，就是还没上品。

> 衔接：上品流程见同板块《商品发布上架与定价》；保证金/练手店见 01-开店入驻。

## 1. 进入装修台 SOP（后台路径）

> 来源：`店铺装修教程`。⚠️ 路径为讲师录制时的后台版本，**以当期卖家后台为准**。

1. 登录**卖家后台**。
2. 左侧导航找到并点击 **「店铺页面」**（Shop / Store Page）。
3. 首次进入会有引导弹窗 → 点箭头处的 **「跳过」** 即可进入装修编辑器。
4. 编辑器布局：
   - **左侧 = 实时预览**（买家端看到的效果）。
   - **右侧 = 模块布局区**，模块**从上到下**就是买家端的展示顺序。
   - 加模块：在右侧选中想要的模块，**直接拖到目标位置**（拖到箭头指示的占位处）。

**关键认知（版本差异坑）：**
- 某些模块在装修台显示 **「将不显示此模块」**——意思是**后台预览看不到、但买家端能看到**。讲师说明：**平台系统升级后，秒杀 / 优惠券类模块在装修台不可见**，只要你**店铺已设置秒杀 / 优惠券，买家端就会自动显示**。→ 不要因为装修台看不到就以为没生效。
- 模块编辑区**若没有可填字段**，代表该模块**系统自动生成、不可人为编辑**。
- ⚠️ 讲师结论：**没有唯一标准模板**，按自己喜好排布即可；重点是**知道每个模块展示什么内容**。

## 2. 各装修模块逐个用法（实操核心）

> 来源：`店铺装修教程`。这是本文最可照做的部分。⚠️ 尺寸/数量为讲师口径，**以上传时后台黄色提示为准**。

| 模块 | 作用 | 图片数量 | 链接 | 模块标题 | 关键参数 / 坑 |
|---|---|---|---|---|---|
| **单品图片**（单张图） | 放 1 张图：店铺活动图 / 热卖商品图 / 达人带货图 | **仅 1 张** | 选填（海报图可不挂链接；挂了则买家点图跳该商品） | **选填** | **图片须 < 800、调到 800×800**；不合规会有黄字提示 |
| **轮播** | 多张图轮播，建议放**具体商品**（非海报） | **建议 2–5 张/链接** | 同单品图，上传图+挂链接 | 选填 | 图片比例二选一：**16:9** 或 **1:1**；买家按住左滑看下一张 |
| **热区图片** | 一张大图（理解为店铺热卖图），**图上不同区域**点击跳**不同同类商品**链接 | 1 张底图 | **最多 10 个热区点**，每点挂一个商品链接 | 选填 | 上传图后逐个"添加"热区链接，上限 10 |
| **多图组件** | 按选定数量并排展示多图 | **选几图就传几图**（选 2 图必传 2 张） | — | 选填 | "图片选项"只改展示数量，按需选 |
| **精选图片** | 按规则自动陈列一组商品 | 系统按排序展示 | — | **必填**，且**标题不能用中文**（用当地语言或英语） | 商品排序可选：**按销量 / 按新品 / 按价格 / 按折扣** |

**模块层面通用提醒：**
- 上传前先把图**存到本地电脑**，编辑时点 **「＋」** 上传，再「添加」挂商品链接。
- "精选图片"标题**禁中文**这条容易踩——填英文/马来文。

## 3. 首页布局策略（林武老师）

> 来源：`10.装修策略`。⚠️ 均为经验建议，非硬规则。

**首页三原则：**
1. **清晰导航栏**——让买家按类别快速找到想看的品类，以"用户方便"为第一。
2. **显眼位放热门 / 新品 / 爆款**——**爆款单张图放最显眼处，甚至首页置顶**（对应 §2「单品图片」模块）。
3. **图文比例合理**——不过于拥挤也不过于空旷；看着越舒服，停留越久、看的品越多、成交概率越高。

**产品展示三点：**
1. 商品图**尽量高清、多角度**，突出细节与优势；**首页用当地语言直接标 1–3 个卖点**（少则 1 点，最多 3 点），利于点进。
2. 商品描述**简洁明了**，强调卖点 / 用途 / 材质，打消顾虑。
3. **组合 / 场景化陈列**——把相关品放一起造场景感（如年货节 / 圣诞把同主题品集合，营造氛围促单）。

## 4. 营销工具 / 活动展示（带可照搬的数量建议）

> 来源：`10.装修策略`。⚠️ 下面的"张数/个数"是讲师经验配置，非平台要求；**优惠券、秒杀的设置入口见同板块《商品发布上架与定价》§5.3（营销中心-促销工具）**。

**在首页显著位展示活动**，并写清活动规则：
- **优惠券**：**至少 1 张，常规展示 3 张左右**。排序优先级：
  - **无门槛券 优于 有门槛券**；
  - **折扣高的放前**、折扣低的靠后；
  - **更早过期的放前**、更晚过期的靠后。
- **秒杀**：**至少 1 个，多则 3 个，建议 3 个以上试**；讲师强调**秒杀要多于折扣**。
- **平台满减 / 折扣**：**1–3 个即可**。
- **限时折扣 / 秒杀**用**倒计时元素**制造紧迫感促单。
- 可编辑属性一般在**编辑标题里标注**。

**客户群运营（提复购）：**
- 路径：**店铺后台 → 客户 → 创建客户群**。
- 按分类建群：**新客 / 近期客户 / 老客 / 常客 / 流失客户**，对各群发**专属优惠福利**，增粘性、提复购与转化。

## 5. 本地化与跨文化（东南亚做生意必读）

> 来源：`10.装修策略`。马来专属与通用已分开。

**通用原则：**
- **语言本地化**：店铺页面用**当地常用语言**（泰→泰语、印尼→印尼语、越→越南语…），尊重当地、方便浏览下单。（呼应 §2「精选图片」标题禁中文）
- **融入当地文化元素**（传统图案 / 节日装饰）增亲和力。
- **避免易引起误会或不适的文化符号 / 表达**。⚠️ **东南亚信教人口多（穆斯林）**，做穆斯林时尚等类目要特别注意尺度。

**装修维护是持续动作：**
- 按**季节 / 流行趋势**及时调整装修与商品推荐；**推荐位与爆款绑定——爆款有生命周期，销量从峰值降到一定程度就换新爆款上推荐位**。
- 定期检查**页面加载速度、图片质量**保障体验。
- **收集买家评价 / 反馈**持续优化。

## 6. 分国配色与元素（5 国速查表）

> 来源：`10.装修策略`。⚠️ 经验性审美建议，仅作装修选色参考，无数据背书。

| 站点 | 配色倾向 | 文化 / 装饰元素 | 布局要点 |
|---|---|---|---|
| **马来西亚** ✅本店 | **明亮而柔和**（淡蓝、淡绿），清新舒适，**避免冲突感** | 多元文化：马来 / 中华 / 印度元素可融入 | 清晰导航，商品按**类别 / 价格 / 销量**分类，整洁简明 |
| 越南 | **鲜艳**（红、黄，似国旗） | 奥黛（传统长衫，似旗袍） | 清晰直观，图片高清突出细节 |
| 泰国 | **鲜艳、对比度高**（粉、紫），注意协调勿杂乱 | 寺庙建筑 / 传统服饰 / 自然元素（花卉绿植） | 简洁有条理、重点突出 |
| 菲律宾 | **明亮活泼**（黄、橙，积极乐观感） | 天主教宗教元素 / 海洋元素（海星、贝壳） | 重**视觉冲击力**，产品图抓眼球，描述简洁 |
| 新加坡 | **简洁大方 / 低饱和**（黑白灰、莫兰迪色系，高级优雅） | 多文化交融，可中西结合自创风格 | 简洁清晰、不拥挤 |

## 8. 大促装修

> 来源：`10.装修策略`。

- **核心思路：氛围感**——大促是店铺飞跃节点，要打造**个性化大促门面**、灵活布置场景。
- **同主题色系集合陈列**：圣诞 / 元旦 / 年货节等长周期节日**以红色为主调**，把一系列红色喜庆服饰集合一起造氛围。
- **首页精心设计**：美观、布局合理、重点突出；把**优惠券 / 关注类促销活动放首页**引导关注下单。
- 开展**店内秒杀、多件营销**等促销做"临门一脚"，拉访客与销售额。
- 衔接：大促活动报名 / 玩法见 06-活动大促。
