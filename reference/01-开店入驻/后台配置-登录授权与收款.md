## 1. 练手小店登录 SOP

> 适用：服务商/孵化机构交付的"练手店 / 代注册店"，给你一个**店铺网址 + 绑定手机号 + 小店密码**的场景。自己注册的店直接走 `seller.tiktokglobalshop.com` 正常登录即可。

**步骤：**
1. **复制店铺网址**到浏览器打开。
2. 登录页选**区号**：选 **香港 +852**（练手店多用 852 手机号）。🟡
3. 填**绑定手机号**（复制粘贴，避免手输错）。
4. 填**小店密码** —— ⚠️**注意密码末尾可能带一个小数点 `.`**，复制时容易漏，登录前核对一遍。这是讲稿专门强调的坑。🟠
   - 口径：登录用的是「绑定手机号 + 小店密码」，不是手机号本身当密码。
5. 点**登录** → 点**发送验证码**。
6. **验证码获取方式（练手店特有）**：另有一个「获取手机号验证码的链接」，复制该链接 → 浏览器打开 → 页面上直接显示验证码（如示例 `9169`）。
   - 即：练手店的短信验证码不进真实手机，而是通过一个专门的取码网址查看。🟠
7. 把验证码填回登录页 → 点**登录** → 进入后台首页。

**避坑点：**
- 密码尾部小数点（最高频登录失败原因）。
- 验证码要去**取码链接**看，不是等手机短信。
- 全程**复制粘贴**网址/手机号/密码/链接，别手输。

## 2. 店铺授权货代（货叮咚）SOP

> 目的：把店铺**授权给货代 ERP（货叮咚 = "货叮咚"，讲稿语音转写写成"或叮咚/或京东/或东"，均指同一工具）**，之后发货、面单、物流可在货代系统里统一操作。
> 同类工具（妙手、其它货代 ERP）授权逻辑类似：**同浏览器同时登录两边 → 在 ERP 里发起授权 → 跳转店铺后台勾选确认**。

**前置条件（关键）：**
- ⚠️ **必须在同一个浏览器里，同时登录 ①店铺后台 + ②货叮咚账号**。
- 若只登了货叮咚、没登店铺后台 → **授权页无法跳转/不显示**，这是最常见的卡点。✅（讲稿反复强调）

**步骤（货叮咚侧发起）：**
1. 登录货叮咚，左侧菜单 → **授权管理** → **店铺授权**。
2. 进入店铺授权界面（首次为空白）→ 选平台 **TK** → 选 **全球跨境店铺**。
3. 跳转到「货叮咚授权店铺」配置页，填写：
   - **授权店铺**：选要授权的那个店。
   - **续保选项（授权期限）**：直接选 **无限期**。🟡
     （若显示"仅重新授权"，是因为之前授过、无需重授；新授权按正常填。）
   - **邮箱地址**：填自己的邮箱。
   - **联系电话**：选填，可留空。
4. 核对无误 → 点**确认安装** → 跳转到店铺侧授权界面。
5. **下滑到页面最底部 → 勾选授权同意框 → 点「授权」** → 显示**授权成功**。
6. 点击后自动跳回货叮咚，界面应出现该店：**店铺名称 / 授权类型 / 店铺类型 / 国家地区 / 授权时间**，状态 = **生效中**。
   - ⚠️ 若货叮咚里仍空白无显示 → **授权未成功，需重新授权一遍**。
7. （可选）店铺**别名**可点编辑，备注成自己习惯的名字，方便多店区分。

**避坑点：**
- 同浏览器双登录是硬前提（头号坑）。
- 授权页要**滑到底部勾选**才出现授权按钮，别在中途找不到按钮就放弃。
- 授权后**回 ERP 核对状态=生效中**，不要默认成功。

## 3. 绑定收款账号（连连国际）SOP —— 店铺侧操作

> 范围：本节是「**在店铺后台把已有连连账号绑为结算账号**」的点位流程。
> ⚠️ **连连账号本身的注册 / 实名 / 绑银行卡**不在本节（本批对应 pptx 为空白，见文末缺口）。**收款服务商怎么选、费率、账户持有人须与店铺主体一致** → 见 `入驻开店实操与政策.md`。

**步骤：**
1. 登录店铺后台 → **商家资料 → 支付信息**。🟡
2. 点**绑定结算账号** → 系统要求先**设置（提现/结算）密码**：
   - 先**接收验证码** → 输入收到的验证码。
   - 设置密码：⚠️**不要用连续或重复的数字**（如 123456 / 111111），否则不通过。
   - **提交**。
3. 再点一次**绑定结算账号** → 输入刚设置的密码 → 跳转到**提现方式**界面。
4. 在提现方式里**选「连连国际」** → **接收验证码** → 填入 → 点**绑定连连国际账户**。
5. 弹出连连登录 → **登录你的连连账号** → 页面显示确认绑定 → 点**确认绑定**。
6. 显示绑定成功 → 店铺即已绑定连连国际，可用于回款提现。

**避坑点：**
- 结算密码**不能连续/重复数字**（高频被拒原因）。
- 全程有**两次验证码**（设密码一次、绑账户一次），分清别填混。
- 绑定前**连连账号要已注册并完成实名/绑卡**，否则到第 5 步无法确认绑定。

## 4. TK 小店后台导航地图（九大板块 · "在哪点什么"）

> 用途：快速定位功能在后台哪个入口。**分析方法论不在这里**（数据罗盘怎么分析 → `08-数据分析/`；联盟带货策略 → `05-流量增长/达人联盟.md`）。本节只回答"功能在哪个菜单"。

**卖家中心左导航栏一排功能按钮：**
商品 · 订单 · 发货 · 营销中心 · 店铺页面 · 资产 · 店铺广告 · 购物视频 · 成长中心 · 我的账号 · 账号诊断。
讲稿口径——**日常用得最多**：数据罗盘、联盟带货、商品、订单、营销中心、店铺页面、成长中心。🟠

| 要做的事 | 后台入口路径 | 备注 |
|---|---|---|
| **装修店铺页** | 后台 → **店铺页面** → 编辑 → 预览 | 编辑后可看预览版 |
| **报名活动 / 提报商品** | **营销中心 → 营销活动** → 参加活动 / 提报商品 | |
| **看交易数据** | **数据罗盘** | 选时间段；看 GMV、成交件数、访客数、下单用户数；可看销售来源（直播/视频/商品卡） |
| **找热门品 / 选品** | 数据罗盘 → **商品卡 → 热门** | 平台当下推荐热门品 |
| **标题优化 / 热搜词** | 数据罗盘 → 商品卡 → **商品标题优化工具** | 复制热搜词→翻中文→判断是否加进标题 |
| **关键词榜单** | 数据罗盘 → 商品卡 → **关键词榜单** | 可选时段（今日/周/月）、渠道（热销/标准/高潜）、类目（男/女/童装/家居/户外…） |
| **店铺页面流量分析** | 数据罗盘 → 商品卡 → **店铺页面** | 看装修核心数据 + 流量来源渠道 |
| **单品 / SKU 销售** | 数据罗盘 → **商品详情信息** | 今日/7天/28天；GMV、成交件数、订单数、库存 |
| **营销工具效果** | **营销中心 → 推广工具** | 看各优惠券/秒杀/折扣分别带来多少业绩 |
| **新老客占比 / 复购** | 数据罗盘 → **客户分析 / 排行榜** | 新客 vs 老客下单占比 |
| **客服回复表现** | 数据罗盘 → **售后 → 即时消息表现** | ⚠️ 客服须 **24 小时内响应**，多次超时影响店铺评分（见下） |
| **差评分析** | 数据罗盘 → **售后 → 差评表现** | 看买家差评原因 |
| **直播 / 视频数据** | 数据罗盘 → **直播表现 / 视频** | 28天/7天；GMV、成交订单/用户/件数、点击成单率、曝光点击率 |
| **达人带货（联盟）** | 左导航 **联盟带货** | 四板块：合作 / 查看 / 样品申请 / 联盟订单 |
| **学习中心** | 后台 → **学习中心** | 搜"运费/佣金/交易手续费"等可查官方说明 + 课程 |
| **规则中心** | 后台 → **规则中心** | 查跨境新店考察期、入驻被拒解读、注册规则、联盟/资金/风险/售后履约规则 |

**联盟带货四板块要点（仅入口，策略见达人联盟文件）：**
- **合作 → 公开计划**：广撒网，打开开关、自动添加商品、设佣金 + 免费样品库存；可批量调佣金。
- **合作 → 定向合作**：指定邀约达人，一条消息一个达人，可多次建联。
- **发现达人 → 查找达人**：按类目/粉丝量/表现（GMV、成交件数）筛选建联；榜单有带货能力榜/快速成长榜/内容销量榜/内容热度榜。
- **样品申请**：按达人**履约率、销量**决定是否出样；⚠️**建议只给履约率 ≥100% 的达人出样，防白嫖**。🟠
- **联盟订单**：看时段内达人转化订单 + 对应佣金。
