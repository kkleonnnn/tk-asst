## 一、四阶段成长模型（增量：把"考察期/破零"串成完整曲线）

> 与现有 `行业背景与经营方法论.md §6.1` 和 `新商必修课时1-8.md` 的"考察期/成长任务"**互补**——那里讲单点，这里给**全周期 + 每阶段量化目标**。⚠️ 数字均为培训口径，**以官方后台考察期页面实际指标为准**。

| 阶段 | 别名 | 核心目标（培训口径） | 运营重点 |
|---|---|---|---|
| ① 冷启动期 | 学习期 | **从 0 到 1 出首单** | 选好款、做对比、学会分析；尽快成交第 1 单 |
| ② 新手期 | 搭建期 | 过新店考察期：**30 天内 ~500 单**（讲稿称"出新手村"最核心指标） | 系统学课 + 实操搭建，别在玩法选择上纠结；快速建立手感 |
| ③ 成长期 | 出单期 | **日均 $200 销售额**（≈¥1400–1500/天） | 用新店**前 50 天流量扶持**快速出单；上 50 个新品、测款、优化标题/详情/视频/价格 |
| ④ 爆单期 | 温饱期 | **日均 $500 以上**（≈¥3500+/天） | 达人放量打爆款、持续上新迭代；开始**多站点布局**（新马泰越菲） |

**关键时间锚点（培训经验值，非官方）**：
- **前 20 天最关键**：大部分商家"商品→出单"约需 **20 天**；超过 20 天没出单要自查哪步没做好。
- **首单来源**：绝大部分新商/新店的**首单来自货架场域**（商城/搜索），所以**货架场必须先运营好**（呼应批次1多场域框架）。
- **新店两大红利务必抓住**：① 前 **50 天流量扶持**；② **成长任务激励**（见 §6）。

## 二、11 步全案运营导图（增量核心：最完整的做店主线）

> 来自《做TK小店的全思路》。批次1 `起店思路与新手指南.md` 的"六大模块"是精简版；**这 11 步是展开版**，可作为新店 SOP 主干清单逐项打勾。

`①定类目 → ②入驻 → ③选品 → ④上架 → ⑤店铺装修 → ⑥测评 → ⑦打造爆款 → ⑧中期打爆款 → ⑨建联达人 → ⑩爆款迭代 → ⑪优化转化`

### ① 定类目 4 要点
1. 配合市场趋势做调研，选**需求高、潜力大**的；
2. **高热度 + 低竞争**（看 TikTok 上近期有没有爆款视频、看同类目竞品商家多不多）；
3. 选**易于视频/直播展示**的品；
4. 选**适合年轻人**（34 岁以下主力人群）的品。
> ⚠️ 与现有规则一致提醒：**一个账号只能选 1 个主营类目、全站点只发该类目**（见批次1 `_消化日志`）。

### ③ 选品 6 型（增量：比现有更细的"选品类型分法"）
| 类型 | 说明 | 操作要点 |
|---|---|---|
| 季节性 | 跟当地气候 | 东南亚：小风扇、防晒头巾/衣、短装；**提前布局** |
| 节日型 | 跟当地节日 | 圣诞（红色元素/圣诞树/帽）、开斋节、泼水节；**协议时效短，别大量备货** |
| 流量型 | 蹭热度、新奇有趣、价格亲民 | 依赖当地热榜视频/高互动内容，模仿热门并优化 |
| 平稳型 | 无季节性、复购高 | 毛巾、垃圾袋、手机壳；优化链接+引流内容做长期 |
| 数据驱动 | 工具选品 | 后台"商品机会"、Google Trends、亚马逊热销、第三方工具 |
| 社媒/平台工具 | 监控表现 | TikTok 自带分析 + 外部工具（讲稿点名"卡罗数据"等） |

**"三好产品"标准（建联达人/重点推广前的选品门槛）**：**质量好（基础）+ 价格好（前提）+ 服务好（保障）**。重点推广的品**先优化好再找达人**，且**名称/主图/价格要提前算好、把达人佣金提前算进售价**。

### ④ 上架 4 要点（红线，呼应批次1 `新商必修课时1-8 课时6`）
- **标题**：含品类、材质、核心卖点、特性、数量等关键词，简洁明了；
- **禁止**：营销/促销词、与产品无关的描述、未授权品牌/冠名；
- **图片**：尽量纯白底或主体突出无杂乱背景，**像素 ≥ 800×800**；禁多张相同图；主图正面、其余多角度；
- **一个商品详情页只展示一款产品**，禁一页多品、禁复制商品页。

### ⑥ 测评 3 阈值（增量：明确的投放达标线）🟡
- **点击率 CTR ≥ 2%**
- **转化率 CVR ≥ 1.5%**
- **ROI（投资回报）**：用 **GMV Max 广告**初期可接受 **≤5（微亏当投资）**，后期须正向，目标 **1:7 ~ 1:9 甚至更高**。
> ⚠️ 经验阈值，实际随类目/客单价/站点差异很大，**以自己店铺真实数据校准**。

### ⑦/⑧ 打造爆款
- **上新节奏分前中后期**：前期**每天上新 ≥10 款（最好 20–30 款）**保持 1–3 个月 → 后期数据稳定后**一周约 5 款**，重心转为"一门心思打爆款"。（⚠️ 高频上新=铺货式，灰区，自评）
- 必参加平台促销/大促活动积累权重；
- 建数据监测体系（曝光/点击/转化/销售额/客单价/复购率）持续优化。

### ⑨ 建联达人（增量要点，主体见批次1 `05-流量增长/达人联盟.md`）
- **两种合作**：公开计划（达人在选品广场主动找你，细水长流）+ 定向计划（你主动邀约特定达人）；
- **寄样筛选维度**：类目高度匹配、粉丝画像相似、近期勤奋度（直播/短视频频繁）、**履约率 ≥ 90%** 才寄样；
- **建联工具**：东南亚常用 WhatsApp / 邮箱 / 当地 IM；用**打底/批量工具一天可邀约几千上万达人**（讲稿提"打密"，应指批量邀约工具）；
- **跟进时点**：**样品物流送达当天**就联系达人沟通带货/直播（效果最佳）。

### ⑪ 优化转化 2 抓手
- 商品页吸引力：清晰描述 + 高质量图/视频（360°视角）+ **社交证明（多展示好评/UGC）**；
- **限时优惠 + 倒计时**：页面显示倒计时，临门一脚促决断。

## 三、做店前实操准备（增量最密：第1课"磨刀"清单）

> 主体与批次1 `新商必修课时1-8` 重叠，但**货代/货叮咚/连连/飞鸽**这几块讲得**更细且有避坑参数**，下面只记增量。

### 3.1 账号与后台入口（速记）
- **登录推荐用邮箱**：用手机号注册一个 **163（网易）邮箱**便于取验证码；注意"邮箱登录"输入的是**TK小店密码**，不是邮箱密码。
- **政策通知走邮箱**：增值税/运费/佣金/手续费变更**都通过邮箱通知**，做 TK 必须养成查邮箱习惯；登录后**尽快开启邮箱安全验证**。
- **后台关键路径**：右上角店铺头像 → **"我的账号 / 商家资料"**，下含：商家信息、店铺信息、仓库与取件地址、假期设置、支付信息、订阅设置、账号安全、关闭店铺。**几乎所有基础设置都在这里改**。
- **保证金路径**：后台"资产/财务 → 保证金 → 选站点 → 充值（手机验证码 → 下一步）"。支付宝扫码付的是**人民币（按当时汇率折 90 美元）**，退还也原路退**人民币**。（金额/免缴政策见批次1，**马来新商免佣/免押以官方为准**）

### 3.2 货代 & 发货地址（★增量价值最高的避坑）
> 主体见批次1 `04-履约发货/发货指南与运费.md`，以下是**讲稿特有的实操参数**：

- **官仓位置**：TikTok 官方中转仓在 **东莞 / 义乌**两地，关仓（揽收）只给最多 **4 天（96h）**。
- **避坑·发货地址要"远"**：发货地址**必须填在东莞/义乌仓揽收范围之外**——否则系统可能只给 2 天，备货来不及。**这个"远地址"由货代提供**，凑足 96h 让你从拼多多/1688 采集到货。
- **目标时效**：最好 **72h（3 天）内**把货送到官仓，至少留 **6h** 给货代排队扫码；雾发物流一般 3 天到，**最迟别超 90h**。
- **面单选择（重要避坑）**：
  - **红梅面单** → 只能发**东莞**；
  - **万盛面单** → **就近原则**（货源近义乌发义乌、近东莞发东莞）→ **优先选万盛面单的货代**。
  - 选货代时**直接问对方是红梅还是万盛**。
- **货代费可议价**：新用户一般 **2.5–3 元/单**（约 3 元居多）；单量起来后可谈到 **2–2.5 元**，老卖家/大单量能谈到 **1.5–1.7 元/单**。**"一毛钱也是利润"，务必谈。**
- 选货代另一条件：**优先选东莞、义乌都有仓的货代**。
- 货代还能在小规模屯货时**提供免费仓库**。

### 3.3 货叮咚（采集+发货工具）—— 选型与绑定
- **货叮咚 vs 妙手（讲稿推荐货叮咚）**：
  1. 货叮咚可**直接采集拼多多**，妙手目前**局限 1688**；
  2. 批量发货时**妙手需导出 Excel 手动操作**较繁琐，**货叮咚可直接批量**。
- **注册**：百度搜"货叮咚"→ 注册（有免费试用期）。
- **绑定货代**：货叮咚后台 → **订单 → 货代管理** → 选服务平台（TikTok）+ 站点 → 在货代列表里选一个**开通货代 + 授权**；
  - ⚠️ **货代服务系统初始密码 = 注册手机号**（如手机 1888888 8888，初始密码也是它），**开通后请及时改密码**。
- **授权店铺**：货叮咚 → 授权管理 → **店铺授权 → 选 TikTok（跨境）** → 填邮箱地址 + 联系代码 → 确认安装 → 授权成功。
  - **采购发货推送的店铺，必须与货叮咚授权的店铺一致**才能接单；货代余额不足会**推单失败**，需及时充值。
- 绑定后即可：① 直接采集上架到店铺；② 在"订单"板块批量处理发货。

### 3.4 连连国际（回款链路）
- **资金链路**：消费者付**泰铢** → TK 平台把泰铢提现到**连连国际换成美元 USD** → 你在连连把 USD 提现到自己账户**变回人民币**。**美元是中间桥梁**。（做马来即 MYR → USD → RMB）
- 连连需**实名认证**（大户账户为主），审核 1–2 工作日、邮件/短信通知；认证后**添加银行账户**作收款账号，回款一般以**美元**到账连连。
- **双向校验**：在 TikTok 后台看绑定成功后，**还要去连连国际后台核对绑定的是不是目标店铺**。
- 后台路径：商家资料 → **支付信息** → 添加提现方式选"连连国"→ 跳转连连国际登录绑定。

### 3.5 飞鸽客服系统（即 TikTok 客服系统，必做设置）
> 设置主体批次1已提，这里补**讲稿强调的开关清单**：
- 入口：后台右上角**信息标识 → 选站点 → 自动化工具**。
- **自动问候语**：必设，且**用当地语言**（泰国设泰语）；设好**打开开关**。
- **自动提供最新消息**：4 种订单状态**全部打开**（系统自动按订单状态推送给买家，免手动）。
- **聊天设置 → 翻译设置：打开**（买家本地语 ↔ 中文自动互译，新手必开）。
- **在线时间/工作时间：务必设**——明确客服时段后，买家在非工作时段咨询即使投诉，**平台一般不判你责任**；不设则可能被判责、扣店铺分。
- **快捷回复/模板**：多建分组（如"从哪发货""拍什么尺码"），不会就问豆包 AI / 查网 / 问导师；发送时模板会自动翻译再发。
- **24h 响应率尽量做到 100%**：超时（尤其严重超时）扣店铺分，前期影响大；做好了转化率也明显提升。

## 五、稳定期 5 维度（增量：现有文件较少展开的"守成"阶段）

来自《做TK小店的全思路》第三大维度，店铺起来后维持竞争力：

5. **维护店铺安全稳定**：**避免频繁改店铺设计**（易触发安全审核）、操作合规防受限/封禁。

## 六、成长任务 & 明星店铺（增量：领取策略 + 明星店铺数据）

> 主体见批次1 `新商必修课时1-8 课时8`。本批新增**两个实用点**：

### 6.1 成长任务"拉长服务期"领取法（实操技巧）
- 任务入口：后台**成长中心 → 我的任务**（每天登录顺手刷新看新任务）。
- 任务类型：**免佣任务**（助新店启动竞争力）+ **新手成长任务**（覆盖可售/动销/冷启动全流程）。
- **⚠️ 别一键全领**：每个任务奖励的优惠券**仅 1 周有效期**。
- **拉长法**：分批完成——例如"5 个任务 / 21 天有效"，则**约每 4–5 天完成 1 个**，让 21 天里**始终有任务在做**。这样平台**持续关注你、持续给流量扶持和满减券等权益**。

### 6.2 明星店铺（卖家奖励计划）
- **是什么**：对运营突出、达成承诺的卖家授予徽章的奖励计划。
- **达成条件（方向，🟡以官方为准）**：履约订单数 + 订单总价值 + 买家数量 + 店铺评分/违规分达标，**无虚假交易/违规、且已过新手村**。
- **权益**：明星店铺徽章、新卖家专属工具权限、**增加商品发布数量**、提升信誉/信任/转化、降低交易风险、增加曝光。
- **before-after 数据（讲稿案例，🟡仅参考）**：拿明星店铺**前** 25 天销售额约 ¥5.6 万 / 1175 单 → 拿到**后**仅 **12 天**销售额冲到 **¥20 万 / 4300+ 单**。结论：明星店铺对销量/订单增长影响显著。
