# 连连国际（LianLian Global）开户 · 实名 · 绑卡 SOP（骨架 · 待校长补官方细节）

> **适用范围**：TikTok Shop 马来西亚跨境店（从中国发货）的**收款开户**——即"连连账户本身的注册 + 企业实名 + 绑定提现卡"。开完户后"在 TikTok Shop 后台绑定连连收款"那步见 `后台配置-登录授权与收款.md` §3（本文件不重复）。
> **时间基准**：2026-06-29 骨架。**可信度：🟡**（连连官网 + 多家跨境导航多源整理，**非官方一手**；步骤/材料/周期会变，一律以连连官方 `global.lianlianpay.com` 当期为准）。
> ⚠️ **这是骨架**：主流程与材料清单已就位；**校长请用连连官方注册教程补当期截图/细节**（尤其 TikTok Shop 专属对接点），补完由 kk 整合定稿、去掉"骨架"标记。
> 🔴 **前置依赖**：连连开户用的**主体/法人**必须与 **TikTok Shop 店铺注册主体/法人**一致——所以要**先定"用谁的执照/法人"**（见 `sop/开放缺口清单.md` 的归属项）；归属没定就开户，主体不一致要返工重开。

## 0. 一条红线（最重要）
- **收款账户持有人 = TikTok Shop 注册主体/法人 = 连连开户主体，三者必须同一。** 连连**不支持提现到他人名下**（合规要求）。
- 大陆企业主体：可提现到**大陆对公账户** 或 **法人境内对私储蓄卡**；香港主体按连连香港口径（⚠️ 待核）。
- 顺序：**先定归属（用谁的执照/法人）→ 再开连连户 → 再回 TikTok 后台绑定**。

## 1. 准备材料（大陆企业主体）
- 三证合一**营业执照**（清晰彩照、无水印，单张 <5MB）
- **法定代表人身份证**正反面彩照
- **提现银行账户**：企业对公账户 或 法人境内对私储蓄卡
- 有效**邮箱** + 可接码**手机号**
- ⚠️ 香港主体 / 个体户主体 的材料差异——待校长按连连官方补

## 2. 开户注册 SOP（连连侧，🟡 以官方为准）
1. 打开 **`global.lianlianpay.com`** → 右上角 **注册**。
2. 选 **企业用户** → 填邮箱 + 验证码。
3. 手机短信 / 邮箱认证。
4. 设置 **登录密码、支付密码、安全保护问题**。
5. 进入 **企业实名认证**（上传营业执照、法人证件等，见 §3）。
6. **绑定境内提现银行卡**（对公 或 法人对私，见 §4）。
> ⚠️ 各源对步骤顺序表述略有出入，**以连连官网当期页面为准**；校长补官方截图。

## 3. 企业实名认证
- 上传三证合一营业执照 + 法人身份证正反面，填企业信息。
- 材料清晰无水印、<5MB。
- **审核周期**：约 **1 个工作日**，短信/邮件通知结果。（整体开户周期一般约 5 天——⚠️ 待核）

## 4. 绑定提现银行卡 + 提现
- 绑定境内提现卡（对公 / 法人对私）——**持有人须与主体/法人一致**、不得他人名下。
- 到账：通常 **1–2 个工作日**；开户 **免费**。
- ⚠️ 连连收款/结算币种、费率、与 TikTok Shop **USD 结算**的对接细节——待校长补（TikTok 东南亚结算币种为 USD，见开店向导 §3 收款部分）。

## 5. 连回 TikTok Shop（开完户之后）
- TikTok Shop 商家后台：商家资料 → 支付信息 → 绑定结算账号 → 提现方式选 **连连国际** → 验证。
- **详细店铺侧绑定步骤见 `后台配置-登录授权与收款.md` §3**——本文件只管"开连连户"。

## 6. 常见坑 / 待校长补的官方细节
- ⚠️ 主体/法人/收款卡三者不一致 → 支付/提现失败或返工（见红线）。
- ⚠️ 香港主体、个体户主体的开户差异（材料、可绑账户类型）。
- ⚠️ TikTok Shop 专属：连连给的**收款账户信息**具体填进 TikTok 后台哪几个字段、USD 结算如何对接——补当期截图。
- ⚠️ 各步骤当期 UI/顺序/周期，以 `global.lianlianpay.com` 为准。
