## 1. 平台是什么（认知打底）

- **TikTok Shop（又称 TikTok 电商）**：把商家、达人、买家聚到一起的电商平台，可通过短视频、直播、商品展示直接卖货；购物路径短，"刷到即买"。可类比**国内抖音小店**。〔来源 6、8〕✅
- **东南亚已上线市场**：新加坡、马来西亚、泰国、越南、菲律宾（**跨境招商覆盖这 5 国**）。印尼本土店"暂停开放"。〔来源 6、8〕✅
- **两种店铺类型**〔来源 8〕：
  - **跨境店 √（我们走这个）**：中国大陆 & 香港企业营业执照；有跨境资质要求；收款用派安盈(Payoneer)等；走跨境物流小包（可开 COD）。
  - **本土店**：需本地执照/身份证/护照 + 本地电话卡 + 本地银行卡 + 海外仓，门槛高，不适合我们。
- **两种生意场（核心模型，务必记牢）**〔来源 6〕✅：
  - **内容场**：靠直播 + 短视频**激发**需求 → "货找人"。
  - **货架场**：靠 TikTok Shop 商城**承接**主动需求 → "人找货"。
  - 平台主张 **内容 + 商城双驱动**：商城在 App 主页一级入口给流量保障；内容场（达人/直播/短视频）滚雪球放大销量、孵化爆款。
- **自运营模式**：商家自主选商城/达人/直播/短视频，自主定价、参加营销活动；适合有电商经验的商家。〔来源 6〕

## 2. 注册入驻 SOP（线上注册五步）

> 〔来源 1〕这是东南亚跨境店注册的**操作级流程**，可与开店向导第 3 节互为补充。

1. **打开注册入口** → `https://seller.tiktokglobalshop.com/account/register`，点"注册"。
2. **注册账号**：手机号 + 邮箱 + 密码。
   - 手机号收验证码、邮箱收验证码分别填入。
   - **密码规则**：须含数字 + 字母 + 特殊字符，**6–20 位**。〔🟡 以页面提示为准〕
3. **填入驻信息，选东南亚市场**：
   - 目前该注册方式**仅支持东南亚 5 国**（新、马、泰、越、菲）；可开一个或多个国家，**建议按实际运营计划申请开通售卖国**（别贪多）。
   - 入驻模式选**普通入驻**；按执照所在地选主体：**中国内地**（提交营业执照）或**中国香港**（提交公司注册证明书 + 商业登记证）。
4. **上传资质 + 人脸识别**：
   - 上传营业执照，系统自动识别带填，识别失败/有误处手动改。**证件图不留边框、信息清晰、不可有水印！**
   - 中国内地法人做**支付宝扫脸**实名：选"中国内地身份证"上传 → 填资料点验证 → 手机支付宝扫码 → 手机端确认法人信息、勾协议、开摄像头扫脸。
   - ⚠️ **中途关闭弹窗 = 放弃验证**，可重新点验证按钮。
5. **完善店铺信息**：
   - **店铺名称用英文填写（拼音也可）**。
   - **选主营类目**：勾与店铺一致的主营类目。资料推荐入驻类目：**男装、运动和户外、时尚配饰、汽车和摩托车配件、家居用品、厨房用品**。〔来源 1，🟡 仅为彼时招商推荐〕

> ⚠️ **与开店向导的衔接**：审核通过后须到「资产 > 保证金」缴保证金才能上架。收款账户持有人**必须与注册主体/法人一致**。详见 `马来西亚-跨境店-开店向导.md`。

## 3. 物流 / 收款 / 全球后台（一站式方案）

### 3.1 三种物流模式〔来源 2、3〕✅
| 模式 | 说明 | 适合 |
|---|---|---|
| **跨境物流小包（新手默认）** | 商品自寄/免费揽收到**国内中转仓** → 平台负责清关 + 尾程派送 | 从中国发货起步 |
| **自有海外仓发货** | 跨境店 + 自有仓当地发货，尾程由平台服务商承接 | 有本地备货能力 |
| **官方认证海外仓发货** | 跨境店 + 官方认证仓当地发货，尾程由平台服务商承接 | 走官方仓 |

- **国内中转仓 5 个**：上海、泉州、义乌、东莞、香港。〔✅，与开店向导一致〕

### 3.2 收款方案〔来源 2〕
- 交易中平台**托管货款**，交易成功后通过第三方支付打款给商家。
- **支持的第三方支付伙伴**：**PingPong、连连支付(LianLian Pay)、Payoneer(派安盈)、空中云汇(Airwallex)**。〔🟡 以后台可选为准；9 号件单提连连，8 号件提派安盈，实际为多选〕
- **结算链路**：消费者下单 → 商家打单发货 → 跨境物流(5–7 自然日) → 消费者收货 → 售后期(7 自然日) → **售后期后 +1 天结算到账**。〔🟡 新商"0元试运营"豁免期内售后期会被拉长，详见开店向导〕

### 3.3 全球商家后台（Global Seller Center）〔来源 2〕✅
- 后台：`https://seller.tiktokglobalshop.com/homepage`（**东南亚跨境目前仅有网页端，无 App**）。
- 优势：一站式管理多站点多店铺；**全球库存合并管理**；快速切站点；**商品一键发布到多站点并自动翻译为当地语言**；一站式对接达人。
- **多市场客服**：后台 > 店铺客服自带**中文自动翻译**，无需外部工具即可与买家沟通。〔懂中文即可，无需英文〕

## 4. 开店成本（⚠️ 旧费率，仅留结构，数字以开店向导/后台为准）

> 〔来源 3〕成本结构 = **保证金 + 平台佣金 + 交易手续费 + 跨境物流费 + 活动服务费**。结构✅；**数字已过时，下表标注冲突**。

- **保证金**：90 USD/店铺，不分类目。〔✅ 与开店向导一致〕缴纳路径：后台 > 资产 > 保证金 > 充值，**推荐用支付宝**支付，收银台会明示汇率与手续费。
- **计费公式**〔✅〕：
  - 平台佣金 = 佣金率% × 商家折扣后价格
  - 交易手续费 =（买家支付费用 − 买家退款）× 交易手续费率
- **马来西亚（旧值，已变）**：佣金按子类目 **0%–4%（不含税）**；交易手续费 **2.16%（含税）**，2024-03-01 起生效。
  - 🔴 **冲突提醒**：开店向导记载**马站交易手续费已于 2024-09-05 上调到 3.78%（含 8% SST）**，且类目佣金 2025-09-13 起再上调。**本文 2.16% 已作废，仅作历史参照。**
- **马来西亚新商免佣**〔来源 3，🟠 各源不一〕：资料称"自佣金规则上线起入驻未满 30 天的商家"享 **90 天（自然日）免佣**，且**马来西亚入驻无需做任务即可享免佣**（其他市场如泰国需在任务中心完成"发布 10 个商品"才享 90 天免佣）。开店向导记为 90/120 天说法不一 → **以后台招商活动实时为准**。
- **活动服务费**：仅**参加活动时**才收（商城周促、超级运费券等），不参加不收，费率按活动而定。〔✅〕

> 其余市场旧费率（仅留档，非马站不展开）：菲律宾佣金 0.7%–3.7%、手续费 2.24%；新加坡固定佣金 2.18%；泰国固定佣金 4%、手续费 3.21%；越南固定佣金 2%、手续费 4%。〔🟠 均为 2024 旧值〕

## 6. 实操要点与避坑（来自 TK 小店项目介绍）

> 〔来源 8〕这份"项目介绍"虽是引流性质，但夹带了**真实可用的实操点**，单列出来：

- **小店三大优势**：① 免费流量利用效率高（但视频爆了若流向非小店开通国，流量浪费 → **呼应第 2 节"按运营计划开国"**）；② 操作成本低（基本无需店铺装修，ERP 上架快）；③ 收款方便（无需 Stripe/PayPal，派安盈等即可）。
- **引流/获客渠道**：短视频挂小黄车、直播挂小黄车、商品橱窗、**邀请达人带货（给带货佣金）**、付费投流、自然流量、店群 + ERP 批量上品（可规模化复制）。
- **新手三步走**：① 定测试品类、上架同时（有余力）发视频/找达人引流，**有预算一定学付费推广（最直接有效的出单方式）**；② 盯官方商品排行榜研究爆款需求；③ 做好日常优化 + 提前备战双 11/双 12 等大促。
- ⚠️ **新手村限制（重要实操避坑）**：
  - 新店**每天订单上限 80 单**。
  - **解除条件**：达到单日 80 单上限 + **15 天内保持各项指标健康**（延迟履约率、差评率、违规分都低）+ 备齐库存打包视频/ERP 操作视频等"真实经营"证明 → **发邮件申请取消**新手村限制。
- ⚠️ **其他坑**：物流"虚假签收"导致差评；务必吃透平台规则（禁售限售、成本核算、发货、流量）。
- **成本核算别漏项**：物流、运营、货品、**VAT/税**、广告投流、银行费用、平台佣金、**退货损耗**——算利润时全要计入。

> 〔来源 9〕该件为"招代理/招学员"话术（强调"不用拍视频、不用懂英文、一件代发拼多多货源、找货代贴国际面单一单 2–3 元"等），**营销夸大、对自营参考价值低**，仅记两点中性事实：① 后台为中文、可设自动回复；② 收款示例用连连支付。其"一件代发"模式与本项目自营定位不符，不采纳。
