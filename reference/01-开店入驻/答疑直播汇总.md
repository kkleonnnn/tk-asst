## 一、发货原理（先建立全局认知）🟢

一条订单的流转链路（讲师反复强调）：

```
店铺出单 → 你去拼多多/1688/供应链采购 → 收货地址填【货代仓库地址】
       → 货代收货后贴单打包 → 送到【TK官方仓】扫描入仓
       → TK官方仓分配物流承运商 → 送到买家手上
```

- **货代 = 物流承运商角色**：替你把国内采购的货集中、贴国际面单、送进 TK 官方仓。
- 关键时效卡点在**「送到 TK 官方仓被扫描」**那一步（见第四节）。

## 二、找货代 + 妙手授权（妙手版差异，补 `后台配置` 篇）🟢

> 完整"在 ERP 发起授权 → 跳店铺后台勾选确认"的逻辑见 `后台配置-登录授权与收款.md`。这里只记**妙手版**的具体入口与易错点。

### 2.1 选货代（讲师建议）
- 妙手后台 **左上角 `订单` → `货代管理`** → 列出大量货代服务商。
- **挑选维度**：送仓价格、送仓速度、服务态度。
- **价格行情**：🟡 约 **2.5–3 元/单**（培训口径，订单量大可议价砍价）。
- **避坑**：① 别只看价格最低的（服务质量可能跟不上，影响订单时效）；② 建议选**排名靠前**、**同时有东莞仓 + 浙江义乌仓**的货代（就近发货更快）。
- 货代手机号：在妙手货代列表**双击**可显示，合作前先电话/微信沟通；**不满意可随时更换**。讲师明确"货代不做推荐，仅演示参考"。

### 2.2 授权（妙手 ↔ 货代系统 双向）
1. 妙手货代列表选中货代 → `使用此货代` → `立即注册`（设用户名/手机号/密码/验证码）。
2. 注册后进入**货代系统** → 左侧 `授权管理` → 复制 **系统授权码**。
3. 回妙手，在「第三步」位置**粘贴授权码 → 提交** → 显示"添加完成"（并自动开启"自动对接货代系统"）；`我的货代` 里能看到该货代。
4. 回货代系统 → `授权管理` → `店铺授权` → 平台选 **TK** → `添加授权店铺`。
5. 跳转页面右上角可切**简体中文** → **先选 `亚洲商家` → 再选 `中国大陆和香港`**。
   - ⚠️ **最大的坑**：这一步**必须先选「中国大陆和香港」，不能直接点你要的站点**，否则授权不成功。
6. `下一步` → 选你运营的**站点**（如泰国）→ 选**授权期限**。
   - 🟠 **期限建议选 1 年以上 / 无限期**（可选无限期/1年/180天/90天）。期限太短会导致后续订单无法同步到货代、耽误时效。
7. 填邮箱（电话选填）→ `去授权` → 勾选 → 右下角 `授权`。授权成功后回 `店铺授权` 列表会显示**店铺 + 授权状态=正常 + 授权时间**；没显示就是没成功，需重做。

## 三、妙手代发采购 SOP（全新增量）🟢

> 核心特点：**妙手只负责"提交订单"，实际付款要自己用手机拼多多 APP 付**。这是讲师反复强调、新手最易漏的点。

1. 妙手 **左上角 `订单` → `订单处理`** → 显示店铺后台所有新订单。
2. 选中要采购的订单 → `代发采购`。
3. 弹窗顶部 **`选择服务商` = 货代的收货仓库**（如东莞仓/义乌仓）。
4. 下滑找到**货源 `下单链接` → `去采购`** → 弹提示（可忽略）→ 右下角 `继续使用拼多多货源下单` → 自动跳转拼多多，登录拼多多账号。
5. 在拼多多按**买家所拍规格**下单（讲师建议**参与拼单**，价格更优）。
   - ⚠️ **妙手翻译可能有偏差**（例："藕色"译成"粉色"）→ **对照买家商品图核对规格**再确认。
6. 核对**收货地址（=货代仓）+ 规格** → `立即支付` → 顶部出现"关联妙手 ERP 订单成功"。
7. **回妙手 `订单处理`** 刷新 → `已出货` 里能看到该单，拼多多订单状态显示**待付款**（因为还没真正付钱）。
8. **必须**用**手机拼多多 APP**：`个人中心` → `我的订单` → 找到**待付款** → `去支付`。付完拼多多订单状态才变**已付款**，商家才会发货。

### 关联/补充操作
- **手机拼多多单独下单（不走妙手代发）**：到货后回妙手手动**录单**——勾选订单 → `录入快递单号`（物流公司系统自动识别）→ `保存`；或批量：勾选 → `更多` → `提交待打包` → 录单号 → `保存`。
- **妙手用拼多多采购需装浏览器插件**（不需绑定/授权拼多多账号）：妙手 `插件下载`（右上角有"插件下载"四字即未装）→ 按浏览器选（谷歌/360…）→ 下载插件文件 → 浏览器右上角开**开发者模式** → 把插件拖进扩展页 → 开启 → 回妙手刷新；右上角"插件下载"四字消失即装好。
- **代发采购里只有 1688、没有拼多多链接**：点 `添加货源链接` → 粘贴拼多多商品链接 → 保存，即可用拼多多采购。
- **货源链接 = 商品详情页链接**（不是首页）。手机端：商品页右上角箭头 → `复制链接`。同一商家/店铺的货源链接基本固定。

## 五、订单时效与店铺健康（补 `起店思路` 篇细节）🟡

> ⚠️ 以下数字均为**培训口径**，务必以官方后台《延迟履约/订单履约》规则当期值为准。

- **96 小时 / 4 天铁律**：**从「买家下单」时刻起算**，96 小时内货物须被 **TK 官方仓扫描入仓**、订单状态更新为**「已发货 / 运输途中」**。
  - ⚠️ 计时起点是**买家下单**，不是你采购。**当天订单尽量当天采购发货。**
  - 超时 → 平台记**延迟履约**；延迟率过高 → **扣分 + 限单**。
- **查发货时效 / 物流**：店铺后台 `订单` → `管理订单` → `待发货 / 已发货`，看买家下单时间、点 `查看物流`（"已到达分拣中心"= 已被官方仓扫描）。
- **查违规扣分**：店铺后台左栏 `账号健康` → `店铺健康`。可见**违规积分**及三类违规：**政策合规违规 / 订单履约违规 / 服务指标违规**；以及 **12 小时响应率**（未及时回买家信息会计入绩效考核）。
- 🟡 **违规积分达 12 分** → 七天内无法参与新的大促/促销活动（培训口径）。
- 🟡 **被限单一般限 7 天**，到期后恢复（须指标改善）。
- **延迟履约率每周更新一次**（按周期统计）。
- 操作建议：**每周**登录看一次店铺健康；分时段查买家消息当天回（不必秒回）。

> ⚠️ **与现有规则的衔接提醒**：本项目主体是**马来西亚跨境店**，违规走 **48 分违规积分制**（见 01 板块 README），**不要套用美/英站 AHR 账号健康分**。讲师这里说的"违规积分/12分/账号健康-店铺健康"是其演示站(泰国)的后台口径，**马来具体阈值与节点处罚以马来后台 `学习中心` → 处罚节点当期为准**。

## 七、售前/售后沟通设置 SOP（9.18 全新增量）🟢

### 7.1 进入聊天 & 自动化设置
- 店铺后台右上角 **`信息` 按钮** → 选当前运营站点（如 TH 泰国）→ 进入与买家沟通页。
- 右上角 `自动化工具`：
  1. **自动问候语**：打开 → `设置`（可用系统推荐或自编）。⚠️ **语言必须是站点当地语**（泰国=泰语；**马来站=马来语**）。
  2. **自动回复（聊天机器人）**：打开 → `设置` → 排期选"始终开启"或"仅限工作时间以外"（按自己时间）。
     - **启用条件**：✅ 接收售前查询、✅ 接收物流查询；**❌ 不要开「接收售后查询」**——⚠️ 否则系统可能**自动给买家发退货退款申请、甚至自动同意**。
  3. **回复建议**（平台建议）：可打开参考。
  4. **自动提供最新消息**（4 种状态）：建议**全开**，系统自动推物流等最新信息，省手动。
- 右上角 `服务分析`：看**转化率、24 小时响应率、满意度**。
  - ⚠️ 不回买家消息**会被扣分**（计入考核）；及时回能提粘性、提转化。

### 7.2 联系单个买家
- `管理订单` → 目标订单右侧 `开始聊天` → 进沟通页。
- **直接输中文，系统自动翻译成当地语**发送；买家回的当地语也自动翻成中文。右侧状态栏可看该买家所拍订单 + 物流详情。
- ⚠️ **回复评价/留言要用当地语言**（别直接发中文，其他买家看不懂）。

## 八、退货退款处理（9.18 全新增量）🟢

### 处理流程
1. `订单` → `管理订单` → `管理退货` → 看买家退货退款详情。
2. **先问退货原因 + 要凭证**：商品破损/漏发等，**先让买家提供签收视频/实物照**核实，**不能买家说什么就是什么**。
3. **确属商品质量问题**：先沟通**补偿优惠券**；买家态度强硬坚持退 → **给退**（质量问题商家吃亏，避免差评）。
4. **判定恶意退款**：直接**拒绝退款** → 有异议找**平台客服申诉**。
   - 找平台客服：聊天页右上角入口 → **多发几遍"人工"** 转人工 → 申诉。
   - 辨别恶意退款：商品无任何质量问题却硬说有问题，大概率恶意；但**具体问题具体分析**。
5. **买家原因**（如自己尺码选错却选"商品发错了"）：先沟通补偿券；不接受就**拒绝申请**。
6. **平台自动通过了退款**：先问原因、尝试补偿券挽回；实在不行就退货；**退回的货有任何问题都可申诉**。
- ⚠️ **退货退到哪**：若货在 TK 官方仓，会退到**店铺后台设置的仓库退货地址**（`我的账号 → 商家资料` 可查/改，可设为货代地址）。
- ⚠️ 沟通**严禁侮辱性词语**（平台严令禁止，会扣分）。

### 8.x 自动批准退款规则设置（11.26 全新增量 / 重要）🟢
> 目的：**尽量不让平台自动审核通过退款，改为商家手动审核**。后台没设过的**及时去设**。

路径：`订单` → **`履约设置`** → `退货和退款` → **`自动批准退款`** → `新增规则`：
- **政策名称**：自定义（便于记忆）。
- **可接受的退货原因**：默认**全选**——⚠️ **要取消全选**，只勾极少见的原因（如"运费过高""需改收货地址"），**留一个即可**。
- **类目**：选店铺类目（如女装→"女装和女士内衣"）。
- **退款范围**：🟠 建议**往低设**（讲师例：10–15 泰铢；马来站换算为小额 MYR）。
- **有效期**：自定义起止时间或"立即生效" → `确认`。

## 九、新手期 / 60 天观察期 / 召回机制（9.18 全新增量）🟡

> ⚠️ 培训口径，机制与指标数以马来后台当期为准。

- **新手期内**：每日有**订单上限**（限单）。
- **出新手期条件**：**6 个指标同时达标**即自动毕业。
- **毕业后进入 60 天观察期**：订单**无上限**。
- **召回机制**：观察期内若**任一指标变差/不达标 → 自动召回新手期**（重新受限）；指标再次达标又自动毕业。
- **累计满 60 天**无违规 → 不再被召回（永久出新手期）。
- 启示：**出新手期后 60 天内务必死守指标**，否则反复进出。

## 十、其他答疑要点（速查）🟢/🟡

- **拆分发货**：一个订单 **TK 官方仓只能扫描一次**。能否拆分发货在 `管理订单` 看（可拆会显示），或问货代。
  - 一单多件分了不同货源/不同仓 → 若**能拆分**则各自送仓、两个都收；**不能拆分**则只能发一个。
- **同一快递单号拦截退回的坑（11.26 重点）**⚠️：已发货途中买家取消、想拦截退回时，**先确认商家是不是把多个订单用「同一个快递单号」一起发的**。
  - 若是同单号合发：**直接拦一个会连累其他订单/包裹**（讲师案例：拦一单结果"全部召回"）。正确做法是**等货代签收后，再单独让货代拦截/退回**那一件。
- **取消订单**（时效赶不及）：**仅限只有一两单**时——优先联系买家取消重拍；买家不回再自行取消，**原因必须选「买家地址无法送达 / 无法送达客户地址」**。路径：`管理订单` → 目标单 `更多操作` → `取消` → 选原因 → 继续。
  - ⚠️ **切勿大批量取消**：易被判**虚假交易、冻结店铺**。量大时发飞书群让助教评估。
- **删差评**：商家**不能自行删评**。若是**恶意评价**→ 提供证据找平台客服举报，核实后平台会下架该评价内容。
  - 改善差评：联系买家了解原因、补偿优惠券请其追评；**邀好评话术要委婉**（别直说"给我好评"，平台会检测、会扣分）。
  - ⚠️ 平台**只记录买家首次评价**：买家事后改成五星，**对店铺权重的影响仍在**（但能影响后续买家下单信任感）。
- **搁置/隔离订单**：不想让货代打包送仓 → **在货代系统**找到该单 → 勾选 → `隔离订单`，或直接告知货代。妙手一般不会自动搁置（多为误触）。
- **发错货（供应链发错）**：先找供应链客服要打包视频/证据核实；属实则先与买家协商补偿券/能否签收，不行先给买家退，再与供应链协商赔付。
- **货源成本对比**：1688 常**两件起批**(有数量门槛，不占优)；**供应链 > 拼多多**(供应链更稳、质量更有保障)，**供应链有货优先供应链**。
- **就近原则**：1688/拼多多商家在浙江发货 → 寄合作货代的**义乌仓**，时效更快。
- **全网无货**：店铺出单品全网无货**及时同步飞书群助教**，平台侧会找货源，没有就调库存/下架，避免出单发不了货拖累时效。
- **快递单号没自动同步到货代**：妙手 `货代管理 → 我的货代 → 编辑`，确认**「自动对接货代系统」已开**。
- **妙手消息提醒/新旧版差异**：找妙手客服确认（右上角有客服入口）。
- **供应链推单**：操作较复杂，找供应链客服要视频/PDF 教程或请其远程演示。
