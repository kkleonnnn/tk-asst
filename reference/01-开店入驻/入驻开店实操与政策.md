## 一、入驻资质门槛（核心）

平台开放两种主体入驻：**企业营业执照** 与 **个体户营业执照**。

### 1. 企业营业执照（推荐路径，门槛最低）

- 接受 **中国大陆或中国香港** 注册的合法企业执照（有限公司 / 个人独资企业）。
- **接受无任何第三方电商平台经验入驻**（这是企业主体最大的优势）。自 2024-06-13 起，新增"无电商经验的传统内外贸商家"也可用企业执照入驻。
- 必填材料：**法人信息 + 实名认证 + 企业营业执照信息**（仅此两类，不需要店铺经验/评分）。
- 香港执照需额外填写"唯一业务识别码"（即商业登记号前 8 位）。

### 2. 个体户营业执照（门槛较高，需经验或定邀）

需满足下列 **全部** 条件：
- 中国大陆注册的合法个体户执照；
- 法人信息 + 实名认证；
- **同执照或同法人名下** 的主流跨境/国内第三方平台经营经验；
- 店铺综合评分 **≥ 4.3 分 或 ≥ 90%**。

**无经验个体户**：可走"平台定邀"通道，**无需三方经验和评分**即可入驻（需填问卷、等审核、拿邀请码，见第三节）。

### 3. 资质通用红线（企业 / 个体户都适用）

| 项 | 要求 | 可信度 |
|---|---|---|
| 执照有效期 | 必须在有效期内，**不可为已注销执照** | 高 |
| 注册时间 | **注册满 14 天后** 才能入驻；新执照因总局信息更新延迟，建议办成功后**等 7 个工作日**再入驻 | 中-高（见⚠️冲突1） |
| 经营状态 | 不可有"经营异常"，有异常需先到工商局解决 | 高 |
| 实名认证 | 2024-09-10 起新增法人"实名认证"模块，**大陆户籍法人必做** | 高 |

> ⚠️ **冲突1（务必以官方为准）**：个体户注册时长门槛在源资料内部不一致——`操作指南`/`招商政策`/`常见问题` 均写 **"> 14 天"**；但 `内贸出海手册` 的"入驻基本要求"表写 **"注册日期 > 6 个月"**。两者差距很大，**个体户入驻前必须在后台/官方公告确认**当前到底是 14 天还是 6 个月。企业执照统一为 14 天，无此歧义。

## 二、入驻六步 SOP（来自《跨境商家入驻操作指南》，最权威的流程）

**官方注册入口**（⚠️ 源资料给了两个不同域名，以后台实际跳转为准）：
- 操作指南：`http://seller.tiktokshopglobalselling.com/settle/start`
- 内贸手册：`https://seller.tiktokglobalshop.com/account/register`

### 第一步｜注册账号
- 用手机号 + 邮箱注册。**坑点**：若该手机号/邮箱已注册过 TikTok Shop 其他市场或 TikTok Business 账户，会创建失败 → 直接登录或换号注册。

### 第二步｜市场选择
- 仅支持东南亚五国（新加坡 / 马来西亚 / 泰国 / 越南 / 菲律宾）跨境入驻。
- **一套营业执照可开多个市场**，可单选也可多选。
- 入驻方式：
  - 企业商家 → 直接选"我没有邀请码"（平台审核发放）；
  - 无三方经验个体户 → 选"我有邀请码"（需先拿到邀请码）。**勾选前慎选**。
- 主体所在地：选"中国内地"交内地执照；选"中国香港"交注册证明书 + 商业登记证。

### 第三步｜资质认证（最关键、最易驳回）
1. **主体信息**：上传执照后系统 OCR 自动识别填充。
2. **法人资质**：支持 中国内地身份证 / 香港永居身份证 / 护照三类；**大陆户籍法人仅能用大陆身份证，不接受护照**。法人证件信息须与执照法人一致。
3. **法人实名认证**：大陆身份证需本人对摄像头完成活体认证（露脸、光线适宜、勿大幅转头）。
4. **设置主营类目 / 店铺名 / Logo / 仓库地址**：
   - **主营类目：一个商家账号只能选一个主营类目**，全站点只能发该类目商品（重要约束，选错影响很大）。定邀类目还需在执照经营范围内。
   - 店铺名/Logo 面向海外消费者、需遵广告法；**不得用中文 logo**，不得用违规/侵权/负面词，不要用带文化敏感的元素（骷髅头、自由女神像等）。

### 第四步｜开通 TikTok 机构号（企业号，可选）
- 平台凭执照帮商家开 TikTok 内容运营账号；昵称/头像继承店铺名与 logo。
- **店铺名须 < 30 字符**，否则创建失败。

### 第五步｜提交等待审核
- 提交后等审核；期间可进官方"商家学习中心 / Seller University"预学。

### 第六步｜缴纳保证金（最后一步，缴完才能经营）
- 收到"开店成功"邮件后，登录后台：**资产 > 保证金 > 充值**，按提示充值 → 收银台选支付方式 → 付款成功即可开店发品。
- 保证金性质：担保款，违规可被扣；店铺调整等情况可申请部分/全部退还（见《TikTok Shop 跨境店铺保证金政策》官方文档）。
- ⚠️ **本批资料未给出马来西亚的具体保证金金额**——金额/币种/退还规则以后台收银台与官方政策为准。

> **关键约束速记**：跨境店必须**缴完保证金才能发品经营**；**官方渠道开店不收任何开店费**，任何"第三方收费开店"都是违规行为。

## 三、无经验个体户"定邀入驻"通道

1. 填写《个体户无经验卖家定邀问卷》——**务必填入驻所用的个体户执照统一社会信用代码 + 注册邮箱**（填错会联系不到）。
2. 等平台审核；通过后 **3 个工作日内** 邮箱收到含邀请码的邮件。
3. 注册时选"我有邀请码"，填入即可完成无经验入驻（跳过三方经验填写）。

| 证件类型 | 上传 | 关键字段 / 校验 |

**证件图片要求**：< 10M，格式 JPG/PNG/BMP/PDF；彩色、清晰、**覆盖证件四角**、无遮挡水印。

### 营业执照字段

- 大陆企业：执照类型选"公司"；填公司名、执照号、注册地址、注册日期、有效期。
- 香港企业：上传注册证明书 + 商业登记证；填中英文公司名、唯一业务识别码、注册地址（与商登证一致）、注册日期、商登证届满日期。
- 个体户：执照类型选"个体工商户"；字段同大陆企业。
- **执照上传坑点**：必须**清晰彩色照片**；黑白复印件需**加盖公章**；**不要传电子营业执照截图**作凭证。OCR 易混字符（S/5、O/0、2/Z、1/I）**务必自查**，**填错无法修改**。

### 个体户三方平台经验证明（如走经验路径）

- 选经营规模最大的一个平台，提供：月均销售额 / 主营店铺链接 / 后台截图。
- 截图需含：① 店铺名；② 注册该店的**完整公司主体或法人**（须与 TikTok Shop 入驻主体/法人一致）；③ 店铺评分。
- 截图：**最多 3 张，< 5M，JPG/PNG/BMP**。链接店铺须与截图店铺为同一个，打开能看到店铺名 + 评分。
- 评分体系换算：1-5 分制 → ≥ 4.3；百分比制 → ≥ 90%；星级制 → ≥ 4 星。
- 非同主体但同法人：需在截图内附同法人证明（如同法人的企业执照）。

## 六、跨境模式基础认知（开店前必须想清楚的，来自《内贸出海手册》）

### 1. 物流模式：跨境店 vs 内贸的本质差异
- **内贸电商**：卖家仓库 → 买家地址（国内快递直发）。
- **TikTok Shop 东南亚跨境**：货物要**过关境**，两种发货方式——
  - **跨境小包**：卖家仓库 →（①商家自送 或 ②平台揽收）→ 平台转运仓 → 平台跨境物流+清关 → 目标市场 → 平台尾程派送 → 买家。**适合中小件、轻小件、客单较低**。
  - **海外仓**：卖家仓库 → 合作承运商物流+清关 → 海外仓 → 平台尾程派送 → 买家。需先备货到海外仓；**时效快、适合大件/高单价/高频次**，但要承担备货与海外仓成本。
- **跨境小包揽收规则**：平台首公里服务覆盖范围内会**免费上门揽收**；范围外商家自送/自寄到平台五大转运仓集货点，**注意：自送不可通过快递寄送**（需自行送至集货点）。

### 2. 跨境结算与定价
- **与国内最大差异**：跨境回款依托**支持跨境结算的第三方收款服务商**。
- **结算回款周期：预计账期 7-9 天**（可信度中，以后台为准）。
- 结算链路：买家付款（信用卡/银行转账/电子钱包）→ TikTok Shop 平台托管 → 交易成功后平台经第三方支付伙伴将货款打至卖家账号。
- 平台合作的 **五大跨境收款服务商**：万里汇(WorldFirst) / PingPong / 连连国际 / 空中云汇(Airwallex) / Payoneer。（资料附各家二维码引流，**选服务商时核对最新合作清单与费率**。）
- **定价要把跨境特有成本算进去**：相对内贸模式，跨境额外涉及 **物流成本、汇率、市场冷启动成本** 等；佣金/交易手续费、保证金、达人带货佣金、广告费等按平台规则计。（资料有"平台成本及定价"对照表，但备注栏为图片小字，**具体费率以官方费率表为准**，本笔记不转录其费率数字。）

## 七、马来西亚市场速览（用于选品/定位参考，来自《内贸出海手册》）

> 数据源标注 Euromonitor / Emarketer / UN DESA 2023 / 马来西亚统计局，**仅供参考、时效偏旧**。

- 近三年电商市场增长率：**约 10%**
- 语言：**马来语、英语、中文**（多语言市场 → 商品/内容需考虑多语言，平台发品支持英文标题描述 + 一键翻译为目标市场小语种）
- 人口特征：年龄中位数 **约 29.2 岁**（偏年轻）；**70% 马来裔、22.7% 华裔**
- 消费趋势：**高度多元化**

**官方选品/市场调研入口**（开店后选品用）：
- 商家大学 - 选品指南
- 商家后台 - 商品机会中心（看热销/需求/爆款趋势）
- 商家后台 - 数据罗盘 - 商品卡片 - 热门（看品类商品排行榜）
- 对应市场 App 搜索框搜关键词看头部产品与价格

**品类经营建议**（资料给的方向，非马来专属）：大快消（美妆护肤、桌면冲饮）、Fashion（手机壳、男装/女装、平价饰品）、家居百货（日用百货、电емат厨具、玩具潮玩）等——倾向**单价低、易标准化、复购/即兴消费**的品类。

## 八、不能/限制销售的商品（开店类目选择前先看）

资料区分三类（具体清单以官方《禁限售规则》为准）：
- **禁售商品**：当地法律或平台禁止，不允许在 TikTok Shop 上销售。
- **限售商品**：需通过特定申请/资质，符合 TikTok Shop 审核要求后才能上架。
- **暂不支持商品**：目前 TikTok Shop 尚未开放该类目上架。

> ⚠️ 资料未给出马来西亚具体禁限售清单，**选主营类目前务必查官方禁限售规则与目标市场法规**。

## 九、附带工具与跨板块指引（非入驻，速记备查）

这些来自《内贸出海手册》后半段与《跨境直通运营指南》，**属于上架运营/选品/达人板块**，此处仅留指针，详细内容应归入对应板块笔记：

- **一次发多国**：商家中心 - 全球商品，上传后"发布到店铺"即可多市场分发；英文标题描述发布到各市场时**自动翻译目标语种**。支持绑定三方 ERP 批量发布（后台"应用和服务市场"授权）。
- **IM 沟通**：店铺自带 IM，与买家用英文沟通时可实时翻译中↔小语种。
- **短视频工具**：官方免费 TikTok Symphony，跨境视频翻译+配音（Translate & dub）、自动加多语种字幕、根据链接自动生成 AI 型视频。
