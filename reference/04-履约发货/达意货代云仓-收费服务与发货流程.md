## 0. 一句话
达意货代 = 专做 **TikTok 小店贴单代发**的东南亚跨境云仓；国内 **东莞仓 / 义乌仓**（另有海外仓），**与货叮咚 ERP 无缝对接**；打包费 **2 元/单**、不限 SKU；营业 10:00–22:00、全年不打烊、每天多批次**当天送官方仓**、24h 客服。**"货"（打包/发货/退货仓）交给它；"钱和主体"（执照/法人/收款）仍是你自己的。**

## 1. 收费标准
| 项 | 标准 |
|---|---|
| **打包费** | TikTok 小店自送仓 **2 元/单**（量大可谈，不限 SKU 数量） |
| **超规附加费** | 包裹**三边之和 ≥ 220CM** 或 **单件重量 ≥ 10KG** 加收 **1 元**（TikTok 订单极少出现，服装类可忽略） |
| **增值耗材** | 特定尺寸泡沫袋、气泡柱、蛇皮袋等**额外付费**，价目在货代系统已列 |
| **免费快递袋** | 免费提供（**气泡膜、定制纸箱除外**） |
| **仓储** | **囤货免仓储费**（客户可免费备货在仓库） |

## 2. 服务内容
主流程：**签收 → 过机扫码分类 → 拆包验货 → 打包（称重、全程录像）→ 贴单 → 出库过检 → 当天多批次送官方仓**。
- **囤货免仓储费**；**拆包验货**（查颜色、款式、尺码、数量、有无破损）、**拿掉好评卡 / 代发礼物**。
- **组合合并订单处理**、**更换高品质包装 / 加固重打**。
- **贴错单、仓库丢件一律包赔**。
- **一天 3–4 次送仓、当天送仓**，保证时效；**每天多批次送官方仓**。
- **打包过程全程视频监控（萤石）**、打包完成**称重保存信息**；**出库过检机核对、出库信息即时上传系统**，包裹可追踪。
- 客户**免费备货**：入库前检查数量/细节/质量，确认无误**贴专属条码上架**。
- **专属客服对接**、到货未录单提醒、问题件订单提醒；**24 小时客服在线**。
- **新手 1 对 1 培训**发货流程与系统使用。
- 产能：流水线打包、约 20 个打包员，日处理超万单，确保订单当天出库。

## 3. 赔付承诺
- **时效赔付**：若因**仓库原因**导致延时送仓（承诺 **24 小时内**送仓），仓库赔付。
- **损坏 / 丢失赔付**：仓库**签收后**发生的丢失或人为损坏，按 **「产品采购价 + 运费」** 赔付。
> ⚠️ 赔付为货代自述承诺，**签约前用合同/书面确认口径与举证方式**。

## 6. 接入 + 发货流程（在货叮咚/达意系统里）
> 来源：微信「发货流程学习」（OCR 有噪声，步骤名以系统实际界面为准）。ERP 操作细节见 [货代备货订单与发货设置.md](货代备货订单与发货设置.md)。

**发货机制（出单后）**：
1. **出单 → 采购**：在采购页选「达意货代」；**推荐用货叮咚 ERP 采购**（自动匹配货源采集单/发货单）。
2. **采购拣货**：**务必只发货到达意仓库**对应订单（发货时选达意货代仓）。
3. **系统拣货**：把订单同步到达意 → 系统拣货 → **打印国际物流面单**再发货。
4. **货品异常处理**：破损/少发/发错等，在达意系统对应订单及时处理、补发并回填单号。
5. **资源核验**：达意系统 → **回单管理** → 按物流单号核对，确认仓库已收到并贴单。

**接入步骤（一次性）**：
1. **注册达意货代系统**（`dyhd.huoyuanjiawms.com`）并绑定登录号 + TikTok 店铺。
2. **在达意绑定台设置货代仓库地址**（东莞/义乌，收件人=你的货代 DI 号）。
3. **TikTok 店铺授权**：达意系统「店铺授权」→ 选 TikTok → 建站选**亚洲商家 / 中国大陆·香港** → 账号密码登录 → 选授权店铺 + 填仓库地址 → 同意协议授权 → 店铺状态显示「**生效中**」即成功。
4. **充值**：账户充值余额（金额期内可用）；**没显示金额就联系达意客服充值**。
5. **订单管理**：按状态流转（待采购 → 已采购 → 待出单 → 已出单 → 待发货）跟单。

## 7. 与既有规则衔接
- **达意仓地址 = 店铺后台「发货地址」**（出单后货代按面单把货送进 TK 官方仓）——设置见 [货代备货订单与发货设置.md](货代备货订单与发货设置.md) §9。
- **国内退货地址建议填达意仓**；**本地退货方式选「退到 TK 官方认证仓」**（无货源商家）——同上 §9。
- **发货 SLA（72h 入仓 / 96h 旧口径）以官方后台学习中心当期为准**；达意的"当天送仓 / 24h"是**货代内部时效承诺**，**不等于平台 SLA 计时**，别混淆。见 [发货指南与运费.md](发货指南与运费.md)。
- 货叮咚 ERP 的授权/采购派单/库存/拒收退件操作 → [货代备货订单与发货设置.md](货代备货订单与发货设置.md)。
