## 1. 物流产品与基础参数

平台提供 3 类产品，跨境直发主要用「全球标准」。**马来西亚跨境店用的是「全球标准运输服务」。**

### 1.1 全球标准运输服务（Global Standard Shipping）— 马来西亚走这个
- **开通国家**：泰国、马来西亚、菲律宾、新加坡、越南
- **参考时效**（中，易变）：泰国 4–15 日 / **马来西亚 5–15 日** / 菲律宾 5–15 日 / 新加坡 5–8 日（均自然日）
- **货件尺寸&重量上限**（中）：

  | 国家 | 长 cm | 宽 cm | 高 cm | 重量上限 |
  |---|---|---|---|---|
  | PH 菲律宾 | 150 | 120 | 100 | 50 kg |
  | **MY 马来西亚** | **140** | **70** | **70** | **30 kg** |
  | TH 泰国 | 220 | 220 | 220 | 30 kg |
  | VN 越南 | 100 | 100 | 100 | 30 kg |
  | SG 新加坡 | 140 | 80 | 80 | 30 kg |

- **计费重量**：10g 为单位，按**包裹实际重量**计
- **走货属性**：普货、内置电池（单个包裹 ≤ 2 节电池，电池总容量 ≤ 100Wh）。
  - **禁收**：纯电（纯电池）、配套电池、移动电源（充电宝等）、枪支弹药等违禁品。
  - 泰国例外：移动电源（充电宝）可特殊申请（不适用马来西亚）。
- **申报金额上限**（中）：泰国 < 24000 THB；越南 < 1,000,000 VND；菲律宾 COD<50000 / 非COD<100000 PHP；新加坡 < 400 SGD；**马来西亚 COD < 1000 MYR、非 COD < 3750 MYR**。

### 1.2 全球经济运输服务（Global Economy Shipping）— 仅越南
- 只开通**越南**；时效 6–11 自然日；尺寸最小 10×15cm，最大 L+W+H<300cm 且单边≤100cm、≤30kg；申报 < 1,000,000 VND。**与马来西亚无关，仅作背景。**

### 1.3 本地标准运输服务（Local Standard，海外仓）— 非跨境直发
- 开通：泰国、马来西亚、菲律宾、越南、新加坡。需先做仓配认证（原文链接 *SEA TikTok Certified Warehouse One pager*）。
- 时效更快（如马来 3–7 自然日）；按**抛重计费**。
- **这是海外仓模式，不是「从中国直发」**。本店当前不用，仅备查。

## 2. 履约时效红线（最重要，务必背下来）

> 这是全篇最该执行的硬规则，超时会影响包裹时效/店铺履约指标。
>
> ⚠️ **口径冲突（以官方后台为准）**：本文 96h 系 2024 旧价卡资料口径；旗舰《马来西亚-跨境店-开店向导》§5 已联网核实**现行入仓考核为出单后 72 小时**（120h 为旧规总时效、96h 属旧/冲突口径）。**下单前务必登录卖家中心「履约考核」页确认当期确切小时数**——72h/96h/48h 可能对应不同计量节点（是否揽收、打包 vs 送仓），别只背一个孤立数字。详见开店向导 §5。

- **付款后 48 小时内**：完成包裹打包和发运。
- **自寄包裹**：必须在付款后 **96 小时内**送到指定仓库。
- **揽收包裹**：服务商需在 **48H 内**完成揽收交接。
- 退货/异常处理另有时效：**48H 内确认收件、96H 内重新发货**（见 §6.2）。

## 3. 发货 SOP（待发货 → 已发货）

**触发**：买家付款后订单进入「待发货」。后台路径：订单管理 → 待发货。

1. **2.1 选包裹**：进「待发货」页，勾选要发的包裹，确认可履约后点「安排发货」。
2. **2.2 选物流**：在弹窗里选发货物流服务（如「全球标准」），确认设置为该物流。
3. **2.3 打印包裹**：照「待发货」操作完成后打印面单（详见官方《商家发货预报操作手册》）。
4. **2.4 打包**：按平台《英国/东南亚跨境包裹包装规范》打包。
   - **包装避坑（影响责任归属）**：
     - 包裹必须打包好、符合平台包装规范，否则现场破损投诉责任归商家。
     - 商品/数量须与订单一致，**少发/漏发由商家担责**。
   - **特别提醒**：东南亚地区快递费用按**包裹实际重量**计收。
5. **打印并粘贴标签**：切到「待发货」选包裹，打印面单并贴到对应包裹。

### 3.1 面单（关键参数）
- 格式 **10×15 cm**，需**热敏打印机 + 热敏纸**。
- 两种面单：① 商家物流面单（含 Order ID、Tracking No、Ship to 地址，标有 ECONOMY/Standard、DROP OFF / Drop-off 字样）；② 物流面单需对应到包裹。
- **同一商品多件 / 一单多件**时，按订单逐件对应贴单。

## 4. 交货方式：自寄 vs 集货点/揽收（含东莞自建仓地址）

跨境直发把包裹送进平台体系有几种方式：

### 4.1 自寄到分拨中心（商家自付头程运费）
- 商家自行把包裹寄/送到平台指定**分拨中心（仓库）**，**96H 内**送达。
- **TikTok 东莞自建分拨中心**（高可信，地址类）：
  - 地址：广东省东莞市洪梅镇望沙路 115 号易商物流园 4 号库 1 楼（进园区左手边 02/12 号收货口）
  - 电话：(0769) 8685 5106
- 资料里另列了**万色**多个分拨中心（上海宝山区共悦路 419 号 / 义乌富通大道与祥瑞路交叉口圆通妈妈商贸二号仓库二楼 C 单元 / 泉州晋江市磁灶镇张林东环路 999 号豪川运派物流园 C 区 / 东莞黄江镇田心星路 39 号巨卓产业园，统一电话 4008206207）及**顺丰（金东）分拨中心**（浙江金华金东区兴盛街 855 号丰泰产业园-金义 B103，09:00-12:00 & 13:30-18:00，19057193381/4007111997）、**菜鸟香港分拨中心**（香港离岛区大屿山香港国际机场骏裕路 8 号菜鸟智慧港 4 楼 B-410 货台，+852 6117 6612）。
  > ⚠️ 这些第三方仓地址是培训资料快照，**可能变更**，自寄前务必在后台「跨境直配-集货点信息」核对当前可用地址与归属服务商。

### 4.2 集货点 / 免费揽收（服务商上门）
- **集货点（自送，非快递）**：适用于现有免费揽收范围无法覆盖、或单量暂不满足免费揽收的卖家；商家自送到集货点（不可通过快递寄达集货点），由集货点统一送货至平台分拨仓，发货快、时效快。
  - **使用条件**：① 无包裹数量门槛；② 自送到集货点（非快递）。
  - **收货标准**：① 包裹需整袋打包，不能散件（小包）交接；② 包裹包装完好、干燥无水渍无污渍。
  - 收货时现场称重计费；集货点收费=揽收商家包裹后到运输至分拨仓期间的货管责任由集货点负责。
- **上门揽收（免费揽收服务）**：服务商上门揽收，按实重计费；上门取消则状态变「未申请上门取件」，可随时重新申请。

### 4.3 揽收/交接凭证（留痕，处理纠纷的证据）
- 服务商上门揽收时清点现场包裹数量、提供交接清单，**商家签字确认**（《承运货物交接凭证》）。
- 揽收后包裹揽收成功，一般 6 天内入仓；商家在揽收完成日 / 发预报到货后 30 日内做查询。
- **若用集货服务揽收包裹到分拨仓**，包裹未更新入仓信息时，需用以下凭证申诉/查询：
  1. 商家需明确入仓包裹对应订单号+物流商单号，及入库订单对应货物的外包装图片+内件实物图片；
  2. 商家地址及揽收服务商接收单签字照片（含揽收大包裹数量、重量、小包裹件数、各揽收大包对应小包裹清单）；
  3. 当天交接给揽收服务商同一揽收大包内对应所有小包清单（订单号+物流商单号、小包数量）。
- **避坑**：若提供凭证后商家存在破损/未妥善对装场景，揽收服务商有权拒收，**商家强制交接则以分拨仓实收为准**。

## 5. 已送达 / 取消 / 拒收 / 消息请求

- **已送达**：包裹入集货仓库后，会根据物流服务商运输能力送到东南亚消费者手中；完成清关、投诉、签收后系统自动移到「已送货」。
- **取消流程**：
  - 用户下单后想取消、或商品发不出 → 用户发起取消，系统自动从待发货扣减并完成或拒绝；
  - 商家库存=0 时系统自动 cancel 并下架商品。
- **7.1 同意请求**：在「订单管理」待发货页或取消订单页查看处理，点「同意请求」→ 退款状态变「已处理」，用户即收到退款。
- **7.2 拒收请求**：在取消申请弹窗填拒收原因后提交。

## 6. 退货退款（两类退货地址）

> 跨境商家有**两个退货地址**：① 中国内地退货地址；② 东南亚当地退货地址。用于不同场景。

### 6.1 中国大陆地区分拨仓退货（适用 POP 直发所有国家）
- 适用：异常件（拒收/取消/未送达/超规/不合格/超重/退回机/仓格不合格/BIC 不合格等）退到中国内地分拨仓。
- 流程要点：默认按原途径退回运费 → 对因电话/号码问题（空号/错号/关机/停机/无法接通）联系不上买家无法退货的包裹，仓库定期进行销毁 → 对可退货地址错误致退货失败的，仓库会再次按异常退货找 AM 联系仓库申诉后恢复退货 → 无法退货则按异常退货流程切断拒收，拒收卖家仓库；服务商**站内信通知前会在商家大学/学习中心公示，公示发布 7 个自然日内可自行联系仓库认领并执行销毁**。

### 6.2 东南亚退货地址
- 适用：当包裹妥投失败/拒收时。
  - 商家在后台维护了东南亚当地退货地址 → 妥投失败/拒收的包裹会优先退到该东南亚当地退货地址；
  - 商家没维护 → 包裹会退回至物流商仓库，后续会安排销毁。

### 6.3 注意事项（时效）
- 只有订单**收件前**或退件设置完成前完成订单时间，新的退货地址才会生效；新订单在新地址设置完成前创建会按原地址退回。
- **48H 内确认收件、96H 内重新发货**（自发货订单）：最迟 96H 内将包裹寄出，超时系统自动移「已发货」。
- 退货逾期未收货 → 系统自动确认收货并货款返还买家。
- **上门取件**（door pickup）退货：后台「所有提收仓库」里维护第三方仓库地址/上门取件地址，重新维护后发起上门取件申请；取消则状态变「未申请上门取件」，可随时重新申请。

## 7. 违禁品 / 禁运品清单（重要避坑，选品就要避开）

> 来源 PDF 第四节，共 18 类。**这是物流专线层面的禁运边界，与各国海关合规另算；可信度中等、边界可能调整，拿不准的品类发货前先咨询 AM/服务商。** 与本店选品强相关，列全要点：

1. **枪支、弹药**：含仿制品、主要零部件（手枪/步枪/冲锋枪/防暴枪/气枪/猎枪/运动枪/麻醉注射枪/钢珠枪/催泪枪等）；及弹药、催泪弹、照明弹、燃烧弹、烟幕弹等。
2. **管制器具、攻击性武器**：弹簧刀/三棱刮刀/带自动锁紧弹簧刀/匕首/弹簧棍/伸缩棍/电击器/钢叉/双截棍等。
3. **易燃易爆物品**：爆破器材（炸药/雷管/导火索/导爆索）、烟花爆竹、易燃固体（火柴/赛璐珞）、易燃液体（汽油/煤油/油漆/花露水/香水/指甲油/啫喱膏/驱蚊水/杀虫剂/酒精）、自燃物品（黄磷）、遇湿易燃（金属钠/钾/碳化钙）、氧化剂、腐蚀性物品（硫酸/硝酸/盐酸/蓄电池/氢氧化钠/氢氧化钾）。
4. **压缩和液化气体及容器**：氢/甲烷/丁烷/天然气/液化石油气/乙烯/丙烯/乙炔；一氧化碳/氯气；喷雾剂/摩丝/压缩气罐等。
5. **植物及植物制品**：花/水果/蔬菜/种子/植物标本/松果等。
6. **动物及动物制品**：贝壳/螃蟹/动物标本/毛皮等。
7. **任何药品**：含麻醉药/药膏/足贴等。
8. **任何食品**：肉类/饮料/咖啡/水果/零食/茶叶等。
9. **液体、粉末及类似制品**：化妆品类受承认（除表内列出品类外不支持运输，**按毛重和包装要求**）；只接受化妆品类液体/粉末/膏状用品及绘画颜料/染料粉/口腔清洁剂/墨水等，其他任何类型的液体/粉末均不收。
   - **可走空运标准（重要数字，中）**：包裹单 pcs 毫升数 ≤ 100ml、整件毫升数 ≤ 500ml（>100ml / >500ml 走海陆运）。所有液体/膏状类及易碎类产品需用纸盒包装填充、避免转运破损。含酒精的液体不收。
10. **纯电池及移动电源**：只可发内置电池（单包≤2节、电池总容量≤100Wh）；**不接受纯电（纯电池）、配套电池、任何形式的移动电源**。
11. **非法仿造物品**：伪造/变造货币/证件/公章等。
12. **侵犯知识产权和假冒伪劣物品**：假冒专利权/商标权/著作权/音像制品，及侵权包括外观侵权品/无标侵权。
13. **毒性物质**：砷/砒霜/汞化物/铊化物/氰化物/磷酸/苯酚/汞/剧毒农药等。
14. **腐蚀性物质**：硫酸/硝酸/盐酸/蓄电池/氢氧化钠/氢氧化钾等。
15. **毒品及吸毒工具**：麻醉药品/精神药品（如鸦片、可卡因、大麻、甲基苯丙胺/冰毒、氯胺酮/K粉、安钠咖等）；易制毒化学品（如胡椒醛/黄樟素/麻黄碱/苯乙酸/丙酮等）；吸毒工具（如冰壶等）。
16. **非法出版物、印刷品及音像制品**：含有反动/淫秽内容/危害国家统一/破坏社会稳定/宣扬邪教/宗教极端思想/淫秽等内容的图书/刊物/图片/照片/音像制品等。
17. **间谍专用器材**：暗藏式窃听/窃照器材/突发式收发报机/一次性密码本/密写工具等。
18. **医疗/防疫物资**：目前所有液体/流质产品均不收；所有防护服/呼吸机/监护仪/医用消毒巾消毒剂等防疫物资专线禁寄；不收额温枪/医用手套/鞋套/护目镜/电子体温计等。

## 8. 运费/物流成本（XLSX 价卡详解）

> 来源 `…东南亚跨境物流运费价格表.xlsx`。**生效日期 2024-04-04，已较旧；报价/核算前必须用后台最新价卡复核。** 计费口径：**实重、10g 为单位**。

### 8.1 概念区分（很容易混）
价卡区分两个成本：
- **跨境物流成本**（卖家承担，「卖家支付运费」的主体）：头程+干线。
- **当地派送物流成本**（买家下单时支付的运费）：尾程派送。
- 公式（价卡原文）：**卖家支付运费 = 跨境物流成本 + 买家支付运费**；总物流成本 = 跨境物流成本 + 当地派送物流成本。

### 8.2 马来西亚价卡（重点，币种 MYR）
- **跨境物流成本（卖家承担，全境同价，线性）**：**0.15 MYR / 10g**，无起步价台阶（10g=0.15，100g=1.5，500g=7.5，1kg=15，2kg=30，5kg=75…线性到表内上限 30000g）。
- **总物流成本（含当地派送，分东西马）**（中，旧）：

  | 计费重量 | 西马（Standard，总成本 MYR） | 东马（Standard，总成本 MYR） |
  |---|---|---|
  | 10 g | 5.05 | 10.15 |
  | 100 g | 6.40 | 11.50 |
  | 500 g | 12.40 | 17.50 |
  | 1000 g | 19.90 | 25.00 |
  | 2000 g | 34.90 | 40.00 |
  | 5000 g | 79.90 | 85.00 |

  - 西马总成本 ≈ 4.9 起步 + 0.15/10g（即「跨境 0.15/10g + 西马当地派送约 4.9 起」）；东马在西马基础上当地派送更贵（基础约 +5 MYR）。
  - **区域划分**：东马 = Labuan / Sabah / Sarawak；西马 = Johor / Kedah / Kelantan / Kuala Lumpur / Melaka / Negeri Sembilan / Pahang / Penang / Perak / Perlis / Putrajaya / Selangor / Terengganu。

### 8.3 成本快速估算工具（XLSX 内置）
- sheet「跨境物流成本计算工具」：填 SKU ID + 重量（克），右侧自动出各市场成本。
- 用法 3 句话：① 填商品信息（SKU、重量）；② 对应市场跨境物流成本自动展示；③ **结果仅供参考**，实际订单可能一单多件 / 包裹重量不符而有差异。
- 工具示例（与马来 0.15/10g 一致）：300g→4.5 MYR；500g→7.5 MYR；1121g→16.95 MYR。

## 9. 与既有规则的衔接 / 待核实

- 本文费率/时效/申报上限均来自 **2024 年价卡与培训资料**，与店铺其他文档若有冲突，**以卖家后台最新价卡和官方学习中心为准**。
