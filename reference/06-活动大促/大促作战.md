## 一、大促到底是什么、为什么必须打（认知）

- **本质**：TikTok Shop 在一年中消费高峰期（年中、黑五、Mega Sales、双十一/双十二等）办的"超级大卖场"。区别于国内大促"商家单方面让利"，**海外平台是平台+商家"双向奔赴"**：平台用补贴、流量倾斜、激励任务（发短视频、达 GMV、投广告、开直播）主动赋能，商家折扣压力被平台分摊一部分。
- **流量量级**【通用/SEA】：
  - 大促期**流量可达平日 3–5 倍**，搜索热度增长 **200%–500%**。
  - 转化率较平日**提升 50%+**（限时折扣 + 平台背书增强信任）。
  - *以上为机构口径，量级参考，不保证*。
- **大促流量三大引擎**【通用/SEA】：
  1. **平台全域资源倾斜**：首页活动入口（如 9.9 专区）、补贴标签、合资券（资料称平台分担 50%–65% 折扣成本——**比例须以官方为准**）。
  2. **内容场 + 货架场双端联动**：短视频挑战赛带官方话题可获流量扶持；头部直播间长时段直播 + LSA（直播购物广告）引流。
  3. **跨文化节日热潮**【MY 适用】：**斋月期间马来西亚用户消费预算 +56%，家居/礼盒类 GMV ×1.4**；春节红色/红包类商品在马来转化率 +70%。→ MY 跨境店要重点卡 **斋月（Ramadan / Hari Raya）** 和 **农历新年** 两个本地节点。

**三阶段动作要点**：

- **预热期（约大促前 1 周）= 蓄水**
  - 测出潜力爆品（选已报名商城货架的头部品 + 店铺爆品）。
  - 店铺主题装修、提前引导领券 / 加购，做流量沉淀。
- **爆发期（大促当周）= 引爆**
  - 集中释放优惠：限时秒杀、满减。
  - 先靠自然流稳定出单，**再精准邀约达人**做二次爆发。
  - **死守发货时效**：明确爆品发货优先级，快速出库，别因履约率被扣分影响权重和转化。
- **沉淀期（大促后 1 周）= 收尾 + 蓄下一波**
  - 把大促期热搜 / 飙升关键词加进商品标题，吃长尾搜索流量。
  - 跟进已售订单、收集好评、沉淀粉丝，为下一个节点蓄力（衔接 PayDay 等）。

## 四、大促筹备 SOP（核心可执行清单）

> 综合 9.9、10.10（双十）两份筹备指南，去重后整理。**这是最该照着做的部分。**

### 时间倒推总表【MY 跨境店重点】

| 距大促 | 必须完成的动作 |
|---|---|
| **T-60 ~ T-45** | 若走**本土备货**：货物提前发出（避免大促期清关慢、到仓慢误了上架）。MY 跨境店若纯跨境直发可放宽，但仍要锁定供应链备货。 |
| **T-60 ~ T-30** | 选定 **5–10 款主打品 + 潜在爆品**；优化价格做出竞争力（**至少提前 30 天上架**以拿官方流量扶持/补贴、养链接权重）。 |
| **T-30** | 启动 **测款三步法**（见第五节）；开始建联达人、发预热视频。 |
| **T-30 ~ T-14** | 报名平台活动（SFP 免运费、商城闪购、当月大促）；铺日常促销（新品折扣、限时秒杀、满减、优惠券、多件优惠）。 |
| **预热周** | 店铺主题装修上线、引导领券/加购；预热推广预算提到平日 **1.2–1.5 倍**。 |
| **爆发周** | 优惠力度拉满；推广预算可**翻倍**（视 ROI 增减）；客服催付；死守发货时效。 |
| **沉淀周** | 关键词回收进标题；售后处理；粉丝/收藏未购沉淀；标签修复（见下）。 |

### 七阶段法（双十筹备指南，可作通用模板）

1. **精准选品**：在 `店铺 → 商品 → 商品机会 → 热门商品` 找热门品，拆解其元素（视觉冲击力、传播性）；结合季节/本地需求与平台热门话题标签、搜索关键词捕捉款式。
2. **仓储与客服准备**：备货量建议为**日常的 3–5 倍**（按历史 + 预估销量，**仅供参考，跨境店按自身履约能力定**）；客服培训三件套——**产品专家**（卖点/尺寸/材质）、**活动专家**（满减/折扣/平台券规则）、**引导话术**（点赞/关注/进小黄车，把公域转店铺粉）。
3. **预热造势**：热销品放首页最前端；高质量图 + 促销海报；营销工具搭配（见下"营销玩法"）。
4. **达人合作矩阵**：**头部达人造势（如 30 万粉以上，开场直播引流）+ 腰部达人铺量（1–10 万粉，短视频种草 + 日常直播）**，选与品风格定位相符的**本土达人**。
5. **广告投放 + 流量扶持**：精准定向（地区/年龄/性别/兴趣）；积极报名官方大促争取首页 Banner、搜索推荐位。
6. **数据监控与即时调整**（大促当天实时盯，见第七节"避坑/应急"）。
7. **售后与沉淀**：及时处理退换/咨询避免 DSR 暴跌（**会影响后续 11.11/12.12 的流量资格**）；用搜索广告投核心卖点关键词把被大促泛流量打乱的**人群标签"拉"回来**；对新粉/收藏未购发复购券沉淀。

## 五、选品与测款（找爆品的方法论）

### 测款三步法【通用/SEA，强烈推荐照做】

> 阈值是机构经验值，**作为相对参考**，不同类目差异大。

1. **数据侦察（提前 30 天）**：用工具（资料点名 **Fastmoss**，亦可 FastMoss/超店有数等）查东南亚热搜品，对照平台**商品机会榜**，锁定潜力类目（例："印尼防晒霜搜索量月增 230%"）。
2. **快速试错（上架 7–15 天）**：盯三个指标——
   - **曝光量 > 5000**（达不到 = 主图/价格不吸引）
   - **点击率 > 5%**（达不到 = 标题关键词不准）
   - **加购率 > 3%**（达不到 = 详情页卖点弱）
3. **精准优化（大促前 10 天）**：对症下药——
   - 点击高、转化低 → 加赠小样促销；
   - 曝光高、点击低 → 换主图 / 封面；
   - 加购多、付款少 → 设库存预警弹窗催单。

### 爆品组套玩法（拉高客单价 / GMV）

- **同品类套组**：正装+体验装、多件装、爆款+新款、正装+补充装。
- **互补套组**：上下装搭配、3C+配件、美妆+工具、洗衣液+增白液。
- **功效套组**：同功效打包，或日霜+晚霜、控油套装等。

## 七、店铺装修、营销玩法与避坑

### 店铺装修与本土化【MY 适用】

- 换大促主题装修，主推品置于首页最前端；首屏放"爆款排行榜"做导购。
- **视觉本土化**：首页用本地实景图，标语带俚语（马来语示例 **"Harga Terbaik!"** = 最优价）。
- 首屏底部打 **"免费配送 + COD 货到付款"** 标识（COD 在东南亚是强信任要素）。
- 详情页标题嵌大促关键词，图/视频展示细节与好评；价格对比海报降低决策成本。

### 营销工具组合

- 平台活动：SFP 免运费、商城闪购、当月大促、闪购/秒杀专区。
- 店铺促销：阶梯满减（泰铢示例：满 200 减 50 / 满 500 减 200——**MY 请换成 RM 阶梯并测试**）、前 N 单 / 前几小时限时折扣、无门槛优惠券、多件优惠、组合套餐。
- 内容：短视频**前 3 秒直击痛点**（如"夏季脱妆急救"）；直播做场景化主题。

### 大促当天应急（实时盯盘）

| 问题 | 原因 | 对策 |
|---|---|---|
| **流量不及预期** | 内容热度不够，没进更高流量池；或转化差被系统减推 | 检查短视频/直播内容吸引力；果断加"随心推"预算定向潜在兴趣人群，撬自然流把排名"助推"上去 |
| **有流量无转化** | 价格力不足 / 卖点不清 / 客服慢 | 比价竞品，必要时叠一张小额限时券造紧迫感；直播强调"仅限今天/最后 XX 件"；客服高效催付（咨询未下单、加购未付款） |

### 关键避坑点

- **发货/履约是红线**：大促包裹暴增，跨境直发更要盯时效；履约率扣分 → 影响店铺权重 → 影响大促转化。爆品要排发货优先级。
- **DSR 评分保护**：大促售后激增，差评拖垮 DSR 会**影响后续 11.11/12.12 的活动/流量资格**，必须及时耐心处理退换货。
- **人群标签会被打乱**：大促泛流量稀释精准标签，结束后用搜索广告投核心卖点关键词把标签"拉"回来。
- **别盲目堆备货**：跨境店清关/到仓有不确定性，备货量"日常 3–5 倍"是机构口径，按**自身履约能力 + 测款数据**定，避免压货。
