## 0. 一句话方法论（所有案例的共同主线）

**选对爆品 → 商城优化攒原始数据（前1-2个月）→ 达人短视频起量 → 报活动+优惠券放大 → 冲明星店铺/TopChoice 拿流量加权。**
官方口径：店铺从 $0/天 → $500/天 **至少需要 3 个月**（来源：女装官方文档，可信度高）。

> ⚠️ 学员案例都跑在**泰国**：靠「纯自然流 + 商城 + 达人」起量，几乎没提付费投流。马来西亚是否同样吃自然流红利，需自行验证（红利期可能已过）。

## 1. 选品 SOP（女装跨境，可复制）

### 1.1 选品逻辑（通用，建议优先用）
1. **先定人群与风格再选品**，不要从众。赵学员案例反向操作：避开年轻人扎堆的 Y2K/韩系，定位 **35-45 岁女性**（白领/主妇，重质感、设计感、不浮夸），用「缎面衬衫 + 棉麻短袖」打差异化 → 100 天做到 5802 单、23.27 万人民币。
2. **参考多平台爆品**：1688、虾皮(Shopee)、小红书、拼多多爆款详情页，反推卖点与款式。
3. **数据工具找飙升词/飙升品**：数据罗盘关键词榜单 + 第三方 ERP（见工具清单）。
4. **结合当地气候 + 宗教文化**（见下方各国差异）。

### 1.2 新店选品组合：「532 法则」（首月测试期）
| 占比 | 类型 | 作用 | 示例 |
|---|---|---|---|
| 50% | 基础款 | 引流 + 测数据 | 白T、黑打底裤等长青款 |
| 30% | 微创新款 | 在爆款上改良 | 汉元素盘扣衬衫（传统+现代剪裁） |
| 20% | 趋势款 | 押注平台新兴风格、冲爆款 | 越南「棋盘格针织衫」 |

> 来源：大卖孵化策略（可信度中，作思路用）。另一处建议 **SKU 精简到 15-20 个**，分「低价引流款 / 高利润款 / 组合促销款」三类（来源：店铺流量提升策略）。

### 1.3 上新节奏与营销节点（官方口径，可信度高）
- **每周上新 5-10 款**，提高商品丰富度、刺激复购。
- 抓节日营销：万圣节、圣诞、双11/双12、本地节日（**马来西亚重点：斋月 Ramadan / 开斋节、各类大促 4.4 / 5.5 等**）。

### 1.4 各国女装风格 / 类目（官方内部数据，可信度高）
- **跨境市场体量排序**：TH > **MY** > PH > VN > SG。
- **五国平均件单价约 $5（区间 $5-$10）**；新加坡可达均价 3 倍。
- **目标人群**：女性，18-24、25-35 岁为主。
- **各国风格**：
  - TH：Y2K / 韩系甜酷
  - PH：OL 通勤 / 欧美辣妹
  - VN：Y2K / 韩系甜酷
  - **MY（马来西亚，重点）：保守 / 与穆斯林人群常服需求重合** → 长袖宽松款、罩袍、头巾、高领连衣裙等
  - SG：OL 通勤 / 现代简约 / 高质量
- **全年 GMV 主力类目**：衬衫、连衣裙、T恤/背心、裤装、内衣裤。
- **春夏高增速子类目**：连衣裙、衬衫、T恤、背心、裤装、短裙、内衣裤。
- **10月后秋冬增速明显（尤其 VN）**：毛衣、卫衣、针织衫、夹克。
- **东南亚热带气候通用选品**：短袖T、吊带裙、防晒衣、速干裤、轻薄开衫、短裤短裙、凉感面料（冰丝、棉麻）。

> ⚠️ 对马来西亚跨境店：因穆斯林文化，**优先长袖、宽松、遮蔽性强、符合 modest fashion 的款式**，谨慎对待泰国那套「短袖/吊带/热裤」打法。

### 1.5 价格带（注意按市场换算）
- 泰国案例定价区间：万学员约 150 泰铢、钱学员约 200 泰铢、赵学员 150-200 泰铢。
- **泰国包邮关键规则（仅 TH，可信度中）**：即使报了 SFP 活动，订单**实付满 150 泰铢**才包邮 → 故定价卡在 150 泰铢以上。**马来西亚包邮门槛不同，须查 MY 当地 SFP/包邮规则。**
- 大卖孵化策略按年龄给的价带（美元，通用思路）：16-24 岁 $5-10；25-35 岁 $15-30；35 岁以上 $50+。

## 2. 商城（货架场）优化 SOP — 新店起量第一抓手

> 官方强调：**新店前 1-2 个月先靠商城攒原始销量数据**，再用达人起量。工厂型商家无内容能力，最快路径就是「**选爆品 + 商城优化 + 达人建联**」。

商城订单来源优先级：**搜索 > 推荐 > 频道 > 其他** → 重点运营搜索、推荐、商城频道。

### 第一步：搜索 & 商品详情页优化（SEO）
- **标题**：完整、含关键词；用「数据罗盘 → 商品标题优化」功能加热搜/飙升关键词。
- **商品详情描述 > 200 字符**（含空格符号）。
- 完整填写基础信息：标题、头图、详情页、类目属性。
- 正向影响排名的治理指标：有竞争力的价格、稳定上新频次、视频/直播积累销量、低取消率、低延迟履约率。

### 第二步：每周提报秒杀（Flash Sale）
- **每周提报秒杀**是商城起量核心动作。
- 各市场秒杀活动入口（来源文档给的链接，**可信度低、大概率已失效，仅说明「每个市场有独立活动页」，须到后台找当期入口**）：PH / TH / **MY** / VN / SG 各有独立 `seller.tiktokglobalshop.com/promotion/campaign/...` 链接。
- 叶学员经验：**秒杀活动要多过普通折扣活动**，建议「**5 个秒杀 + 新客优惠券**」组合，曝光更高。

### 商城优化为什么有效
- 上架即自动进商城，操作简单；商城入口在 TikTok 主页显著位置；内含秒杀、新客等坑位帮促单拉新；前期起量快、攒原始数据。

## 3. 达人联盟（Affiliate）建联 SOP — 起量主引擎

> 官方：**联盟出单是女装商家主要出单渠道**，也是新店冷启高效方式，对商家能力要求低于自播/自产短视频。

### 第一步：公开计划「应开尽开」+ 设置免费样品
- 计划类型：定向计划 + 公开计划可同时设，**佣金展示优先级：定向 > 公开**。
- **新商家公开计划要应开尽开**：越多商品开公开计划，被达人选上概率越高；部分达人直接拿商品图做挂车视频，不申样也能带货。
- **建议公开计划佣金 10%-15%**（官方）；东南亚整体佣金率建议 **10%-20%**（其它文档）。
- 开通路径：商家中心 → 联盟带货 → 联盟中心 → 公开合作 → 设佣金（≥10%）→ 选商品 + 填佣金 + 设免费样品数 → 打开同意按钮（商家手动审核达人）。

### 第二步：高效批量建联达人
- **筛选关键指标**（官方）：
  1. 主营类目 = 服饰，粉丝性别 = 女性
  2. 视频平均播放量 ≥ 1000，过往有挂车视频，账号内容垂直、带过服装、有上身/细节展示
  3. 达人回复率、履约率
  4. **粉丝量级不是关键** —— 几千到一两万粉的达人带货数据反而更好
- **批量建联节奏**：1-2 个运营，备好话术，**每天批量建联 100-150 个达人，约 2 小时**；回复率约 5-10%；持续 1 个月攒经验。
- **目标**：挂车达人先做到 **1000**，最终累计 **1000-5000+** 联盟出单才稳定。
- **头部店铺达人结构**：2-5 个强带货达人贡献 **60%-80%** 出单 + 几千个腰尾达人零散贡献 ~40%。
- **建联工具**：第三方「**达秘**」批量邀约（提效）；筛选可用 NoxInfluencer / SocialBook / TikTok Fake Check（查刷粉，印尼/东南亚刷粉较多）。

### 第三步：合作模式与跟进
| 合作形式 | 适用阶段 | 优缺点 |
|---|---|---|
| 纯佣金 | 新店测试达人 | 0 成本，但达人积极性低 |
| **寄样 + 佣金（主力，建议新店用）** | 主力模式 | 成本可控、转化较高 |
| 付费 + 佣金 | 冲刺爆款期 | 成本高但流量有保障 |

- 新店建议「**寄样 + 10%-20% 佣金**」组合。
- **避坑**：要求达人发布后**至少保留视频 30 天**（东南亚达人常删旧视频）；寄样后盯物流，到货当天跟达人确认出视频。
- **深度合作**：对数据好的达人设定向计划、提高佣金、给爆款视频投流（把投流授权二维码发给达人扫码同意）、谈新品合作。
- 商品要先优化好再建联：达人不只看价格，也挑商品质量；佣金要提前算进定价里。

### 话术模板
- **破冰（英文，官方参考）**：
  > Hi [达人昵称]! Love your video about [具体视频内容]! Our new store has similar [服装类型] that would look perfect on you. Would you be open to a free sample?
- 合作说明：提供免费样品 + 佣金比例；东南亚（尤其越南/泰国）对免费寄样接受度极高。
- **原则：释放诚意 + 清晰说明来意 + 提及对方具体视频内容（避免群发被识破）+ 强调利益点**（主推款信息、佣金率、完善售后、响应快）。

## 4. 优惠券 / 营销活动玩法（提客单价 + 撬流量）

### 4.1 优惠券与活动组合（学员实操）
- **善用平台给新店的有限补贴券**：如「160-40 泰铢」券 → **先把售价定到券门槛之上（≥160 泰铢）**，让券能用上，是前期快速出单关键（叶学员）。
- **多档优惠券测转化**：高流量低转化的链接，设不同面额优惠券测哪种转化更高，比直接调价更灵活（万学员）。
- 新客优惠券、多买优惠、组合促销、满减 → 拉高客单价、刺激一单多买。

### 4.2 多买 / 组合 / 心理定价（通用思路）
- **新人首单**：无门槛减 3-5 元；满 $5 减 $2、满 $10 减 $3（东南亚对小额优惠敏感）。
- **秒杀引流**：1-2 款基础款设 $0.99（需亏本测算），展示在首页/详情页/达人直播挂车。
- **多买优惠**：买2件9折、3件8折、第二件半价（适合T恤/打底裤等基础款）；限同品类避免混搭低价。
- **组合购**：爆款 + 滞销款打包（如 $15 连衣裙 + $5 打底裤 = $16 套装）；季节套装（雨季「防晒衫+速干裤」打8折）。
- **价格锚点**：「单买$20 · 2件$30 · 3件$36」引导买3件；零头定价「.99」转化更高。

### 4.3 报官方活动（撬流量加权）
- 报名 **SFP 运费活动、商城闪购、每月大促**，提报后进「限时折扣」推荐池、拿活动专属标签流量入口（如 #TikTokDeals），提升 FYP 曝光。
- 报活动通常要求**销量达门槛**（叶学员提到「销量 > 1xx」之类，具体以后台为准）。
- 节日即使平台没设活动，相关属性商品也会很热 → **提前一个月上架**。

## 5. 数据复盘指标与诊断（板块核心）

### 5.1 商品卡三段转化漏斗（赵学员复盘，最值得借鉴）
单品商品卡盯三个转化率，逐段优化：
1. **曝光 → 点击转化率**
2. **点击 → 加购转化率**
3. **加购 → 成交转化率**

赵学员单品优化一周内的变化（个案数据，作参照）：
- 第二周单品曝光人数升到 2590 人，环比 **+1780%**；
- 点击→加购转化率环比 **+1700%**；
- 加购→成交：从「一天1单」到「一天约20单」；
- 单品周曝光 6150 次 → 拆到每天约 1000 访客，能带动其他单品曝光。

> 关键认知：TikTok 是内容平台，**视频带货页只有「立即购买」没有「加入购物车」**；所以「加购」数据主要来自**商城页面**流量 → 加购量高 = 商城 SEO（标题/主图）做得好。**「加购」比「收藏」更值得盯**。

### 5.2 常用监测维度
- **数据罗盘**：每天看商品曝光度、点击量；看不到即时变现属正常（新店有滞后）。
- **商品卡数据**：主图+标题在各渠道的曝光/点击/转化，按单品优化。
- **关键词榜单**：拿热搜/飙升词优化标题；大量贴合榜单上传商品 → 拉搜索流量（赵学员靠这点把转化拉起来）。
- **销售渠道分析**：分清短视频/直播/商品卡各自贡献，资源投向高效渠道。
- **销售排行榜**：识别畅销/滞销，优化商品组合。
- **客单价**：算出来后用满减拉高。
- **大促高访客低转化时**：上限时秒杀 + 限量优惠券救转化。

### 5.3 推荐池资格诊断（女装官方文档提到的后台路径）
- 方法1：DataCompass（数据罗盘）→ 商店标签 → 推荐 → 表现 → 看哪些商品卡没有推荐池资格。
- 方法2：数据罗盘 → 商品卡 → 商品卡详情 → 看单卡问题与优化操作。

### 5.4 季度目标拆解模板（东南亚千万GMV文档，营销味浓、个案/激进，仅作框架）
分季度「稳基础→快增长→强供应链→冲刺收官」节奏（数字为该卖家个案目标，**非通用基准**）：
- Q1：日均500单 / 月70万；核心=新手村突围、测市场、爆品测试。
- Q2：日均600单 / 月90万；核心=网红直播带货、用户互动放大流量。
- Q3：日均800单 / 月110万；核心=市场监控、营销优化、产品迭代、供应链攻坚。
- Q4：日均1000单 / 月140万；核心=双11/双12大促冲刺、备货。
- **风险机制**：当日销售额 < 目标 80% 触发预警 → 运营经理 2 小时内定止血方案 → 每 4 小时同步恢复进度。

## 6. 履约 / 时效硬规则（官方，可信度高，但仍以后台为准）

> 来源：女装官方文档。**这是会扣分限单的红线**，对马来西亚跨境店同样关键。

- **延迟履约会扣分 → 店铺限单。**
- **商家自发货**：从用户支付成功到「物流商回传入库 + 后台显示已发货」，**不超过 120 小时（5 天）**。即货物**最迟第 4 天**送到仓库，给物流商留 24 小时扫描时间。⚠️ **注：120h 为 2024-12 前旧规；现行已收紧至出单后 72h 入仓，见开店向导 §5、以后台为准。**
- **揽收商家**：用户支付成功后 **48 小时内点击发货**，且**物流商第一次上门揽收就要被成功揽收**（两个条件须同时满足）。
- **赵学员强调**：爆单但发不出货 → 4+2 时效内无法发货 → 达人联盟受限 + 限单 → 别提冲明星店铺。**供应链稳定是冲量前提**。

### 物流方案（跨境实战宝典，可信度中，2025 时点）
- 平台四种方案：**FBT 官方仓 / 3PL 对接仓 / 4PL 尾程配 / 跨境直邮**。
- 中小件爆品官方优先推荐 **FBT 官方仓**（2025 FBT 可享「3日达」，平台一对一支持）；4PL 尾程配是 2025 平台核心策略之一（拓展尾程服务商、降本）。
- 海外仓：头程海运比国际快递省 30%-60%；7天内到货可提复购、退货暂存质检二次销售（文档举例「喜运达」海外仓，**属第三方，非官方背书**）。

> ⚠️ 物流方案、是否开放、费率都在变，**马来西亚跨境店务必以卖家后台「物流/履约」当前可选方案为准**。

## 7. 合规红线（女装官方判罚案例，可信度高）

> 违规会扣分、商品冻结下架、联盟受限。对马来西亚（穆斯林市场）尤其要注意裸露与文化尊重。

1. **山寨假冒**：照搬品牌 / 外观logo类似正品 → 实锤侵权**直接扣 12 分且无法申诉**。上品遇特定图案/logo「以图搜图」核查；背景透出大牌logo虽不一定处罚但不建议。
2. **品牌遮挡**：禁止用贴纸/涂抹/马赛克/胶带遮挡品牌信息（规避行为）。
3. **过分裸露**：内容电商对图像很严。敏感地区裸露 < 1/3，乳沟不过分、不漏点凸点、不露敏感部位；假模不可不穿衣、无暗示动作；不能用马赛克/贴纸遮过度暴露区；**禁售情趣内衣**（普通内衣可直接上品，注意发布规范）。（露肚脐不算暴露；问题多在「遮挡」与「凸点/股沟/臀部」）。
4. **其它**：背景含站外引流信息（疑似引流）会被判问题；类目摆放、图片与SKU名匹配、图片清晰无诱导动作、尊重当地习俗。
5. **差评率**：把控质量、核对尺码、避免货不对板。

## 8. 店铺装修 & 客服（通用要点）

### 8.1 装修（跨境实战宝典 / 大卖孵化策略）
- 用平台设计工具拖拽组件，目标：提 GMV、增停留、提转化。
- **轮播图**重点放：① 流量最高爆款 ② 店铺优惠券 ③ 一键跳转商品支付页。
- 「精选商品」组件：设吸睛标题（爆款专区/新品首发）+ 选布局（列表/网格）+ 最多 30 款（手动/自动排序）。
- 大促做「氛围感 + 强促销」主题装修；秒杀过期要恢复折扣价。

### 8.2 客服 SOP（跨境实战宝典）
- **响应时效**：固定时间查消息（如每 7/12 小时），越快越好；提 12 小时响应率能提店铺评分（钱学员）。
- **差评/纠纷**：先稳情绪 → 24 小时内公开回复 → 引导私信「转移战场」避免发酵 → 给补偿（部分退款/优惠券/补发）→ 协商修改差评。
- **主动索好评**：订单完成后客服联系，可用小额优惠券激励好评 + 促复购。
- 文档附**标准化客服模板库**（售前产品/物流、未收到货、产品不满意、差评应对话术），可直接改用。

## 10. 关键避坑点速查

- **首图别用白底图**：用户1秒决定点不点，借鉴爆品首图风格（换背景/加场景/加卖点短句）。
- **标题别直接翻译供应商原文**：拆解竞品收热搜词，符合当地语言习惯。
- **详情页别只放1张模糊图**：参考拼多多/1688爆款详情，强调卖点+痛点。
- **运费别全让买家承担**：尽量参加免运费计划，用满减/包邮提转化。
- **别一味从众**（赵学员核心）：差异化人群定位比跟风Y2K更能打大店。
- **别先追求一时爆量**：核心是做成明星店铺 / 拿 TopChoice 徽章（提曝光、平台流量扶持、提买家信任）。
- **冲量前先对接稳定供应链**：否则爆单发不出货 → 限单 + 联盟受限。
- **马来西亚特别提醒**：穆斯林市场偏保守，选品/图片避开裸露与不尊重文化的内容；价格带、包邮门槛、活动入口都要按 MY 重新核对，不能直接套用泰国案例数字。
