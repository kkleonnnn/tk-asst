## 1. 店铺没流量

### 原因（自查清单）
1. **体验分低**：商品质量 / 物流服务 / 售后等多维度过低 → 平台判定服务不可靠 → 减少流量扶持。
2. **权重未起**：新品没沉淀、没销售记录与评价，权重低、流量自然少；**长期不优化**会一直起不来。
3. **违规操作**：刷单、虚假信息 → 权重下降、流量减少。
4. **选品**：① 冷门品类本身市场有限；② 红海品类（已有高知名度/好评/价格优势的卖家）难脱颖而出。
5. **不应季 / 不应节**：产品不符合当下季节或节日需求。
6. **商品信息差**：① 标题无用户常搜关键词、描述不自然 → 平台识别不到核心内容，影响搜索排名/推荐；② 主图模糊、不展示细节、不合目标受众审美 → 没点击；③ 详情页空洞、不展示功能/优势/使用场景 → 浏览不转化。

### 流量提升策略（本课口径）
1. **选品贴合当地**：参考当地人喜好 + 季节 + 地域特色找款，满足真实需求。
2. **商品信息优化**：按热搜关键词优化标题、补全商品属性；提升主图/副图/视频质量。
3. **用平台资源**：上架热门品 + 设优惠券/折扣/秒杀/满减提曝光。
4. **善用「商品机会」**：里面的热门商品**自带流量**，思路是「**收同类不同款**」。
5. **推广策略**：① 设**合理佣金**，用公开计划吸引达人主动带货；② 蹭热门搜索词热度优化标题；③ 积极报平台活动。

> 增量提示：本课唯一的"独门"动作是 **「商品机会」里抄同类不同款热门品** 这条选品-引流捷径；其余（关键词/主图/详情/活动）在 `05` 与 `08` 已有更细 SOP，按那边执行即可。

## 2. 店铺不出单

### 原因
1. **运营**：权重低→流量少（新品无流量正常，靠优化积累销售记录与评价才能起权重）；**勿碰违规**（虚假宣传、刷单）。
2. **体验分低**：同 §1。
3. **没接住销售机会**：视频爆了**没及时上架商品链接**错过流量高峰；**忽略评论区互动**流失潜客。
4. **选品**：市场需求小 / 不贴当地趋势热点 / 红海无独特竞争力。
5. **价格不合理**：①过高且无高价值感知 → 卖不动；②过低 → 用户质疑质量 + 利润太薄难保品质服务。
6. **详情页不吸引人**：功能/优势/场景介绍不清、图片视频模糊 → 进店不转化。

### 出单秘籍（本课口径）
1. **盯商品卡数据**：找出**曝光率/点击率高、但转化率低**的链接重点优化（主图、详情图、价格）。
2. **价格调整优先用「折扣」**：先查是不是价格过高拖累出单；是就用**折扣/秒杀**调到有竞争力的价位（讲师推荐直接用折扣，"方便、简单、直接"）。
3. **动态定价**：高流量低转化的品**先适当降价**，转化/销量起来后**再逐步抬价**。（详见 `03-商品上架/商品发布上架与定价.md`）
4. **持续优化商品信息**：商品卡 + 详情页（功能/优势/场景充分展示）。
5. **设计优惠活动**：报促销、设券/秒杀/满减做"临门一脚"。
6. **长期数据分析**：找出"流量好+转化高"的款，往这个方向扩大上新，**避免上新零流量产品**；按结果灵活调选品/价格/推广。

## 3. 店铺出单少

> 与 §2 同源，差别是"已经能出单、但量上不去"。原因聚焦**款式宽度**与**价格/达人**。

### 原因
1. **选品**：① 款式太少，满足不了买家多样需求（**选品方向尽量多几个**）；② 价格过高/过低（用动态定价）；③ 市场需求小、不贴当地趋势热点。
2. **运营**：推广不足（没用足平台工具/活动）、详情页弱、没接住爆点（同 §2）。
3. **竞争**：红海里新店难脱颖而出，**产品无独特卖点更难**。
4. **价格无优势**：再次强调动态定价——**前期要有竞争力的价格（偏低但不亏钱）**才吸引购买。

### 爆单攻略（本课口径）
1. **达人合作**：
   - **公开计划**让达人主动找你；**定向合作**你主动找达人。
   - 对**数据突出的达人提供免费样品**，帮你更好带货、提知名度与销量。
   - **价格/佣金**：达人带货**初期可让出利润、提高佣金**；流量与曝光打开、数据转好后**再逐步调价**。（达人实操见 `05-流量增长/达人建联与邀约实操.md`、`达人联盟.md`）
2. **平台推广优化**：用好广告投放/活动报名；优化标题/图片/详情提搜索排名；**提升产品质量与服务**以提满意度与复购。
3. **数据分析 + 调整策略**：持续找"流量好+转化高"的款扩大上新；按数据调选品/价格/推广；持续优化盯数据。

## 4. 不赚钱 / 赚得少

### 原因
- **成本过高**：① **采购成本**（没找到性价比供应商）；② **库存成本**（积压占资金 + 仓储成本）；③ **推广费用**没转化为利润；④ **物流成本**（重货/大件运费高）。
- **定价不合理**：前期为出单把价压到**低于成本**亏损；或预留利润空间太小难盈利。
- **销量低**：出了单但量有限，形不成规模效应，**摊不薄固定成本**。
- **其他**：① **复购率低**（开发新客成本远高于维护老客）；② **平台政策限制**——**收款周期长影响资金周转**（⚠️以官方当期回款规则为准）；③ **市场竞争激烈、价格战**压缩利润。

### 投产优化策略（本课的核心增量：分两步 + 建供应链）
**第一步 · 先降成本、拿低价起店**（前期利润低是正常的，为后面铺垫）：
- 采购**货比三家**，横向比价格与质量挑最优。
- 订单**起量后转变"一件代发"思路**：从供货商/货源直接拿货 → **小批量发到货代** → 进而**囤货到海外仓**。可提利润、提发货速度。

**第二步 · 再提利润**：
- 按成本与市场**合理定价、适当抬价**，平衡"有竞争力"与"能盈利"。
- 优化选品/推广/体验 → 提销量、提复购、形成稳定客群。
- **拓展业务领域增加收入来源**（如扩大跨境市场范围）。

**建供应链（讲师强调的长期解）**：
- 不赚钱时从**采购价**入手找更优供应商降成本。
- **海外仓**：缩短配送时间、提满意度，并**降低长途运输破损**（破损也是成本，破损少→利润高）。
- 一句话路线图：**一件代发 → 货代小批量 → 海外仓囤货**，从货源和物流两端建供应链提利润空间。

> 衔接：货代/海外仓的具体操作与运费见 `04-履约发货/发货指南与运费.md`；选供应商/货源见 `02-选品/*`。

## 5. 店铺评分低（差评/低分的来源与修复）

### 原因（四大类）
1. **商品质量**：材质低劣、做工粗糙、功能故障 → 低分差评。
2. **货不对板**：功能不达标、效果不如意、虚假宣传；描述图与实物明显差异、规格参数不准 → 用户误解、体验差、降分。
3. **物流体验**：① **发货延迟**（超承诺时间）；② **包裹破损/丢失**；③ **物流信息不透明**（查不到运输状态、信任感差）。
4. **售后服务**：① **响应不及时**（咨询等太久）；② **处理不当**（流程繁琐、效率低、态度差、推诿）；③ **信息不准确**（退换货政策不明确 → 误解）。

### 优化策略（按来源拆）
- **质量把控**：建质检流程 + 选优质供应商并定期评估；**入库/发货前全面检查**（外观、功能、材质）；建**快速响应机制**，质量问题第一时间沟通处理（退换货/退款）；对差质量产品**溯源整改**（改工艺、换供应商）。
- **避免虚假宣传**：详情页**准确**描述材质/功能/规格；提供**详细且正确**的图片与使用说明。
- **物流优化**：物流信息**实时可查**；出问题（破损/丢失）第一时间沟通并给方案（补发/退货）；与物流伙伴协调溯源防复发；**用海外仓缩短时效**、提前做预案。
- **售后优化**：提响应速度（专业客服团队 + 定期培训）；建**多渠道客服**（在线/电话/智能）；设**响应时间标准**并监督；简化退换货流程、建分类记录与回访机制。
- **信息优化**：上架前核对描述/图片/规格与实物一致；加使用场景/搭配建议增吸引力；优化详情页排版；**定期检查更新商品信息**，避免模糊/夸大/误导用语，发现有误立即改。
- **收集反馈**：重视评价、深挖差评、定期收集建议持续改。
- **店铺形象**：报平台促销提曝光知名度；做精美店铺页面 + 优质服务增信任与好感。

> 体验分的**官方机制（三大领域、考核指标、入口、申诉路径、展示门槛）**不在本课范围，**见 `07-客服与店铺健康/客服与店铺评分.md`**——那是官方招商口径，比本课讲稿更准。

## 6. ⚠️ 风控警示：本课"提高综合评分 6 点"里的好评操纵灰区

讲师在结尾给出"提高店铺综合评分"6 点，其中**第 1、2 点涉及操纵评价，本店不建议照做**，其余 4 点是正当客服建议：

| # | 原文做法 | 风控判断 |
|---|---|---|
| 1 | 用「**达密**」(达人/客户批量私信工具) **批量私信已完成订单的买家"大胆讨好评"**，一次几百几千 | 🔴 **高风险**。诱导/批量索取好评属**操纵评价**，TikTok 对评价操纵、激励性好评有治理，可能扣分/限流甚至处罚。**不建议**。 |
| 2 | 给**之前差评/退货退款的买家"补偿"换一个好评 / 换撤差评** | 🔴 **高风险**。**以补偿换好评/撤差评**是典型的评价操纵，平台明确禁止。**不建议**。把不满用户转为满意可以，但**不应与"给好评"做交易**。 |
| 3 | 设**自动回复 + 人工客服**快速响应，**要求 24h 内必响应** | ✅ 正当，对齐体验分的 24h 响应率。 |
| 4 | 制定**清晰退换货政策**、**主动**解决问题 | ✅ 正当。 |
| 5 | **培训客服避免情绪化沟通**（大卖客服都及时且专业） | ✅ 正当，且重要。 |
| 6 | 选**稳定物流合作方**、提供更多物流选项 | ✅ 正当。 |

> 衔接 09 板块口径：**提升评分的正路是把商品/物流/售后做好 + 真实满意用户自然留评**，而非批量索评或拿补偿换评。这与 `账号注册养号与防骗.md`、09 README 一致——**官方硬规则（评价真实性、违规分制）≠ 培训里的"江湖打法"**。刷单、虚假宣传在本课多处也被列为"会掉权重"的违规，自相印证：**别碰**。

## 7. 衔接与冲突提醒

- **定位**：本文偏"诊断地图/索引"，不重复机制细节。遇到具体怎么做，按第 0 节表格"细节去哪看"跳转既有文件。
- **与既有文件无硬冲突**，但有**口径差异需注意**：
  - 本课把"刷单/虚假信息"列为掉权重原因（正确导向），但**第 6 节 1–2 点的好评操纵与之自相矛盾**——已标红，**以"不操纵评价"为准**。
  - 本课所有"先低价起店、后抬价"的动态定价建议，**前提是"低但不亏钱"**；落地时务必结合 `04 发货指南与运费` 的真实到手成本（含运费/退货/破损）核算，别真做成亏损单。
- **时效**：本课无后台数字，但凡涉及**收款周期、活动入口、佣金/广告工具界面**，一律以**官方后台当期**为准。
