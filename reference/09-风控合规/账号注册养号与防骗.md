## 0. 一句话总览

注册一个能正常用、不黑屏、能带货的 TikTok 个人号，本质是**让"店铺 IP、账号 IP、手机 IP"三方一致地伪装成目标国家的真实本地用户**，再给新号一段**养号缓冲期**积累权重。但对**跨境店**而言，开店走官方入驻流程即可，下面这套重资产环境配置不是必须——**最该记住的是第 6 节防骗清单**。

## 1. 注册环境配置：硬件 + 软件（短视频玩法的"五大基础技能"）

来源 3（140页教程）把起号拆成五块，是这派玩法的总框架：

**手机选购 → 海外环境初始化 → 海外手机卡 → TikTok APP 下载安装 → 海外邮箱注册 → 海外网络环境**

### 1.1 设备（手机）

- **建议买二手/闲置机**，原因：① TikTok 运行时不能插国内卡且飞行模式无效；② 运营中常要重置/恢复出厂；③ 批量做号要控成本。〔经验，可信度中〕
- **iOS vs 安卓对比**（来源 3）：
  | | 易用性 | 性价比 | 批量运营 | 保值率 |
  |---|---|---|---|---|
  | 苹果 | 简单 | 低 | 难 | 高 |
  | 安卓 | 较复杂 | 高 | 好 | 低 |
- **苹果**：iPhone 8 及以上（刷机 SOP 的最低要求）；直播建议 iPhone 11+（有广角）。
- **安卓**：推荐 Google 亲儿子机、选欧版（可解 BL 锁），自带"谷歌三件套"；**资料明确标注安卓"比较麻烦不建议"**（来源 1）。

### 1.2 海外参数配置（iPhone 刷机 SOP，来源 4）

> ⚠️ 刷机会清除手机全部数据，务必用闲置机。核心是把手机"伪装"成目标国家的全新本地设备。分步要点（旧版 iOS 界面，仅记关键决策）：
- **恢复出厂 → 激活时**：选**对应国家的语言与地区** → 手动设置 → 连 WiFi → 等待激活 → 同意协议。
- 关闭指纹/Face ID、暂不登录 Apple ID（选"忘记密码或没有 Apple ID"→ 稍后在设置中设置）、不设密码、不从旧机/iCloud 传输 App 与数据。
- 定位、Siri、使用情况分析、自动更新——**全部关闭/不启用**。
- **环境设置（进系统后）**：仅用 WLAN；改手机名为本地化名字；时间→关闭"自动设置"→手动选**目标国家时区**（如 GMT）；**辅助功能里关闭：应用程序跟踪、分析与改进、苹果广告**；关闭位置警报。
- 之后才登录"海外 Apple ID"→ App Store 下载 TikTok（认准 Sign up 是注册、Log in 是登录）。

> 这些"关追踪/关定位/对齐时区语言"的动作，目的是降低设备指纹暴露"中国身份"的概率。**是否仍有效、是否触犯条款，自行判断。**

### 1.3 软件环境影响"视频 0 播放"的因素（来源 1）

节点（翻墙线路）、伪装度、地区、广告追踪、语言、邮箱（Gmail / Outlook）——任一不对都可能 0 播放。

## 2. 海外 SIM 卡（仅短视频/直播玩法相关）

来源 3 给出的"为什么需要卡"时间线（**均为 2019–2021 旧事，仅作背景**）：插国内卡无法浏览 TT、香港不可用、插国内 SIM 会黑屏、未插海外卡时购物车直播/下单异常。

- **能不能不插卡？** 来源 1 明确：**不插卡也可以，手机卡不是必要条件**；但**不能插国内卡**，要插**目标地区**的卡。

## 4. 养号 SOP 与账号权重（来源 1，最核心、最该用的部分）

> 这一节即使做跨境店、将来要不要自己发内容引流，也用得上。但所有数字都是**经验值**，非官方。

### 4.1 一机一号一 IP（防关联经验）

- 新号尽量**一机一号一 IP**；运营满一个月或已有一定粉丝量后，可同机加号，**建议一台手机最多 3 个号**。
- **不要短时间频繁换 IP**，易被判定为问题设备、掉权重。

### 4.2 包装账号（用当地语言/习惯用语）

- **名字**：与产品/店铺名相关、易记、与店铺名一致。
- **头像**：产品图 / 真人 / Logo。举例——真人头像（如穿白大褂的"牙科医生"）增加信任；品牌型可放 Logo。
- **简介 + emoji**：一句话说清账号做什么（示例排版"前职业化妆师🪞 现全职美妆博主💄"）。
- 能绑的社媒（Instagram、YouTube）都绑上；**注册时一次性把资料填全**。
- **不要频繁改资料**；要改就**错开时间**（今天改名、明天改简介），别一次全改。

### 4.3 缓冲与养号节奏（关键时间参数，均为经验值）

1. 注册好后**别急着绑店**，先刷对标同类目视频提活跃度。
2. 每天**自然刷系统推荐 0.5–1.5 小时**，分时间段刷；喜欢的视频**看完再点赞**。
3. 刷 **3–5 天**后开始仿拍、发布原创。
4. **早中晚分时段测曝光**（按店铺所在国时间）：早 7–9 点、午 11–13 点、下午 16–18 点各发 1 条；测出最优时段后次日集中在该时段发。
5. **起始播放 200–500 属正常**，继续优化内容。
6. **音乐用商业曲库**（否则后期会被静音）；文案要有共鸣（引导评论/收藏/转发）；**标签 3–5 个**（垂直词 + 泛标签如 #fyp #ForYou），别堆太多。
7. **运营 1 周到半个月**、看账号状态再绑店。

### 4.4 账号权重影响因素（运营注意事项）

- **删视频**：大量删会影响推荐；**数据差/活动过期的视频要"隐藏"而非删除**。
- **更新频率**：断更/低频会被判"不积极"掉曝光；建议**一周更 3–7 天、每次 1–3 条**。
- **健康基线**〔经验值〕：条均播放达 **500** 为健康；**72 小时播放达 2k–3k**；最佳发布时间按粉丝画像在线时间反复测。
- **内容违规**：违禁词、违规画面/操作、不良留言——都会扣权重。

### 4.5 核心指标（推荐逻辑）

- **完播率**：从头看到尾算一次有效播放，越高越易被推荐（最重要）。
- **点赞量**：看**赞播比**，不只看绝对数。
- **评论量**：有话题性 + 及时回复 → 提高二次曝光、建立口碑。
- **转发量**：是否引发共鸣、粉丝黏性与"带货能力"。

## 5. 常见问题排查（来源 1 + 来源 5）

| 现象 | 可能原因 / 处理 |
|---|---|
| **账号绑不上店铺** | 多半是注册时**环境/国家选错**；要点：**店铺 IP、账号 IP、手机 IP 必须一致**。 |
| **新号无法关注别人** | 给缓冲期，**等 1–3 天**再试。 |
| **视频 0 播放** | 先查内容（静态/静音/糊/搬运/重复度高），再查环境（节点/伪装度/地区/语言/邮箱）。 |
| **新号流量断崖**（前期上千上万，2–5 天后跌到 0） | **80% 是内容质量问题**（优化内容）；**20% 是账号问题**（查节点，或弃号重起）。 |
| **老号（有粉丝）掉量** | 平台预估播放降低，多因**断更**或**严重违规**。 |
| **登录路径**（来源 5） | Login → "Use phone / email / username" → 输邮箱密码 → 过形状验证（选两个相同形状）→ 登录成功。 |
| **切中文界面**（来源 5） | 右上角"三横"→ Settings and privacy → Language → App language → 中文（简体/繁体）。 |

## 6. 防诈骗清单（来源 6，最实用，务必牢记）

> ⚠️ 这是整个 C5 资料里**对实操最有保护价值**的一节。

### 6.1 跨境骗局的典型特征（命中即高度警惕）

1. **"什么资料都不用、一分钟开好店"** —— 不可能。**所有官方跨境店都要个体/企业执照。**
2. **不走正规聊天渠道** —— 不在官方工单/常用渠道，而把你引到微博、公众号等奇怪地方聊。
3. **"免费开店"** —— "免费才是最贵的"，套路在后面。
4. **诱导下载山寨 App** —— 会细心问你"安卓还是苹果"，给一个来路不明的网址（正规 App 上不了 App Store/应用商店，只能私下让你装山寨包）。
5. **设"回款周期"骗货款** —— 告诉你不同级别不同回款时间（一周/两周），让你不停出单铺货，**骗你垫的发货货款，但永远无法提现**。

> 老韭菜心得原话："**免费才是最贵的，你担心被人占便宜，人家惦记你本金。**"

### 6.2 如何识别正规电商

- **认准官方网站 / 官方 App / 正规渠道**；除应用商城里的软件外，**不要安装任何对方甩链接的不明软件**。
- 百度搜 "tiktokshop" 应看到带**官方认证标识**的 TikTok Shop 跨境电商入口。
- 正规 TK 小店玩法（资料自述）：**不需翻墙、不需下载额外应用**（官网入口登录即可）、**采购拿货走官方平台、与"招商方"无资金往来**。

> 用第 6.2 节当**自查标尺**：如果有人让你"翻墙 + 装 APK + 买/租 Apple ID + 按回款级别铺货"，对照 6.1 逐条核验。**注意：来源 4（刷机 PDF）里"找开课老师买翻墙软件、买/租海外 Apple ID"等做法，本身就触及 6.1 的灰区——这正是要警惕的对象。**

## 7. 与现有规则的衔接 / 冲突提醒

- 本笔记的**养号/防关联/环境伪装**属于**行业经验**，与 09 板块 README 强调的口径一致：**官方硬规则（禁限售、违规分制、申诉窗口）≠ 行业经验（一机一店、住宅 IP）**——后者只作运营建议。
- **资料内部就有冲突**：养号资料默认"做短视频/直播、需海外卡/翻墙"；而防骗资料自述"做商品卡、不翻墙、不做短视频"。**说明这批资料混了两派玩法，本店应以官方跨境店入驻流程为准**（见 `01-开店入驻`），不要被"必须刷机翻墙才能开店"误导。
- 所有数字（3 个号/台、72h 2k–3k、200–500 起播、Apple ID 价格、SIM 资费等）**一律视为经验值，以平台/官方后台实际为准**。
