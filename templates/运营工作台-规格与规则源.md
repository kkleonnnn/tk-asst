# 运营工作台 · 规格与规则源（给"建表会话"的施工图）

> **用途**：让一个新的 Claude Code 会话（已加载本 skill）据此构建 `templates/TK运营工作台.xlsx`。
> **形态**：**一个 Excel 工作簿**，多个 sheet，围绕同一批 SKU 串联。使用者=校长（运营·会 RPA）。
> **三层关系**：本文件 = **规则源（markdown，可版本化）**；生成的 `.xlsx` = **模板**（进 `templates/`，小、可入库）；校长填真实数据的**工作副本 = 留本地、不入 git**（频繁变 + 含真实店铺数据）。
> ⚠️ **所有费率/税/SLA 以官方后台当期为准**——集中放「参数配置」sheet，官方一变只改一处；每个数值旁注"以后台为准 + 核对日期"。

## Sheet 规格

### Sheet「参数配置」（先建，其余 sheet 引用它）
| 参数 | 默认值 | 备注 |
|---|---|---|
| 汇率 MYR→RMB | 1.6997 | 用户按当前改 |
| 交易手续费 | 3.78% | 含 8% SST，2024-09-05 起；以后台为准 |
| 平台支持费 | RM0.54/单 | RM0.50 + 8% SST，2026-02-15 起 |
| LVG 税率 | 10% | 适用售价 ≤ RM500；>RM500 走常规进口（表内提示）。注：LVG 平台代收，此处算给卖家看真实税负 |
| 类目佣金率 | 用户填 | 按后台费率表；示例 非Mall 7–11%、Mall 10–15%（含SST） |
| 达人佣金 | 5% | 主推款 8–15% |
| 目标利润率 / 退款损耗预留% / 广告预留% | 用户设 | |
| 四象限阈值（曝光/成交率分界） | 用户设 | 见 Sheet3 |

### Sheet0「SKU主表」（贯穿全表）
列：`编号 | 品名 | 类目 | 货源链接(1688/PDD) | 进货价RMB | 重量g | 尺寸 | 清真合规(是/否/NA) | 禁限售核查(通过/需资质/禁售) | 532分类(基础/微创新/趋势) | 状态(候选/待上架/在售/淘汰) | 上架日期 | 备注`
> "禁限售核查"做条件格式:**禁售=红(一票否决)、需资质未备=黄**。

### Sheet1「选品打分」
对主表每品按维度打分(1–5)×权重 → 加权总分降序 → 标"优先上架 Top N"。
- 维度(权重可调):客单价/利润空间、扛运费能力(轻小件优先)、需求热度(数据罗盘)、竞争度(蓝海高分)、退货风险(低风险高分)、趋势/季节契合(斋月等)、清真合规契合。
- **一票否决**:禁售 / 需资质未备 / IP 无授权 / 带球队 logo → 直接判淘汰、不进利润表。

### Sheet2「利润计算」（核心，公式联动）
输入:售价(MYR,可试多档)、进货价RMB(引主表)、跨境物流(MYR)、活动让利%。
- 进货成本(MYR) = 进货价RMB ÷ 汇率
- 佣金 = 售价 × 类目佣金率　｜　交易手续费 = 售价 × 3.78%　｜　平台支持费 = RM0.54
- 达人佣金 = 售价 × 达人佣金率　｜　LVG = (售价≤500 ? 售价×10% : 提示走常规进口)
- 活动让利 = 售价 × 让利%　｜　退款损耗 = 售价 × 退款预留%　｜　广告预留 = 售价 × 广告预留%
- **利润(MYR) = 售价 − 进货成本 − 跨境物流 − 佣金 − 交易手续费 − 平台支持费 − 达人佣金 − LVG − 活动让利 − 退款损耗 − 广告预留**
- 利润率 = 利润 ÷ 售价
- **最低成交价红线** = 反解令利润=0(或利润率=目标%)的售价;当前售价 < 红线则标红。
> 字段顺序参考 起店思路 §6"商品定价表"模板。

### Sheet3「商品四象限分层」
上架后回填:曝光、点击、加购、成交、退款。
- 三段漏斗(源自 实战案例 §5.1):点击率=点击/曝光、加购率=加购/点击、成交率=成交/加购。
- 流量轴=曝光(或点击)对阈值;转化轴=成交率对阈值 → 自动判四象限 + 建议动作:
  - 高流量高转化→主推(加库存/活动/达人/试广告)
  - 高流量低转化→查价格/运费/优惠/主图/详情/评价/客服
  - 低流量高转化→加曝光(达人/短视频/直播/商城活动)
  - 低流量低转化→下架 / 改图改标题重测 / 放弃
> 阈值放「参数配置」,可调。

### Sheet4「达人邀约话术」
取自 `reference/05-流量增长/达人联盟.md`:破冰(英)、马来语版、寄样、跟进、提佣维护;列"渠道(马来=WhatsApp)";留 `[达人昵称][视频内容][品类]` 占位符,多版本可复制。

### Sheet5「客服FAQ+自动回复」
取自 `reference/07-…客服与店铺评分.md`:欢迎语;按订单状态(待支付/待发货/已发货/已送达/已完成/已取消/退货退款)的自动回复;FAQ(物流时效/尺码/材质/退换货/优惠券/发票/售后)。中/英(+马来)对照。

### Sheet「说明」（给校长）

3. 费率/税变了,只改「参数配置」。
4. 话术/FAQ 不够用,找 AI 现生成马来语版再贴回。
