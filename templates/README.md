# templates —— 工具与模板

- `TK运营工作台.xlsx` —— ★**运营工作台**（8 个联动 sheet：参数配置 / SKU主表 / 选品打分 / 利润计算 / 商品四象限分层 / 达人邀约话术 / 客服FAQ / 说明）。
  - 用法：**复制成你自己的工作副本填真实数据**——工作副本**留本地、别提交 git**（本仓库 PUBLIC 公开）；只填输入区，勿覆盖公式区。
- `运营工作台-规格与规则源.md` —— 工作台的**规则源/施工图**。改工作台先改这里，再用建表会话重生成 xlsx（源→产物一致）。
- 其余可复用模板（listing 文案、客服话术、报告格式）后续按需补充。
