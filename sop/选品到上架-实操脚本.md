# 选品 → 找货源 → 采集 → 定价 → 上架 · 实操脚本（可复用循环）

> **类型**：sop（标准操作流程）——**动作级、可照着一步步点**的上新流水线脚本。
> **定位**：本脚本是旗舰《[TikTok-Shop-0到稳定出单-执行计划](TikTok-Shop-0到稳定出单-执行计划.md)》**「阶段 2：选品/定价/首批上架」的动作级展开 + 可复用循环版**。执行计划讲"第几天做什么"，本脚本讲"每上新一款/一批，在货叮咚里具体点哪、每格填什么、哪几关必须过"。
> **适用范围**：TikTok Shop **马来西亚 · 跨境店（从中国发货）· 无货源/铺货**打法，用第三方 ERP「货叮咚」。切泰/越/菲/新或本土店，费率/运价/类目/合规需重核。
> **前置（本脚本假设已就位，未就位先回执行计划阶段 0–1）**：店铺已注册通过、保证金/收款已配、发货仓与退货仓已设、货叮咚已授权 TikTok 店铺 + 已授权 1688 会员账号、采集插件已装。
> **校准铁律**：所有**费率/佣金/税率/运价/发货 SLA/禁限售**一律以**官方后台 + 学习中心当期**为准，不用旧 PPT / 泰国录制示例值（如 9.63%）硬套马来。
> **可信度标记**：✅高（官方口径）｜🟡中（服务商/多源，方向可信）｜🟠低（招商/讲师经验值，仅作节奏参考、非承诺）。
> **用法**：每上新**一款或一批**就照走 Step 1→5 一遍；`[ ]` 是勾选项；🔴 = 必过卡点，不过不许进下一步。

## Step 1 · 选品：找到潜力款 → 方法全集见 [02 选品洞察](../reference/02-选品/选品洞察方法品类与策略.md) / [02 测款与市场](../reference/02-选品/选品测款与东南亚市场.md)

**核心心法**：数据找赛道 → 量化筛竞品 → （测款留到上架后）→ 一国一策裁剪。

1. **找赛道（官方后台 4 个免费入口，优先）**
   - [ ] 数据罗盘·热门商品榜（数据分析 → 商品卡 → 热门）
   - [ ] **商品机会中心**（商品 → 商品机会）——可直接从热销主图反推同类不同款
   - [ ] 关键词榜单（数据分析 → 商品卡 → 热门/飙升/高潜力词）
   - [ ] 畅销商品排行榜（数据分析 → 商品卡 → 畅销）
2. **量化筛竞品（别靠感觉，🟠 招商经验阈值）**
   - [ ] 近 30 天**销量增长 > 50%**；评论区高频 **"Where to buy"**；达人带货视频**点赞 > 1 万**
   - [ ] 优先**轻小件 < 500g、客单价 $15–30 的标品**（运费低、好测款）
3. **挑 1–2 个选品法用**：关键词卡位（后台热词→百度翻译看含义→布进标题）／季节+节日双筛（雨季防潮、斋月穆斯林时尚、开学季）／差异化错位（避美妆/穆斯林时尚红海，深耕细分）／1688 选品法。
4. **打分排序**：把候选品填进工作台 **「选品打分」sheet**，按分排序，选 Top 款进 Step 2。

- 🔴 **卡点 A（选品扛不扛运费）**：**进货价 > 50 元人民币要慎重、尺寸太大千万别选**（马来运费高，选错直接亏）。
- 🔴 **卡点 B（合规初筛）**：先过一遍禁限售（**矫正/框架眼镜禁售**、盲盒/珠宝限售、宗教材料等，见 [09 风控合规](../reference/09-风控合规/)）；马来**清真合规**（食品 Halal、服饰避暴露/禁忌图案与颜色、**居家不带宗教关键词**）。
- **验收物**：候选清单 N 款 + 工作台打分 Top 款（带对标竞品链接/数据）。

## Step 2 · 找货源：1688 锁定源头

1. [ ] 在 **1688** 搜同款/相似款，锁定 1–N 个货源。
2. [ ] 逐个确认：**支持一件代发**（无货源必需）、**进货价**、**重量与尺寸**（直接影响 Step 4 藏价利润，较重的向源头确认实重）。
3. [ ] 优先选**发货快、库存足、支持小批量/一件代发、仓库离货代仓近**的供应商。

- 🔴 **卡点 C（供应链守 72h）**：无货源是"出单才去 1688 下单"——**供应商发货慢/断货 → 你超 72h 入仓 → 掉分**（发货 SLA 见 [04 履约发货](../reference/04-履约发货/)）。发货慢/易断货的源头直接淘汰。
- **验收物**：每个候选品对应**至少 1 个可一件代发的 1688 货源链接** + 进货价 + 实重。

## Step 3 · 采集：货叮咚采到"采集箱" → 详见 [03 §2–§3](../reference/03-商品上架/货叮咚采集上架与定价工具链.md)

1. **采集配置**（采前先设过滤，`产品 → 采集配置`，§2.1）
   - [ ] 打开过滤：**带品牌 / 零库存 / 不包邮 / 客单价过低过高 / 关键词** 的货源自动跳过；设好**保存配置**。
2. **采集**（`产品 → 插件采集`，§2.2）——四选一：1688 商品页点采集 / 拼多多页点采集 / 粘 1688·淘宝链接 / **1688 精选货源一键铺货** → 全部进**采集箱**。
3. **认领配置**（`产品 → 认领配置`，选 TikTok 模板，§3）——设：统一 SKU 生成、图片补主图、**产品翻译**、**违禁词/敏感信息过滤**、尺码图、包裹重量尺寸预设。
4. **认领生成草稿**：`采集箱 → 选中 → 认领 → 选 TikTok 平台 → 选 TikTok 店铺 → 选认领配置 → 认领并（立即/定时）发布` → 生成**草稿商品**。

- 🔴 **卡点 D（图片合规，最易踩）**：采来的 1688 图**必须去中文/工厂信息/联系方式/品牌 Logo/水印**（在认领配置或 Step 5 编辑里处理；**⚠️ 图片翻译/图片语言翻译是货叮咚付费功能**，文字翻译一般免费）。**违禁词过滤务必开**。
- **验收物**：采集箱有货 + 认领生成草稿商品。

## Step 4 · 定价：藏价模板，保证不亏 → 详见 [03 §4](../reference/03-商品上架/货叮咚采集上架与定价工具链.md) + 定价公式

> **定价公式（🟡 讲师口径，结构对，费率以官方为准）**：
> **售价 = 采购价 + 运费 + 货代费 + 达人佣金 + 利润 + 平台佣金 + 交易手续费**
> 把所有成本 + 目标利润**全部前置藏进售价**，倒推保证不亏。定价模板 = 帮你自动藏价的工具（`产品 → 定价模板 → 创建模板`）。

1. **① 模板信息**（§4.2）：模板名自取；**站点选马来西亚**；**店铺类型选「跨境」**。
2. **② 成本信息**（§4.3）—— 🔴 **卡点 E（费率必须马来当期）**：系统自动带出的、教程里泰国录制的示例值（**平台佣金 9.63%、交易手续费 3.21%、增长服务费 8.03%、基建费 0.23%、关税 32.68%、VAT 0** 等）**一律不可照抄**，逐项去 `店铺后台 → 帮助与支持 → 学习中心` 查**马来站当期**填。
   - [ ] 头程运费：选**「官方跨境物流」**（无货源=货→货代→官仓→出境），点**「官方运价表」查马来站**。
   - [ ] 尾程运费：**填 0**（买家付，卖家不出；必填项填 0）。
   - [ ] 货代费用：**2~3 元/件**（示例填 3；须与 Step 6 利润明细回填的货代费对齐）。
   - [ ] 重量取值：默认「商品包裹重量」（**重量要准，直接影响藏价利润**）。
3. **③ 利润信息**（§4.5）：利润方式选**「售价利润率」**；利润率示例 **20%**；**必填折扣**（填 **7** = 店铺后台做 **30% off**；填 6 = 40% off）——**只填利润不填折扣容易亏**。
4. **达人佣金藏价**（§4.6）：模板藏几个点 = 你打算给达人几个点。**藏 5% 但实付 15% → 多的 10% 从你利润里扣**；前期不给达人 → 藏的 5% 变你利润。按主推/普通款预留 5%~15%。
5. **④ 尾数处理 → 价格试算 → 保存**（§4.7）：点「价格试算」输采购价+重量看折前/折后价与利润；**务必点「保存」**。
6. **交叉复核**：把采购价/运费/货代费/佣金填进工作台 **「利润计算」sheet**，确认**单件利润 > 0（且覆盖退款/广告预留）**，与货叮咚藏价对照，两边对得上才算过。

- 🔴 **卡点 E 小结**：①费率用马来当期（别用泰国示例）②必填折扣 ③重量要准 ④工作台利润 > 0。
- **验收物**：定价模板已保存 + 工作台确认利润为正。

## Step 5 · 上架：编辑草稿 → 发布 → 审核可售 → 详见 [03 §5](../reference/03-商品上架/货叮咚采集上架与定价工具链.md) + [03 商品发布上架与定价](../reference/03-商品上架/商品发布上架与定价.md)

1. **发布配置**（`产品 → 产品发布` 先设，§5.1）：默认发货仓、多国配置、站点语言翻译、长度优化 → 保存。
2. **编辑草稿商品**（§5.2 + 上架自查）：
   - [ ] **标题公式**：核心商品词 + 关键词/属性 + 使用场景 + 人群/卖点 + 规格；用**马来站**热搜词/蓝海词，不照搬泰/菲；**≤ 255 字符、不写营销促销语、不用未授权品牌词、不用特殊符号**。
   - [ ] **类目选对**（🔴 **选错类目会被平台下架**）；商品属性按类目准确填。
   - [ ] **主图 ≥ 5 张**（少于 5 张曝光转化偏低）：白底/完整/清晰，无阴影抠图痕迹/过暗过曝；副图换场景换用法。
   - [ ] **视频**：≤ 1 分钟、**开头 3 秒先展示外观**（可选但强烈建议，直接影响测款转化）。
   - [ ] **服装类目必传尺码图**；SKU 命名简洁；**重量核准**（影响藏价）。
   - [ ] 详情页：欢迎语 + 核心卖点 + 材质/规格 + 使用场景 + 尺寸/注意 + 物流售后；**不夸大、禁绝对化/无资质宣称**（"全美销量第一"等禁用）。
3. **合规终检**（发布前最后一关）：
   - [ ] 不上假货/侵权/**处方·矫正型眼镜**/药品医疗/成人/金融/虚拟币/礼品卡/危险品/禁运品；
   - [ ] 品牌词/Logo/授权资质匹配；**IP 款（三丽鸥/迪士尼/漫威等）须授权**、运动/童装**不带球队 Logo**；马来违规看 **48 分违规积分制**。
4. **发布**（§5.3）：草稿「发布」或「定时发布」；批量发布勾多个 → 输**间隔时间** → 批量发布；**审核成功后才能开启售卖**。
5. 🔴 **卡点 F（藏价 ≠ 最终价）**：套了定价模板发布后，**仍要在店铺后台做对应折扣/促销**（模板只是把成本藏进原价，不做折扣买家看到的是高价）。

- **验收物**：商品状态「在线/可售」+ 主推款标题/主图/详情二次检查通过 + 后台折扣已配。

## Step 6 · （出单后）闭环 —— 交接给履约，不在本脚本主体，但要知道去哪
> 本脚本到"上架可售"结束。出单后的采货—发货—收款闭环，走这条（详见对应板块，别在这重复操作）：

出单 → 货叮咚**采购派单到 1688**（官方 API 采购 / 插件拍单，[03 §6](../reference/03-商品上架/货叮咚采集上架与定价工具链.md)）→ 在**利润明细回填实际采购价 + 货代费**（[03 §7](../reference/03-商品上架/货叮咚采集上架与定价工具链.md)，与 Step 4 藏价对齐）→ **货代打包贴单送东莞官仓**（[04 履约发货](../reference/04-履约发货/)）→ **守 72h 入仓 SLA** → 平台物流到马来买家 → **连连国际收款**（[01 连连开户 SOP](../reference/01-开店入驻/连连国际开户SOP.md)）。

上架后进入**测款期**：按 **7 天看曝光→点击→加购→转化 / 2 周下架高曝光零转化 / 28 天零单判失败复盘**（🟠 讲师经验节奏），用工作台 **「商品四象限分层」sheet** 对症优化（低曝光高转化=改标题关键词；高曝光低点击=换主图调价；点击高转化低=改详情卖点）——详见 [02 测款与市场 §1.3](../reference/02-选品/选品测款与东南亚市场.md) + [08 数据分析](../reference/08-数据分析/)。

## 一页速查（每款过一遍）
| 步 | 一句话 | 必过卡点 |
|---|---|---|
| 1 选品 | 后台 4 入口找赛道 → 量化筛（增长>50%/点赞>1万）→ 工作台打分 | 🔴A 进货价>50/大件不选　🔴B 禁限售+清真初筛 |
| 2 货源 | 1688 找可一件代发的源，确认价/重量 | 🔴C 发货慢/易断货的源淘汰（守 72h） |
| 3 采集 | 采集配置过滤 → 采集进箱 → 认领生成草稿 | 🔴D 图片去中文/logo/水印，违禁词过滤开 |
| 4 定价 | 建马来/跨境定价模板藏价 → 工作台复核利润>0 | 🔴E 费率用马来当期（勿抄泰国 9.63%）+必填折扣+重量准 |
| 5 上架 | 编辑（标题/类目/主图≥5/尺码图）→ 合规终检 → 发布 | 🔴F 发布后店铺后台还要做折扣（藏价≠最终价） |
