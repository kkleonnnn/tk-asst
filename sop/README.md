# sop —— 标准操作流程

可以一步步照着执行的操作清单（人或 AI 都能跟着走）。

**本目录文件**：
- `TikTok-Shop-0到稳定出单-执行计划.md` —— ★旗舰执行计划（阶段 0–7 时间线，统领 01–09 知识落地顺序）
- `选品到上架-实操脚本.md` —— ★上新流水线脚本（选品→找货源→采集→定价→上架，货叮咚动作级、可复用循环；= 执行计划「阶段 2」的展开版）
- `角色分工.md` —— 校长运营 / kk 技术 / AI 产出；两遍消化制；归属待议

每个 SOP 都应注明：适用国家站点、是否需要人工确认、上次核对日期。
